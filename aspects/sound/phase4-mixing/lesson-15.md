- コンプレッサーの4つのパラメータを理解し、基本設定ができる

前回のモニターミックスで、ミュージシャンに「素の音」を届ける方法を学んだ。今日はその音に「味付け」をする --- エフェクトの話だ。

最初に言っておく。**ライブ音響のエフェクトは「少なめ」が正解。** レコーディングと違って、ライブは部屋自体に残響がある。そこにエフェクトを盛りすぎると、音がぼやけて何を言ってるか聴こえなくなる。

理由がわかれば忘れない --- 「なぜこのエフェクトを使うのか」を常に考えよう。

## 1. エフェクトの基本ルール（5分）

### ライブ音響の鉄則

```
┌──────────────────────────────────────┐
│                                      │
│   エフェクトは「感じるもの」であって  │
│   「聴こえるもの」ではない           │
│                                      │
│   Feel it, don't hear it.            │
│                                      │
└──────────────────────────────────────┘
```

- リバーブ: ON/OFF を切り替えたとき、**OFF にして「あれ、ちょっと寂しいな」と思う程度**が正解
- コンプレッサー: かけていることに**気づかないのが良いコンプ**
- エフェクトが「聴こえて」しまったら、かけすぎ

> **中村メモ:** レコーディングでは派手にエフェクトをかけることもある。でもライブは違う。会場の自然な残響 + エフェクトの残響が合わさるから、エフェクトは控えめにする。教会の礼拝堂は特に残響が多い空間だから、なおさら少なめで。

## 2. リバーブ --- 空間を作る（15分）

### リバーブとは

リバーブ（reverb = reverberation = 残響）は、**音が壁や天井に反射して返ってくる現象**を人工的に再現するエフェクト。

```
原音だけ（リバーブなし）:
  「ハレルヤ」 → ・・・（すぐ消える）

リバーブあり:
  「ハレルヤ」 → ヤヤヤヤ...ヤ...ヤ...(余韻が残る)

  ┌─────────── 壁 ───────────┐
  │                           │
  │    🎤 ──→ 直接音          │
  │     ↓↗     ↗  ↗          │
  │   反射音  反射  反射      │
  │                           │
  └───────────────────────────┘
```

### リバーブの種類

| 種類 | 特徴 | 教会での用途 |
|------|------|-------------|
| **Plate**（プレート） | 明るく滑らか、密度が高い | ボーカルに最適。最もよく使う |
| **Hall**（ホール） | 広くて荘厳、余韻が長い | バラード、静かな賛美 |
| **Room**（ルーム） | 自然で短い、部屋にいる感じ | ドラム、アコギ |
| **Chamber**（チェンバー） | Plate と Hall の中間 | オールマイティ |

### 教会での選び方

```
迷ったらこれ:

  ボーカル全般    → Plate（明るくて聴き取りやすい）
  説教マイク      → 短い Plate or なし（言葉の明瞭さ優先）
  バラード賛美    → Hall（荘厳な雰囲気）
  アップテンポ賛美 → 短い Plate（テンポが遅く聴こえるのを防ぐ）
  ドラム          → Room（自然な空気感だけ足す）

  ※ リバーブタイム（余韻の長さ）の目安:
  ├── 短い: 0.8〜1.2秒（説教、アップテンポ）
  ├── 中間: 1.5〜2.0秒（通常の賛美）
  └── 長い: 2.5〜3.5秒（バラード、静かな賛美）
```

### リバーブの送り方（Aux センド / リターン）

**重要: リバーブはインサートではなく、Aux センド / リターンで使う。**

```
  ✗ 間違い: チャンネルに直接インサート

  ch1 [Vo1] → [リバーブ] → Main L/R

  ※ 原音100%がリバーブを通過する = 原音にリバーブが混ざり制御不能

  ---

  ○ 正解: Aux センド / リターン

  ch1 [Vo1] ──────────────────→ Main L/R（原音 = ドライ）
      │
      └── Aux 5 ──→ [リバーブ] ──→ Main L/R（残響 = ウェット）

  ※ 原音とリバーブ音を別々に制御できる
  ※ 複数チャンネルで1つのリバーブを共有できる（効率的）
```

**Aux センド/リターンの全体図:**

```
ch1 [Vo1] ─┬────────→ Main L/R
            └─ Aux5 ─┐
ch2 [Vo2] ─┬────────→ Main L/R
            └─ Aux5 ─┤
ch3 [Vo3] ─┬────────→ Main L/R
            └─ Aux5 ─┤
ch7 [K1L] ─┬────────→ Main L/R
            └─ Aux5 ─┤
                      ↓
              ┌──────────────┐
              │ Aux 5 バス    │
              │  (リバーブ)   │
              │ Plate 1.5sec │
              └──────┬───────┘
                     ↓
                  Main L/R（リバーブ音だけ足される）
```

**なぜ Aux センド/リターンか:**
1. **1つのリバーブを複数チャンネルで共有** --- CPU/DSP の節約
2. **各チャンネルから送る量を個別に調整** --- Vo1 は多め、Keys は少なめ
3. **原音（ドライ）と残響（ウェット）を独立制御** --- リバーブのリターンフェーダーで全体の残響量を調整

> **中村メモ:** 「リバーブをインサートに入れるな」--- これだけ覚えておけばいい。Aux で送る。理由は上の通り。デジタルミキサーなら内蔵エフェクトを Aux に割り当てるだけで簡単に設定できる。

## 3. コンプレッサー --- 音量を整える（15分）

### コンプレッサーとは

コンプレッサー（compressor = 圧縮器）は、**大きすぎる音を自動で抑えるエフェクト。**

```
コンプレッサーなし:
  小さい音   ░░          ← 聴こえにくい
  普通の音   ░░░░░░      ← ちょうどいい
  大きい音   ░░░░░░░░░░░░░░░░  ← 大きすぎ！クリップ！

コンプレッサーあり:
  小さい音   ░░░░        ← 少し持ち上がる（※）
  普通の音   ░░░░░░      ← そのまま
  大きい音   ░░░░░░░░░░  ← 抑えられる
                          ← ダイナミックレンジが「圧縮」される

  ※ コンプ後にメイクアップゲインで全体を持ち上げた場合
```

**なぜ必要か:** ボーカリストはサビで急に声が大きくなったり、静かな部分でささやいたりする。その音量差が大きすぎると、静かな部分が聴こえない or 大きい部分がクリップする。コンプレッサーがその差を自動で縮めてくれる。

### 4つのパラメータ

```
                  threshold（スレッショルド）
                      ↓
  音量 ──→ [この線を超えたら圧縮が始まる] ──→ ratio で圧縮
                                                    ↓
            attack（アタック）= 圧縮が始まるまでの時間
            release（リリース）= 圧縮が終わるまでの時間
```

#### (1) Threshold（スレッショルド = しきい値）

```
  ──────── Threshold ────────
       ↑ この線を超えた音だけ圧縮される

  音量
  ▓▓▓▓▓▓▓▓▓▓▓▓▓  ← 超えた分が圧縮される
  ────────────────  ← Threshold ライン
  ▒▒▒▒▒▒▒▒▒▒▒▒▒
  ░░░░░░░░░░░░░    ← Threshold 以下は影響なし
```

**設定の目安:** 普通に歌っているときはほぼ圧縮されず、サビや叫ぶときだけ圧縮がかかる程度。ミキサーのゲインリダクションメーター（GR）が **-3dB 〜 -6dB** 程度振れるのが目安。

#### (2) Ratio（レシオ = 圧縮比）

```
  Ratio 1:1 = 圧縮なし（そのまま通す）
  Ratio 2:1 = Threshold を超えた分を半分にする
  Ratio 4:1 = Threshold を超えた分を 1/4 にする
  Ratio ∞:1 = リミッター（Threshold 以上は絶対に出さない）

  例: Threshold = -10dB、Ratio = 2:1 のとき
  入力 -10dB → 出力 -10dB（Threshold 以下なのでそのまま）
  入力  -4dB → 出力  -7dB（6dB 超過 → 3dB に圧縮）
  入力  +2dB → 出力  -4dB（12dB 超過 → 6dB に圧縮）
```

**教会での目安:**

| 用途 | Ratio |
|------|-------|
| ボーカル（軽く整える） | 2:1 〜 3:1 |
| ボーカル（しっかり整える） | 4:1 |
| ドラム（パンチを出す） | 4:1 〜 6:1 |
| 説教マイク（安定させる） | 3:1 〜 4:1 |

#### (3) Attack（アタック = 反応速度）

```
  音が Threshold を超えてから、圧縮が始まるまでの時間。

  速い Attack（1-5ms）:
  入力:  ╱▓▓▓▓▓▓▓▓
  出力:  ╱▒▒▒▒▒▒▒▒  ← すぐ圧縮、アタック感が弱まる

  遅い Attack（20-50ms）:
  入力:  ╱▓▓▓▓▓▓▓▓
  出力:  ╱▓▒▒▒▒▒▒▒  ← 最初のアタックは通す、その後圧縮
```

- **ボーカル:** 中程度（10-20ms）--- 発音の明瞭さを保ちつつ圧縮
- **ドラム:** 遅め（20-40ms）--- アタックのパンチを残したい
- **説教:** 速め（5-10ms）--- 急な音量変化を素早く抑える

#### (4) Release（リリース = 復帰速度）

```
  音が Threshold を下回ってから、圧縮が解除されるまでの時間。

  速い Release（50-100ms）:
  圧縮 → すぐ元に戻る（自然だが、ポンピングの原因に）

  遅い Release（200-500ms）:
  圧縮 → ゆっくり元に戻る（滑らかだが、次のフレーズも圧縮されがち）
```

- **目安:** 100-200ms で始めて、聴きながら調整
- **ポンピング:** Release が速すぎると音量がふわふわ揺れる現象。これが聴こえたら Release を遅くする

### コンプレッサーの接続方法（インサート）

**リバーブと違い、コンプレッサーはインサートで使う。**

```
  ○ コンプレッサー: インサート（チャンネルに直接挿す）

  ch1 [Vo1] → [コンプ] → EQ → Main L/R
               ↑
          チャンネルの信号が100%コンプを通過する

  ※ 原音を「処理」したいから、直接信号経路に入れる
```

**なぜインサートか:** コンプレッサーは原音の音量を制御するエフェクト。原音とエフェクト音を混ぜるのではなく、原音そのものを処理する。だからインサート。

```
エフェクトの接続方法まとめ:

  ┌──────────────┬──────────────┬──────────────────────┐
  │ エフェクト   │ 接続方法     │ 理由                 │
  ├──────────────┼──────────────┼──────────────────────┤
  │ リバーブ     │ Aux S/R      │ 原音+残響を混ぜる    │
  │ ディレイ     │ Aux S/R      │ 原音+やまびこを混ぜる│
  │ コンプレッサー│ インサート   │ 原音自体を処理する   │
  │ EQ           │ インサート   │ 原音自体を処理する   │
  │ ゲート       │ インサート   │ 原音自体を処理する   │
  └──────────────┴──────────────┴──────────────────────┘
```

> **中村メモ:** 「混ぜるエフェクト → Aux」「処理するエフェクト → インサート」。この分け方だけ覚えればいい。理由がわかれば忘れない。

## 4. 教会での具体的な設定例（10分）

### ボーカル用コンプレッサー

```
パラメータ:
  Threshold : -18dB 〜 -12dB（普段の歌声で軽くかかる程度）
  Ratio     : 3:1
  Attack    : 10-15ms
  Release   : 150ms

GR メーター: 普段 -2〜3dB、サビで -5〜6dB 程度

※ ゲインリダクション（GR）が -10dB 以上振れたら
   Threshold を上げるか Ratio を下げる
```

### 説教マイク用コンプレッサー

```
パラメータ:
  Threshold : -15dB 〜 -10dB
  Ratio     : 4:1（しっかりめ）
  Attack    : 5ms（速め = 急な声量変化に対応）
  Release   : 200ms

※ 説教は「聴き取りやすさ」が最優先
※ 牧師がマイクから離れたり近づいたりする → コンプで安定させる
```

### ボーカル用リバーブ

```
タイプ    : Plate
Time      : 1.5秒
Pre-delay : 20-30ms（原音とリバーブの間に隙間を作る）
High Cut  : 6-8kHz（リバーブの高域をカット → シュワシュワしない）
Mix       : Aux リターンで調整（100% ウェットで送り、リターンフェーダーで量を決める）
```

**Pre-delay（プリディレイ）とは:**

```
Pre-delay なし:
  「ハレルヤ」→ ヤヤヤ...（原音と残響が同時に始まる → ぼやける）

Pre-delay 20ms:
  「ハレルヤ」→ (20ms) → ヤヤヤ...（原音がクリアに聴こえてから残響が始まる）
```

> **中村メモ:** Pre-delay は「言葉の聴き取りやすさ」を保つ秘訣。特に説教マイクにリバーブをかけるなら必須（でも説教にはリバーブなしが一番クリア）。

## 5. ディレイ --- 礼拝での使いどころ（5分）

ディレイ（delay = 遅延）は音を繰り返すエフェクト。いわゆる「やまびこ」。

```
原音:     「ハレルヤ」
ディレイ: 「ハレルヤ」→「ハレルヤ」→「ハレルヤ」...
                         (小さく)    (さらに小さく)
```

### 教会での使い方

**ボーカルスローイング（Vocal Throw）:**

フレーズの最後の一瞬だけディレイを ON にして、余韻を作るテクニック。

```
歌:  「Amazing Grace...」
                     ↑ ここだけディレイ ON
結果: 「Amazing Grace...ace...ace...」

※ 常時 ON ではなく、フレーズの切れ目だけ
※ Aux センドのミュートを一瞬だけ解除する
```

**注意:** ディレイは使いすぎると音がごちゃごちゃになる。教会では「ここぞ」という場面だけに使おう。最初は使わなくてもいい。

## 実践ドリル（10分）

### ドリル 1: リバーブの量を判断する

1. 音楽を再生する（できれば賛美）
2. リバーブを強くかける（明らかに「聴こえる」くらい）
3. 少しずつ下げていく
4. **「OFF にしたらちょっと寂しい」ポイント**を見つける
5. そこが適正量

### ドリル 2: コンプレッサーのパラメータ

以下の設定から始めて、声を出しながら調整してみよう:

```
  スタート設定:
  Threshold : -15dB
  Ratio     : 3:1
  Attack    : 10ms
  Release   : 150ms

  やること:
  1. 普通に話す → GR メーターが -2〜3dB 振れるか確認
  2. 大きな声を出す → GR が -8dB 以上振れたら Threshold を上げる
  3. Attack を速く/遅くして音の印象の変化を聴く
```

### ドリル 3: Aux とインサートの使い分け

| ギターにディレイ | ？ | ？ |

## 教会での応用

### エフェクト設定チェックリスト

```
礼拝前のチェック:
  □ リバーブタイプと時間を確認（Plate / 1.5sec）
  □ Aux 5 にリバーブをアサイン
  □ ボーカルch から Aux 5 への送り量を設定
  □ リバーブリターンのフェーダーを適切な位置に
  □ ボーカルch のコンプを ON にして GR メーターを確認
  □ 説教マイクのコンプを設定
  □ テスト: 声を出してリバーブの量を耳で確認
```

### やってはいけないこと

| NG | 理由 |
|----|------|
| リバーブを全チャンネルにかける | ドラムや説教がぼやける |
| コンプの Ratio を 10:1 以上にする | 音が潰れて不自然になる |
| エフェクトを礼拝中に大きく変える | 突然音が変わって気が散る |
| モニターにリバーブを返す | ハウリングのリスクが上がる |

> **中村メモ:** 「迷ったらかけない」が正解。エフェクトなしでも礼拝はできる。エフェクトありきで考えない。まず素の音をしっかり作って、その上で「もうちょっと空間感が欲しいな」と思ったときだけリバーブを足す。

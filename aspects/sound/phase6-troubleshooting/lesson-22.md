## 1. ノイズの種類と聞き分け方（10分）

まず、ノイズには種類があります。音を聞いて原因を推測できるようになりましょう。

```
  ┌──────────────┬──────────────────┬──────────────────────┐
  │ ノイズの種類  │ どんな音？        │ よくある原因          │
  ├──────────────┼──────────────────┼──────────────────────┤
  │ ハム (Hum)   │ ブーーーン        │ グラウンドループ      │
  │              │ 低い一定音        │ 電源の 50/60Hz       │
  ├──────────────┼──────────────────┼──────────────────────┤
  │ バズ (Buzz)  │ ジジジジジ        │ 電磁干渉 (EMI)       │
  │              │ ハムより粗い      │ 照明ディマー          │
  ├──────────────┼──────────────────┼──────────────────────┤
  │ ヒス (Hiss)  │ シャーーー        │ ゲイン上げすぎ        │
  │              │ ホワイトノイズ的  │ ノイジーなプリアンプ  │
  ├──────────────┼──────────────────┼──────────────────────┤
  │ クラックル    │ パチパチ・ガリガリ │ ケーブル不良          │
  │ (Crackle)   │ 不規則な断続音    │ 接触不良              │
  ├──────────────┼──────────────────┼──────────────────────┤
  │ ドロップアウト │ ブツッ...無音... │ ワイヤレスの電波障害  │
  │ (Dropout)    │ 音が途切れる     │ バッテリー切れ        │
  └──────────────┴──────────────────┴──────────────────────┘
```

### 周波数で見るノイズ

```
  音量
  ▲
  │ █           ← ハム（50Hz / 60Hz にピーク）
  │ █ █         ← バズ（倍音も含む: 100Hz, 150Hz, 200Hz...）
  │ █ █ █
  │ █ █ █ ░░░░░░░░░░░░░░░░░░ ← ヒス（高域全体に広がる）
  │ █ █ █ ░░░░░░░░░░░░░░░░░░
  ├──┬──┬──┬──┬──┬──┬──┬──→ Hz
  50 100 200 500 1k 2k 4k 8k

  █ = ハム/バズ   ░ = ヒス
```

> **田中メモ:** 日本の電源は東日本 50Hz、西日本 60Hz。教会がどちらの地域にあるかで、ハムの基音が変わる。横浜なら 50Hz だね。

## 2. グラウンドループ（15分）

### グラウンドループとは

2つ以上の機器が異なる電源コンセントに繋がっていて、オーディオケーブルでも繋がっているとき、**電位差によって電流がループ状に流れる**現象です。

```
  【正常な状態】

  コンセントA ─── [ミキサー] ───オーディオケーブル─── [PC] ─── コンセントA
                                                              （同じ電源）
  → グラウンドの電位が同じ → ノイズなし


  【グラウンドループ】

  コンセントA ─── [ミキサー] ───オーディオケーブル─── [PC] ─── コンセントB
       │                    ↑                              │
       │                    │ ← この経路に微小な電流が流れる │
       │                    │    （= ブーンというハム音）     │
       └──── 建物のアース線 ────────────────────────────────┘
                  電位差あり！
```

**もっと簡単に言うと:**

```
  ┌──────────┐  オーディオケーブル  ┌──────────┐
  │ ミキサー  │ ◄─────────────────► │   PC     │
  └────┬─────┘                     └────┬─────┘
       │ 電源ケーブル                    │ 電源ケーブル
       ▼                                ▼
  [コンセントA]                    [コンセントB]
       │                                │
       └──── 建物のグラウンド線 ─────────┘
                    ↑
              ここに電位差があると
              ループに電流が流れる
              → ブーーーン
```

### グラウンドループの見つけ方

**「1本ずつ抜く」方法:**

```
  手順:

  1. ノイズが出ている状態で、ミキサーに繋がっている
     機器のオーディオケーブルを1本ずつ抜く

  2. 抜いた瞬間にハムが消えたら → その機器がループの原因

     [ミキサー] ←── キーボード1    ← 抜いてみる → ハムまだ出る
     [ミキサー] ←── キーボード2    ← 抜いてみる → ハムまだ出る
     [ミキサー] ←── PC（BGM/配信） ← 抜いてみる → ハム消えた！★
     [ミキサー] ←── iPhone（BGM）  ← まだ調べてない

  3. PC が犯人 → PC とミキサーの電源経路を確認
```

### 解決策

**解決策 1: DI ボックスのグラウンドリフト**

```
  [PC] ──→ [DI ボックス] ──→ [ミキサー]
                 │
           GND LIFT スイッチ
                 │
              ON にする
                 │
           グラウンドの接続を切断
                 │
           ループが断ち切られる
                 │
           ハム消滅！
```

教会でよくグラウンドループを起こす機器:
- **PC（配信・カウントダウン用）** → DI ボックス経由で接続
- **キーボード（ステレオ2台）** → DI ボックス経由で接続
- **iPhone（BGM）** → DI ボックス経由で接続

**解決策 2: 同じ電源タップに繋ぐ**

```
  ┌──────────────────────────────┐
  │         電源タップ            │
  │  [ミキサー][PC][キーボード]   │
  │  ← 全部同じタップに繋ぐ →    │
  └──────────────┬───────────────┘
                 │
            [コンセント]  ← 1つの電源から
```

**解決策 3: オーディオアイソレーショントランス**

```
  [PC] ──→ [アイソレーショントランス] ──→ [ミキサー]
                    │
            電気的に完全に分離
            （磁気結合で信号だけ通す）
```

> **田中メモ:** 配信 PC は特にグラウンドループを起こしやすい。USB オーディオインターフェースを使ってる場合、PC の電源アダプターが原因のことが多い。DI ボックスは1台 3,000-5,000 円で買えるから、教会に数台常備しておくといい。

## 3. 電磁干渉（EMI）対策（10分）

### EMI の発生源

```
  教会で EMI を起こすもの:

  ┌─────────────────────────────────────────────┐
  │ 📱 スマートフォン（マイク近くに置かない！）    │
  │ 💡 照明ディマー（最大の敵！）                 │
  │ ⚡ 電源ケーブル（オーディオケーブルと並走NG）  │
  │ 🖥️ PC のモニター                             │
  │ 📡 Wi-Fi ルーター                            │
  └─────────────────────────────────────────────┘
```

### バランス（平衡）接続の重要性

```
  【アンバランス接続】（ノイズに弱い）

  信号線  ─────────── ←ノイズが乗る→ ─────────
  GND線   ─────────── ←ノイズが乗る→ ─────────

  → ノイズがそのまま出力される


  【バランス接続】（ノイズに強い）

  HOT(+)  ──── + ←ノイズA→ ────── (+信号 + ノイズA)
  COLD(-) ──── - ←ノイズA→ ────── (-信号 + ノイズA)
  GND     ────────────────────────

  受け側で: (+信号 + ノイズA) - (-信号 + ノイズA)
          = +信号 + 信号 + ノイズA - ノイズA
          = 2倍の信号 + ノイズゼロ！
```

**教会での対策:**
- マイクケーブルは全て **XLR（バランス）** を使う
- キーボード・PC は **DI ボックス**でバランスに変換
- ギター・ベースは **DI ボックス**経由

### ケーブルの配線ルール

```
  【NG: 並行配線】

  ═══════════════════ 電源ケーブル
  ─────────────────── オーディオケーブル

  → 電源ケーブルの磁界がオーディオに干渉


  【OK: 直交配線】

  ═══════════════════ 電源ケーブル
            │
            │  ← 90°で交差
  ──────────┼───────── オーディオケーブル
            │
```

```
  【OK: 十分な距離】

  ═══════════════════ 電源ケーブル


  ～～30cm以上離す～～


  ─────────────────── オーディオケーブル
```

> **田中メモ:** 教会の照明にディマーを使っている場合、ジジジジという独特のバズが出ることがある。対策としては、調光器を PWM 方式に変更するか、音響用の電源を照明とは別系統にすること。ステージの電源と照明の電源を分けるだけで劇的に改善する場合がある。

## 4. ケーブル不良のトラブルシュート（10分）

### ウィグルテスト

```
  ケーブルの端子付近を軽く動かす:

  [機器] ──── ╳ ←ここを指で揺する→ ────── [ケーブル]
               │
          ガリガリ・パチパチ鳴ったら
          → そのケーブルは不良！交換！
```

**注意:** テスト時はフェーダーを下げてから行う。いきなりバリバリ音が出ると危険。

### ケーブルテスターの使い方

```
  ┌──────────────┐
  │ ケーブル      │
  │ テスター      │
  │              │
  │  PIN 1: ✅   │  ← GND 導通OK
  │  PIN 2: ✅   │  ← HOT 導通OK
  │  PIN 3: ❌   │  ← COLD 断線！
  │              │
  │  SHORT: ✅   │  ← ショートなし
  └──────────────┘
```

### スペアケーブル戦略

```
  教会に常備すべきスペアケーブル:

  ┌──────────────────────────────────────┐
  │ XLR ケーブル（3m）      × 3本       │
  │ XLR ケーブル（5m）      × 2本       │
  │ XLR ケーブル（10m）     × 1本       │
  │ TRS ケーブル（3m）      × 2本       │
  │ 楽器ケーブル TS（5m）   × 2本       │
  │ スピーカーケーブル       × 1本       │
  │ USB ケーブル（各種）    × 2本       │
  │ HDMI ケーブル           × 1本       │
  │ 電源延長コード          × 1本       │
  │ 各種変換アダプター      各1個        │
  └──────────────────────────────────────┘

  → ケーブルバッグにまとめてミキサー横に常備
```

> **田中メモ:** ケーブルは消耗品。安いケーブルでもちゃんと交換サイクルを決めれば問題ない。うちでは半年に一度、全ケーブルをテスターで検査して、怪しいものは即廃棄している。配信用の USB ケーブルも意外と盲点。

## 5. ワイヤレスマイクのトラブル（5分）

教会にはワイヤレスボーカルマイク4本と説教用マイク2本がありますね。

### バッテリー管理

```
  ワイヤレスマイクのバッテリールール:

  ┌──────────────────────────────────────┐
  │ ✅ 礼拝前に必ず新品バッテリーに交換  │
  │ ✅ 使用済みバッテリーは別の箱に入れる │
  │ ✅ 充電池なら前日に満充電を確認       │
  │ ❌ 「まだ残ってるから大丈夫」は禁止   │
  └──────────────────────────────────────┘

  バッテリー残量の目安（単三アルカリの場合）:
  ██████████ 100% → OK
  ███████░░░  70% → 礼拝1回はギリギリ
  █████░░░░░  50% → 交換必須
  ██░░░░░░░░  20% → 使ってはいけない
```

### 電波干渉とドロップアウト

```
  ドロップアウトが起きやすい場所:

  ┌──────────────────────────────────────┐
  │            会衆席                     │
  │                                      │
  │  ◎ スピーカー          スピーカー ◎  │
  │                                      │
  │  ─── ステージ ───                    │
  │                                      │
  │  🎤🎤🎤🎤                             │
  │                                      │
  │  📡 受信機（ステージ横）              │
  │                                      │
  │  ×× 金属の棚の裏 ← ここは電波が弱い  │
  │  ×× 鉄骨の柱の陰 ← ここも危険       │
  └──────────────────────────────────────┘
```

**対策:**
- 受信機のアンテナをステージ前面に出す
- リハーサルで全エリアを歩いてドロップアウトゾーンを確認
- 周波数スキャンで干渉の少ないチャンネルを選ぶ

## 6. 分割統治法（Divide and Conquer）（5分）

ノイズの原因がわからないとき、**信号経路を半分に分けて、どちら側に問題があるか絞り込む**方法です。

```
  信号経路全体:
  [マイク] → [ケーブル] → [ステージボックス] → [スネーク] → [ミキサー] → [アンプ] → [スピーカー]

  STEP 1: 真ん中で切る
  [マイク→ステージボックス→スネーク]  |  [ミキサー→アンプ→スピーカー]
  前半にヘッドホンを繋いで確認        |  後半にテスト信号を入れて確認
          │                          │
      ノイズあり？                ノイズあり？
          │                          │
     前半に問題あり              後半に問題あり
          │                          │
          ▼                          ▼
  STEP 2: さらに半分に切る       STEP 2: さらに半分に切る
  [マイク→ケーブル]|[ステージBOX]  [ミキサー]|[アンプ→スピーカー]
          │                          │
     繰り返して絞り込む           繰り返して絞り込む
```

> **田中メモ:** この方法は20チャンネル以上ある教会のシステムで特に有効。全チャンネルを1本ずつチェックすると時間がかかるけど、分割統治法なら半分ずつ除外できるから、4-5ステップで原因に辿り着ける。配信トラブルでも同じ考え方が使える。

## 実践ドリル（10分）

1. **ノイズ聞き分けクイズ**: ミキサーのゲインを上げてヒスを聞く。次にスマホをマイクケーブルに近づけてバズを聞く。違いを体感する
2. **グラウンドループ実験**: PC をミキサーに直接繋ぎ、次に DI ボックス経由で繋ぐ。ノイズの違いを確認する
3. **ケーブル配線チェック**: 教会の配線を確認し、電源ケーブルとオーディオケーブルが並走している箇所をリストアップする
4. **ワイヤレステスト**: 4本のワイヤレスマイクで会場を歩き回り、ドロップアウトする場所がないか確認する

## 教会での応用

- **DI ボックスの常備**: キーボード2台 + PC + iPhone = 最低4台の DI ボックスが必要。グラウンドリフトスイッチ付きを選ぶ
- **電源計画**: ステージの電源と照明の電源を別系統にする。電源タップはオーディオ専用を用意する
- **ケーブル管理**: 色分けやラベリングで「このケーブルはどこに繋がっている」を一目でわかるようにする
- **配信品質**: 配信の音声は会場よりもノイズが目立つ。定期的に配信のアーカイブを聴いてノイズチェックする

# Lesson 9: 電源の順番 --- なぜスピーカーが最後なのか

> **Phase 3 / L9: 電源ON/OFFの正しい順番を覚えて、機材を壊さずに済むようになる**
> **担当:** 中村響（リードインストラクター）
> **所要時間:** 約60分
> **前提:** Phase 1-2 完了

---

## 今日のゴール
- 電源を入れる順番・切る順番の「黄金ルール」を理解する
- うちの教会の機材で、具体的にどのスイッチをどの順番で操作するか覚える
- 順番を間違えたらどうなるか、理由ごと理解する

---

さて、今日はPA操作の中でも一番最初にやること——「電源を入れる」話です。

「え、電源なんてスイッチ入れるだけでしょ？」って思うかもしれません。でもね、電源の順番を間違えると、スピーカーから「バンッ！」ってすごい音が出て、最悪スピーカーが壊れます。修理代、何万円もしますよ。

**理由がわかれば忘れない。** 今日はなぜその順番なのか、しっかり理解していきましょう。

## 1. 信号の流れをおさらいしよう（5分）

Phase 1-2で学んだ信号の流れ、覚えてますか？

```
音の源 → ミキサー → アンプ/スピーカー
（マイク、   （音を     （音を大きく
  キーボード   混ぜる）    して出す）
  等）
```

電源の順番は、この **信号の流れ** がそのままルールになります。

```
┌─────────────────────────────────────────────────┐
│           電源ON = 信号の流れと同じ方向            │
│                                                   │
│   音源 ──→ ミキサー ──→ エフェクト ──→ スピーカー  │
│   [1番]     [2番]       [3番]        [4番・最後]  │
│                                                   │
│           電源OFF = 逆方向                        │
│                                                   │
│   スピーカー ──→ エフェクト ──→ ミキサー ──→ 音源  │
│   [1番・最初]   [2番]        [3番]       [4番]    │
└─────────────────────────────────────────────────┘
```

> **中村メモ:** この「ON=前から後ろ、OFF=後ろから前」が黄金ルール。世界中どこのPA現場でも同じです。これだけ覚えれば、初めて触る機材でも安全に扱えます。

## 2. なぜこの順番なのか？（10分）

### 電源を入れた瞬間、何が起きるか

機材の電源を入れると、回路に電気が流れ始めます。このとき、一瞬だけ不安定な電気信号（**ポップノイズ**）が発生します。

```
電源ON直後の信号

通常:    ────────────────────  （静か）

実際:    ──╱╲──────────────── （一瞬「バッ」という信号が出る）
            ↑
         ポップノイズ
```

この「バッ」は小さな信号ですが、もしスピーカーが先にONになっていると...

```
【危険なパターン】スピーカーがONの状態でミキサーの電源を入れる

ミキサーの     ──╱╲──→  アンプで   ──╱╲╲╲──→  スピーカー
ポップノイズ              増幅！！                「バンッ！！」
（小さい）               （大きくなる）          （大音量）
```

### 具体的にどうなるか

1. **スピーカーのコーン（振動板）が急激に動く** → 物理的に破損する可能性
2. **聞いている人の耳が痛い** → 礼拝中なら大問題
3. **アンプに過大な負荷がかかる** → アンプも壊れる可能性

### だから「スピーカーは最後にON、最初にOFF」

```
【安全なパターン】

ON時: 音源 → ミキサー → スピーカー
      ポップノイズが出ても、スピーカーはまだOFF → 音は出ない → 安全！

OFF時: スピーカー → ミキサー → 音源
       スピーカーを先に切る → ミキサーを切ってもノイズは聞こえない → 安全！
```

> **中村メモ:** 15年やってきて、「バンッ」を鳴らしたスタッフを何人も見てきました。機材が壊れなくてもめちゃくちゃ焦ります。順番を覚えておけば、そんな冷や汗をかかなくて済みますよ。

## 3. うちの教会の電源ON手順（15分）

教会の具体的な機材で、実際の手順を確認しましょう。

### 電源ON チェックリスト

```
=== 電源ON（礼拝30分前） ===

ステップ1: 準備
  □ ミキサーのマスターフェーダーが下がっていることを確認
  □ すべてのチャンネルフェーダーが下がっていることを確認
  □ スピーカーの電源がOFFであることを確認

ステップ2: 音源（ステージ側）
  □ キーボード①の電源ON
  □ キーボード②の電源ON
  □ エレキギターのアンプ電源ON
  □ ベースのアンプ/DI電源ON
  □ iPhone（BGM用）を接続
  □ PC（カウントダウン用）を接続

ステップ3: ミキサー
  □ デジタルミキサーの電源ON
  □ 起動完了まで待つ（30秒〜1分）
  □ 前回のシーン/設定がロードされたか確認

ステップ4: 配信機材
  □ 配信用PCの電源ON
  □ 配信用オーディオインターフェースの電源ON

ステップ5: スピーカー（最後！）
  □ メインスピーカー L の電源ON
  □ メインスピーカー R の電源ON
  □ サブウーファーの電源ON
  □ フロアモニターの電源ON
```

### 電源OFF チェックリスト

```
=== 電源OFF（礼拝終了後） ===

ステップ1: 準備
  □ ミキサーのマスターフェーダーを下げる
  □ すべてのチャンネルフェーダーを下げる

ステップ2: スピーカー（最初！）
  □ フロアモニターの電源OFF
  □ サブウーファーの電源OFF
  □ メインスピーカー R の電源OFF
  □ メインスピーカー L の電源OFF

ステップ3: 配信機材
  □ 配信用オーディオインターフェースの電源OFF
  □ 配信用PCの電源OFF

ステップ4: ミキサー
  □ ファンタム電源をOFF（※後述）
  □ 少し待つ（10秒）
  □ デジタルミキサーの電源OFF

ステップ5: 音源（ステージ側）
  □ キーボード①②の電源OFF
  □ エレキギターのアンプ電源OFF
  □ ベースのアンプ/DI電源OFF
  □ iPhone・PCを取り外し
```

> **中村メモ:** チェックリストを印刷してミキサー横に貼っておくのがおすすめ。慣れるまでは毎回確認してOKです。プロでもチェックリストは使います。

## 4. ファンタム電源って何？（10分）

ワイヤレスマイクやダイナミックマイクは電池や自分の仕組みで動きますが、**コンデンサーマイク** という種類のマイクは外部から電気をもらう必要があります。

```
【ダイナミックマイク】
  マイク ──音声信号──→ ミキサー    ← 電気は不要
  （自力で動く）

【コンデンサーマイク】
  マイク ←─48V電源──── ミキサー    ← ミキサーから電気を送る
         ──音声信号──→              （これがファンタム電源）
```

### ファンタム電源（+48V）のルール

```
┌──────────────────────────────────────────────┐
│           ファンタム電源の ON/OFF              │
│                                                │
│  ON: ミキサー起動後、マスターフェーダーを      │
│      下げた状態でファンタム電源をON             │
│      → 10秒待ってからフェーダーを上げる        │
│                                                │
│  OFF: フェーダーを下げてから                   │
│       ファンタム電源をOFF                      │
│       → 10秒待ってからミキサーの電源OFF        │
└──────────────────────────────────────────────┘
```

### なぜ待つのか？

ファンタム電源のON/OFF時にもポップノイズが出ます。フェーダーを下げておけば、その信号がスピーカーに届かないので安全です。

### うちの教会でファンタム電源が必要なマイク

```
ファンタム電源が必要:
  - ドラム用オーバーヘッドマイク（コンデンサー）
  - その他コンデンサーマイクがあれば

ファンタム電源が不要:
  - ワイヤレスボーカルマイク × 4（レシーバーで受信）
  - 説教用マイク × 2（ワイヤレス）
  - キーボード（ライン入力）
  - ギター・ベース（DI経由）
```

> **中村メモ:** ファンタム電源を入れたままケーブルを抜き差しすると「バチッ！」と大きなノイズが出ます。ケーブル作業は必ずファンタムOFFの状態で。これは本当に大事。

## 5. 順番を間違えたらどうなる？（10分）

実際に起きる問題を知っておきましょう。

### ケース1: スピーカーONの状態でミキサーをON

```
結果: 「バンッ！！」

  ミキサー起動 → ポップノイズ発生 → スピーカーから大音量

  被害:
  ├── スピーカーのドライバー破損（修理 3〜10万円）
  ├── 会衆がびっくりして耳が痛い
  └── PA担当への信頼が落ちる
```

### ケース2: ミキサーONの状態でスピーカーをOFF（電源OFF時に逆順忘れ）

```
結果: スピーカー側はあまり問題ないが...

  でもその後ミキサーをOFFにすると、
  まだONのスピーカーがあった場合にノイズが出る

  → 「あれ、もう切ったはず」と思って油断するのが危険
```

### ケース3: ファンタム電源ONのままケーブルを抜く

```
結果: 「バチッ！」

  48Vの電圧がかかった状態でケーブルを抜く
  → 接点でスパーク → 大きなポップノイズ

  被害:
  ├── マイクが壊れる可能性
  ├── スピーカーに大きなノイズ
  └── ケーブルの接点が傷む
```

### 合言葉

```
┌────────────────────────────────────────┐
│                                          │
│   ON は「前から後ろ」（川上から川下へ）   │
│   OFF は「後ろから前」（川下から川上へ）  │
│                                          │
│   スピーカーは「最後にON、最初にOFF」     │
│                                          │
│   迷ったらフェーダー全部下げる！          │
│                                          │
└────────────────────────────────────────┘
```

> **中村メモ:** もし順番を間違えて「バンッ」と鳴ってしまったら、まず落ち着いて。すぐにスピーカーの電源を切って、正しい順番でやり直せばOKです。焦ってさらに間違えるのが一番危ない。

## 実践ドリル（10分）

### ドリル1: 順番並べ替え

以下の機材を電源ONの正しい順番に並べ替えてください。

```
A. メインスピーカー L/R
B. デジタルミキサー
C. キーボード①②
D. フロアモニター
E. サブウーファー
F. 配信用PC

正しい順番:
→ [  ] → [  ] → [  ] → [  ] → [  ] → [  ]
```

答え: C → B → F → A → E → D （音源 → ミキサー → 配信 → スピーカー類）

### ドリル2: トラブル対応

「礼拝前にスピーカーの電源を先に入れてしまった！ミキサーはまだOFF。どうする？」

```
答え:
1. スピーカーの電源をOFFに戻す
2. 正しい順番で最初からやり直す
   （音源 → ミキサー → スピーカー）
3. 焦らない。間に合う。
```

### ドリル3: 実際にやってみよう

教会の機材で、チェックリストを見ながら以下を実践:

1. 全機材の電源がOFFであることを確認
2. チェックリストに従って電源ON
3. 全機材が正常に起動したことを確認
4. チェックリストに従って電源OFF
5. もう一度、今度はチェックリストを見ないでやってみる

## 教会での応用

### 礼拝前の準備タイムライン

```
礼拝30分前:
  └── 電源ON開始（チェックリスト通りに）

礼拝25分前:
  └── ミキサー起動完了、前回の設定確認

礼拝20分前:
  └── スピーカーON、BGM流し始め
      （会衆が集まり始める前に準備完了）

礼拝終了後:
  └── 最後の人が退出してから電源OFF開始
      （突然音が消えると寂しい雰囲気になる）
```

### よくある教会あるある

- **「あ、キーボード忘れた！」** → ミキサーのそのチャンネルのフェーダーを下げてから、キーボードの電源をON。フェーダーを下げておけばポップノイズはスピーカーに行かない
- **「途中で機材を追加したい」** → フェーダー下げる → 接続 → 電源ON → ゆっくりフェーダー上げる
- **「停電した！」** → 電気が戻ったら、スピーカーの電源スイッチをOFFにしてから他の機材の復旧を待つ。全部立ち上がったら最後にスピーカーON

## 1. EQ って何？（5分）

### 音の「成分」を調整する道具

音は低い音から高い音まで、いろんな「成分」が混ざっています。EQは特定の成分だけを大きくしたり小さくしたりできます。

```
【料理の例え】

  味噌汁の味が変だとき:
    塩辛い → 塩を減らす
    甘みが足りない → 味噌を足す
    コクがない → 出汁を足す

  音がおかしいとき:
    こもってる → 低い成分を減らす
    キンキンする → 高い成分を減らす
    声が通らない → 中域の成分を足す

  EQ = 音の味付けを調整する調味料
```

### EQのつまみ

```
ミキサーのEQセクション（基本的な3バンド）:

  HIGH  ──(つまみ)──  高音の調整
  MID   ──(つまみ)──  中音の調整
  LOW   ──(つまみ)──  低音の調整

  右に回す = その帯域を大きく（ブースト）
  左に回す = その帯域を小さく（カット）
  真ん中   = 何もしない（フラット）
```

> **中村メモ:** デジタルミキサーだともっと細かく調整できますが、基本は「どの高さの音を上げ下げするか」です。まずはHIGH/MID/LOWの3つで考えましょう。

## 2. 周波数帯域を知ろう（15分）

音の高さは **Hz（ヘルツ）** という単位で表します。数字が小さいほど低い音、大きいほど高い音です。

### 7つの帯域

```
┌──────────────────────────────────────────────────────────┐
│                    周波数帯域マップ                        │
│                                                          │
│  帯域        範囲            特徴            キーワード    │
│ ─────────────────────────────────────────────────────── │
│  Sub         20-60Hz         地響き          ズズズ       │
│  Low         60-250Hz        温かみ・太さ    ブーン       │
│  Low-Mid     250-500Hz       こもり・膨らみ  モワモワ     │
│  Mid         500Hz-2kHz      存在感・鼻声    ンー        │
│  Upper-Mid   2-4kHz          アタック・鋭さ  カッ        │
│  High        4-8kHz          明るさ・刺さり  シャリシャリ │
│  Air         8-20kHz         空気感・息      シュー       │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 各帯域の詳しい解説

```
■ Sub（サブ）20-60Hz — 「地響き」

  感じる音:  地鳴り、雷の低音、キックドラムの「ドスッ」
  多すぎると: お腹に響くような不快な振動、スピーカーが壊れる
  足りないと: 音が軽い、迫力がない
  教会で:    キックドラムとベースだけがこの帯域を使う
             → 他の楽器はここをカットしてOK

────────────────────────────────────────

■ Low（ロー）60-250Hz — 「温かみ・太さ」

  感じる音:  ベースの音程、キーボードの左手、男性の声の低い部分
  多すぎると: 「ブーン」とこもる。全体が濁る
  足りないと: 音が細い、貧弱に聞こえる
  教会で:    ベース・キックに温かみを持たせたい帯域
             → ボーカルやギターでは少しカットすると抜けが良くなる

────────────────────────────────────────

■ Low-Mid（ローミッド）250-500Hz — 「こもり・膨らみ」

  感じる音:  段ボール箱を叩いたような「ボコッ」という響き
  多すぎると: 「モワモワ」「ボコボコ」こもった感じ ← 教会でよくある問題！
  足りないと: 音が薄い
  教会で:    多くの楽器が重なるゾーン
             → ここが溜まると全体が「モワッ」とする
             → 各楽器で少しずつカットすると全体がスッキリ

────────────────────────────────────────

■ Mid（ミッド）500Hz-2kHz — 「存在感・鼻声」

  感じる音:  人の声の中心、ギターのコード感
  多すぎると: 鼻をつまんだような「ンー」という音
  足りないと: 声が奥に引っ込む、歌詞が聞こえない
  教会で:    ボーカルの「言葉の明瞭さ」に直結
             → ボーカルではこの帯域を大切にする
             → 他の楽器はここを少し譲る

────────────────────────────────────────

■ Upper-Mid（アッパーミッド）2-4kHz — 「アタック・鋭さ」

  感じる音:  ピックで弦を弾く「カッ」、声の「パ行」「カ行」
  多すぎると: 耳が疲れる。長時間聴くとキツい
  足りないと: 音にメリハリがない、ぼんやりする
  教会で:    ボーカルの明瞭さ、ギターのカッティング感
             → ちょうどいい量が大事。上げすぎ注意

────────────────────────────────────────

■ High（ハイ）4-8kHz — 「明るさ・歯擦音」

  感じる音:  「サ」「シ」「ス」の子音、シンバルの輝き
  多すぎると: 「サ行」が「ツァ行」に聞こえる（歯擦音・シビランス）
  足りないと: 音がくぐもる、暗い印象
  教会で:    ボーカルの子音を適度に出すと歌詞が聞き取りやすい
             → 出しすぎると耳に刺さる

────────────────────────────────────────

■ Air（エアー）8-20kHz — 「空気感・息」

  感じる音:  録音の「生っぽさ」、ボーカルの息づかい
  多すぎると: 「シュー」「サーッ」というノイズが目立つ
  足りないと: あまり気にならない（なくても困らない帯域）
  教会で:    コンデンサーマイクでは自然に収録される
             → あまりいじらなくてOK
```

### 楽器ごとの周波数マップ

```
Hz:  20   60   250  500   1k   2k   4k   8k   20k
     |    |    |    |     |    |    |    |    |
     |    |    |    |     |    |    |    |    |
キック ▓▓▓▓▓▓▓▓░░░░░░░░░                         ドスッ
ベース      ▓▓▓▓▓▓▓▓▓▓▓▓░░░                      ブンブン
ギター           ░░▓▓▓▓▓▓▓▓▓▓░░░                 ジャカジャカ
キーボ      ░░▓▓▓▓▓▓▓▓▓▓▓▓▓▓░░░░░                幅広い
ボーカル         ░░▓▓▓▓▓▓▓▓▓▓▓▓░░                アーアー

     |    |    |    |     |    |    |    |    |

```

> **中村メモ:** このマップを見ると、250Hz〜1kHz あたりに多くの楽器が集中しているのがわかりますよね。ここが「渋滞」するんです。だから各楽器の「いらない部分」をカットして、道を空けてあげることが大事なんです。

## 3. EQ の種類（10分）

### パラメトリックEQ

デジタルミキサーで使う主力のEQ。3つの要素を調整できます。

```
パラメトリックEQ の3つのパラメーター:

  1. 周波数（Frequency / Freq）
     → どの高さの音を調整するか

  2. ゲイン（Gain）
     → どれくらい上げる/下げるか（dB）

  3. Q（幅）
     → 調整する範囲の広さ

  Qが狭い場合:          Qが広い場合:
       ╱╲                  ╱────╲
      ╱  ╲                ╱      ╲
  ───╱    ╲───        ───╱        ╲───
  （ピンポイント）      （広い範囲に影響）

  → 問題をピンポイントで直す時: Qを狭く
  → 全体の印象を変えたい時:     Qを広く
```

### シェルフEQ

特定の周波数 **以上** または **以下** を全部上げ下げする。

```
ハイシェルフ（高音域を持ち上げ/カット）:

  上げた場合:                 下げた場合:
         ┌──────────             ────────────
         │                               │
  ───────┘                               └──────
      ↑ この周波数から上を全部         この周波数から上を全部下げ

ローシェルフ（低音域を持ち上げ/カット）:

  上げた場合:                 下げた場合:
  ──────────┐                ────────────
            │                │
            └────────    ────┘
            この周波数から下を全部     この周波数から下を全部下げ
```

### ハイパスフィルター（HPF）— 最重要ツール

```
┌──────────────────────────────────────────────────┐
│                                                    │
│  ハイパスフィルター = PA で一番よく使うツール        │
│                                                    │
│  別名: ローカットフィルター                         │
│  意味: 低い音をカット（= 高い音だけ通す）           │
│                                                    │
└──────────────────────────────────────────────────┘

動作:
                          HPF ON
  ────────────────    →      ╱──────────
                             ╱
                            ╱
                        ───╱
                        ↑
                     カットオフ周波数
                     （ここから下をカット）

例: ボーカルマイクに 80Hz のHPFをかける
  → 80Hz以下の低音（足音、空調ノイズ、振動）がカットされる
  → ボーカルの声には影響しない（声は80Hz以上）
  → クリアな音になる！
```

### なぜHPFが大事なのか

```
HPFなし:
  ボーカルマイク → 声 + 足踏みの振動 + 空調の低音ノイズ + ステージの振動
                   ↑                 ↑
                 欲しい音          いらない音

HPFあり（80Hzカット）:
  ボーカルマイク → 声
                   ↑
                 欲しい音だけ！

HPF を入れるべきチャンネル:
  ├── ボーカルマイク:  80-120Hz
  ├── アコギ:          80-100Hz
  ├── エレキギター:    80-100Hz
  ├── キーボード:      40-80Hz（低音も使うので控えめ）
  ├── スネア:          80-100Hz
  ├── ハイハット:      200-300Hz
  ├── オーバーヘッド:  100-200Hz
  │
  HPF を入れないチャンネル:
  ├── キックドラム:    ×（低音が命）
  └── ベース:          ×（低音が命）
```

> **中村メモ:** HPFは地味だけど効果絶大。HPFを全チャンネルに適切にかけるだけで、ミックスの半分は完成したと言ってもいい。低音のゴミがなくなるだけで、全体が驚くほどスッキリします。騙されたと思ってやってみてください。

## 4. 「カット優先」の考え方（5分）

### Cut before Boost（ブーストの前にカットする）

```
❌ よくある間違い:
  「ボーカルが聞こえない → 中音域を上げよう！」
  → 上げると他の帯域とのバランスが崩れる
  → 他の楽器も上げたくなる → 全部上がる → 何も変わらない（ただ音量が上がっただけ）

✅ 正しいアプローチ:
  「ボーカルが聞こえない → 他の楽器の中音域をカットしよう」
  → ボーカルの居場所が空く → ボーカルが自然に聞こえる
  → 全体の音量は変わらない → スピーカーにも優しい
```

```
【座席の例え】

  映画館で前の人の頭が邪魔で画面が見えない

  ❌ 自分の座席を高くする（ブースト）
     → 後ろの人も見えなくなる → 全員が高くする → 意味なし

  ✅ 前の人に少し低くしてもらう（カット）
     → 自分は見えるようになる → 全体のバランスは保たれる
```

> **中村メモ:** 「足す前に引く」。これはプロのエンジニアが口を揃えて言うことです。EQを触る時はまずカットから考える癖をつけてください。カットだけで解決することが本当に多いですよ。

## 5. 教会の楽器別 EQ ガイド（10分）

### ボーカル

```
ボーカル EQ スタートポイント:

  HPF:       80-100Hz（必須）
  Low-Mid:   300Hz あたりを少しカット（-2〜-3dB）→ こもり除去
  Mid:       基本いじらない（声の芯）
  Upper-Mid: 3kHz あたりに少し存在感があると歌詞が聴きやすい
  High:      歯擦音がキツければ 6-8kHz を少しカット

  HPF    Low-Mid   Mid    U-Mid    High
   ╱     ╲
  ╱       ╲  ──────── ╱╲ ───────────
              ↓       ↑
           こもりカット 明瞭さ
```

### キーボード

```
キーボード EQ スタートポイント:

  HPF:       40-60Hz（低音パートがあるなら控えめに）
  Low:       膨らみすぎたら 100-200Hz をカット
  Low-Mid:   ボーカルと被る 300-500Hz を少しカット
  High:      デジタルの硬さがあれば 5kHz 以上を少しカット

  ※ キーボードの音色（ピアノ、パッド、オルガン等）で調整は変わる
  ※ パッド系はLow-Midをしっかりカットすると他の楽器を邪魔しない
```

### アコースティックギター

```
アコースティックギター EQ スタートポイント:

  HPF:       80-100Hz（必須。ボディの共鳴ノイズをカット）
  Low-Mid:   200-400Hz を少しカット（ボコボコ感の除去）
  Mid:       基本そのまま
  Upper-Mid: 2-5kHz でピッキングのアタック感を調整
  High:      弦のキラキラ感。出しすぎるとシャリシャリ

  ポイント: アコギのボディをポンポン叩いた時の「ボコッ」という
           周波数を見つけてカットすると劇的に良くなる
```

### エレキギター

```
エレキギター EQ スタートポイント:

  HPF:       80-100Hz（アンプの低音ノイズをカット）
  Low:       ベースとの棲み分けのため 100-200Hz を控えめに
  Low-Mid:   必要に応じて調整（ギターの太さ）
  Mid:       ギターの核心部分。慎重に調整
  Upper-Mid: 2-4kHz でエッジ感。キツすぎたらカット
```

### ベース

```
ベース EQ スタートポイント:

  HPF:       使わない（低音が命）
  Sub:       超低音が出すぎなら 30-40Hz をカット
  Low:       60-100Hz がベースの太さ
  Low-Mid:   200-400Hz のこもりをカット
  Mid:       700Hz-1kHz に「ブリブリ」感。指弾きの音色

  ポイント: キックドラムと住み分ける
           キックが 60Hz メインなら、ベースは 80-100Hz を強調
           → 同じ帯域で喧嘩しない
```

### ドラム

```
キックドラム:
  HPF:    使わない
  Sub:    50-60Hz のアタック感
  Low:    100Hz の胴鳴り。多すぎたらカット
  Mid:    400-800Hz の「段ボール感」をカット
  U-Mid:  3-5kHz の「ペチッ」というアタック

スネアドラム:
  HPF:    80-100Hz
  Low:    200Hz の厚み
  Mid:    ボックス感（400-800Hz）をカット
  U-Mid:  2-4kHz のアタック（パリッ）
  High:   スナッピー（針金）の「ジャリ」感

ハイハット / シンバル:
  HPF:    200-400Hz（低音は不要）
  High:   シャリシャリ感。キツすぎたらカット
```

> **中村メモ:** これはあくまで「スタートポイント」です。実際の音を聴いて調整してください。同じ楽器でも機種や演奏者によって全然違います。大事なのは「何を聴くべきか」を知っていること。そうすれば「ここが変だ」と気づけます。

## 6. EQ 実践テクニック: スイープ（5分）

問題の周波数を見つける方法です。

```
【スイープのやり方】

  1. パラメトリックEQで1バンドを選ぶ
  2. ゲインを +8〜+10dB にブースト（わざと上げる）
  3. Qを狭く設定（ピンポイント）
  4. 周波数をゆっくり動かす（低い方から高い方へ）

  ゆっくり動かしていくと...
  「あ、ここが嫌な音だ！」という周波数が見つかる

  5. その周波数でゲインを -3〜-6dB にカット
  6. ブースト前より良い音になったか確認

  ┌─────────────────────────────────────┐
  │ 注意:                                 │
  │ スイープ中はヘッドフォンで確認する。   │
  │ スピーカーから大音量のブーストが        │
  │ 出ると耳に悪いし迷惑です。             │
  └─────────────────────────────────────┘
```

## 実践ドリル（10分）

### ドリル1: 周波数帯域クイズ

以下の問題を解いてみましょう。

```
Q1: 「ボーカルがこもって聞こえる」
    → どの帯域をカットする？

Q2: 「キックドラムの低音がブーンと響きすぎる」
    → どの帯域をカットする？

Q3: 「アコギの音がボコボコする」
    → どの帯域をカットする？

Q4: 「ボーカルの歌詞が聞き取りにくい」
    → まず何をする？（2つ）
```

```
答え:
Q1: Low-Mid（250-500Hz）をカット
Q2: Low（60-250Hz）をカット。またはSub（60Hz以下）をカット
Q3: Low-Mid（200-400Hz）をカット
Q4: 1. 他の楽器のMid（500Hz-2kHz）をカットしてボーカルの居場所を空ける
    2. ボーカルの Upper-Mid（2-4kHz）を少しブースト
```

### ドリル2: HPF 設定練習

教会のミキサーで以下のチャンネルにHPFを設定してみましょう。

```
チャンネル         HPF設定
──────────────────────────
ボーカル ch1-4     [   ] Hz
説教マイク ch5-6   [   ] Hz
キーボード ch7-10  [   ] Hz
アコギ ch          [   ] Hz
エレキ ch          [   ] Hz
ベース ch          HPF [ON / OFF]
キック ch          HPF [ON / OFF]
```

### ドリル3: 実機でEQ体験

1. ボーカルマイクに向かって話す
2. HPFをON/OFF切り替えて違いを聴く
3. Low-Midを大きくブーストして「こもった音」を体験
4. 元に戻してからLow-Midをカットして「スッキリした音」を体験
5. Upper-Midをブーストして「キンキンした音」を体験
6. フラットに戻す

## 教会での応用

### EQ設定の保存

```
デジタルミキサーの強み:

  1回良いEQを作ったら保存できる！

  保存するもの:
  ├── チャンネルごとのEQプリセット
  │   （「ボーカル用」「キーボード用」等）
  ├── シーン全体の保存
  │   （礼拝用、特別集会用 等）
  └── ライブラリ機能（ミキサーによる）

  次の礼拝では:
  1. 保存したシーンを呼び出す
  2. サウンドチェックで微調整
  3. → 毎回ゼロから作る必要なし！
```

### よくある教会の音の問題と対処

```
問題: 「全体的にモワッとしている」
原因: Low-Mid（250-500Hz）の渋滞
対処: 各楽器の Low-Mid を少しずつカット
      + 全チャンネルに HPF

問題: 「ボーカルの歌詞が聞こえない」
原因: Mid帯域で楽器がボーカルに被っている
対処: キーボード・ギターの Mid をカット
      + ボーカルの Upper-Mid を少しブースト

問題: 「長時間聴いていると耳が疲れる」
原因: Upper-Mid（2-4kHz）が出すぎ
対処: エレキギター・シンバルの Upper-Mid を少しカット

問題: 「ブーンという低音が鳴り続けている」
原因: Sub帯域のかぶりまたは部屋の共鳴
対処: HPF の見直し + メインEQで問題の周波数をカット
```

### EQ の心構え

```
┌─────────────────────────────────────────────────┐
│                                                   │
│  1. まず HPF を入れる（キック・ベース以外）       │
│  2. カットから始める（ブーストは最後の手段）       │
│  3. 少しずつ調整する（一気に大きくいじらない）     │
│  4. 必ず音楽の中で聴く（ソロで良くても混ぜてダメ   │
│     なことがある）                                │
│  5. 耳が疲れたら休憩する（5分休むだけで判断力回復）│
│                                                   │
└─────────────────────────────────────────────────┘
```

> **中村メモ:** EQは「直す」道具であって「飾る」道具ではありません。まずフラットで聴いて、「ここが問題だ」と感じたところだけ直す。何も問題なければEQはフラットでOK。触らない勇気も大事です。

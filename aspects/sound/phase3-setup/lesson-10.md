# Lesson 10: ゲインストラクチャー --- 適切な音量の設定法

> **Phase 3 / L10: 各段階で正しい音量を設定し、ノイズも歪みもない綺麗な信号を作れるようになる**
> **担当:** 中村響（リードインストラクター）
> **所要時間:** 約60分
> **前提:** Phase 1-2 完了

---

## 今日のゴール
- ゲインストラクチャーの考え方を理解する
- PFL（プリフェーダーリッスン）を使ったゲイン設定ができる
- ボーカルマイク・キーボード・ベースのゲインを実際に設定する

---

前回、電源の順番を覚えましたね。無事にスピーカーを壊さずに起動できたら、次は「どのくらいの音量で信号を流すか」です。

音が小さすぎると「サーッ」というノイズが目立つ。大きすぎると「バリバリッ」と歪む。ちょうどいいところに合わせるのが **ゲインストラクチャー** です。

**理由がわかれば忘れない。** なぜ「ちょうどいい」が大事なのか、交通の例えで理解していきましょう。

---

## 1. ゲインストラクチャーとは？（10分）

### 信号が通る「道路」のイメージ

音の信号は、マイクからスピーカーまで何段階も通過します。それぞれの段階を「道路」に例えてみましょう。

```
【交通の例え】

信号（音） = 車
各段階     = 道路
ゲイン     = 道路の幅

道路が狭すぎる（ゲイン低すぎ）:
  ┌──┐
  │車│← ギリギリ。周りのゴミ（ノイズ）も一緒に通る
  └──┘
  ════  ← 狭い道路

道路がちょうどいい（ゲイン適正）:
  ┌──┐
  │車│← 余裕を持って通れる。ゴミ（ノイズ）は目立たない
  └──┘
  ════════  ← 適切な幅の道路

道路が狭いのに車が大きい（ゲイン高すぎ）:
  ┌────┐
  │ 車 │← はみ出す！＝ 歪み（クリッピング）
  └────┘
  ════  ← 道路からはみ出している
```

### 音量が小さすぎる場合の問題

```
信号レベル:  ▓░░░░░░░░░░░░░░░░░░░  （すごく小さい）
ノイズ:      ░░░▒░░▒░░░▒░░░░▒░░░░  （常にある）

→ 後段で音量を上げると、ノイズも一緒に増幅される

増幅後:      ▓▓▓▒▓▓▒▓▓▓▒▓▓▓▓▒▓▓▓▓
              ↑         ↑
           信号      ノイズも大きくなった！

結果: 「サーッ」「ジーッ」というノイズが目立つ
```

### 音量が大きすぎる場合の問題

```
信号レベル:  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓██  （大きすぎ）
                                  ↑↑
                                クリップ！

→ 信号の波形が上下を切り取られる（クリッピング）

正常な波形:     ╱╲    ╱╲
              ╱    ╲╱    ╲

クリップ波形:  ──╲    ╱──
                  ╲╱
              ↑ 頭が潰れている

結果: 「バリバリ」「ジリジリ」という歪んだ音
```

> **中村メモ:** ゲインが適切だと、音が「透明」に聞こえます。プロの現場と教会の違いは、実は機材の値段じゃなくてゲインの合わせ方だったりします。

## 2. メーターの見方（5分）

ミキサーには信号の大きさを表示する **メーター** があります。

```
【メーターの見方】

  dB
  ─── CLIP ← 絶対ここまで行かせない！赤ランプ
  +6  ████  ← 赤ゾーン：歪みの危険
  +3  ████
   0  ████  ← ── ここが「0dB」基準
  -3  ████
  -6  ████  ← 黄色ゾーン：ちょうどいい
  -9  ████     ↑
  -12 ████  ← 緑ゾーン：通常ここを狙う
  -15 ████
  -18 ████  ← このあたりが目標
  -24 ████
  -30 ████
  -40 ██    ← 小さすぎ
  -60 █
```

### 信号レベルの目安

```
┌─────────────────────────────────────────┐
│ 赤 （0dB以上）  : 歪む！すぐゲインを下げる │
│ 黄 （-6〜0dB）  : 注意。ピークで赤に入る    │
│ 緑 （-18〜-6dB）: ここが目標レンジ         │
│ 下 （-18dB以下）: 小さい。ゲインを上げる    │
└─────────────────────────────────────────┘

目標: 通常の演奏で -18〜-12dB あたり
      一番大きい瞬間でも -6dB を超えない
      → 0dB まで余裕（ヘッドルーム）を残す
```

> **中村メモ:** この「余裕（ヘッドルーム）」がめちゃくちゃ大事。礼拝中に賛美が盛り上がって急に音量が上がっても、余裕があれば歪まない。ギリギリで設定すると、盛り上がった瞬間に「バリッ」って歪みます。

## 3. PFL（プリフェーダーリッスン）とは？（10分）

### PFL って何？

PFL = **Pre Fader Listen**（プリ・フェーダー・リッスン）

フェーダーの **前** の信号を聴くための機能です。

```
【信号の流れとPFLの位置】

マイク → ゲイン → [PFL] → フェーダー → マスター → スピーカー
                    ↑
                ここで聴く！
                （フェーダーの位置に関係なく、
                  ゲインだけの音量が確認できる）
```

### なぜPFLを使うのか？

フェーダーを上げた状態でゲインを調整すると、その音がスピーカーから出てしまいます。サウンドチェック中にいきなり大音量が出たら迷惑ですよね。

PFLを使えば、**ヘッドフォンだけで** ゲインの調整ができます。

```
PFLボタンを押すと:
  - そのチャンネルの信号がヘッドフォンに送られる
  - メーターにそのチャンネルのレベルが表示される
  - メインスピーカーには影響しない

使い方:
  1. ヘッドフォンをつける
  2. チャンネルのPFL（またはSOLO）ボタンを押す
  3. ゲインつまみを回して、メーターが適正範囲に入るよう調整
  4. 確認が終わったらPFLボタンを解除
```

> **中村メモ:** デジタルミキサーでは「SOLO」と表記されていることもあります。機能はほぼ同じ。PFLボタンを押し忘れたまま操作すると、意図しない音がスピーカーから出ることがあるので、**使い終わったら必ず解除** してください。

## 4. ユニティゲインとは？（5分）

ミキサーのフェーダーには基準の位置があります。それが **ユニティゲイン（0dB / Unity）** です。

```
【フェーダーの位置】

  +10 ─── 最大（ほとんど使わない）
   .
   .
   0  ─── ユニティゲイン（▶ ここが基準）
   .       「増幅も減衰もしない」位置
   .       多くのミキサーで太い線やマークがある
  -10
   .
  -20
   .
  -∞ ─── 完全に音を消す（一番下）
```

### なぜユニティゲインが大事なのか

```
考え方:

  ゲイン = 信号の「入口」の音量調整
  フェーダー = 信号の「出口」の音量調整

  理想:
    ゲインで適切なレベルに合わせる
    → フェーダーはユニティ（0dB）付近で運用
    → 上げ下げの余裕が上にも下にもある

  ダメな例:
    ゲインが低すぎてフェーダーを目一杯上げている
    → これ以上音量を上げられない
    → ノイズも増幅されている
```

> **中村メモ:** フェーダーをユニティ付近で使えるようにゲインを合わせる。これがゲインストラクチャーの基本です。フェーダーが極端に上がっていたり下がっていたりしたら、ゲインの見直しサインです。

## 5. ゲイン設定の手順（15分）

### ステップバイステップ

```
=== ゲイン設定手順 ===

準備:
  □ ヘッドフォンをミキサーに接続
  □ すべてのフェーダーを一番下に下げる
  □ マスターフェーダーも下げる

チャンネルごとに:
  1. □ そのチャンネルのPFL/SOLOボタンを押す
  2. □ ミュージシャンに「普通の音量で演奏してください」とお願い
  3. □ ゲインつまみをゆっくり上げる
  4. □ メーターを見て -18〜-12dB あたりを狙う
  5. □ 一番大きく演奏してもらい、-6dBを超えないか確認
  6. □ PFL/SOLOボタンを解除
  7. □ 次のチャンネルへ

全チャンネル完了後:
  □ マスターフェーダーをユニティ（0dB）に上げる
  □ 各チャンネルフェーダーをゆっくり上げてバランスを取る
```

### ボーカルマイクのゲイン設定

```
ボーカルマイク（ワイヤレス × 4）

  1. PFLを押す
  2. 「マイクテスト、1、2」と普通に話してもらう
  3. メーター: -18〜-12dB を目安にゲイン調整
  4. 「一番大きく歌うところ歌ってみてください」
  5. ピークが -6dB を超えないか確認

  注意:
  - ワイヤレスマイクは受信機のレベルもある
  - 受信機の出力を先に適切にしてからミキサーのゲインを合わせる

  メーターの理想:
  ├─ 話し声:    ████████████░░░░░░░░  (-18dB)
  ├─ 通常の歌:  █████████████████░░░  (-12dB)
  └─ 最大音量:  ██████████████████░░  (-6dB)
                                    ↑
                               ここに余裕
```

### キーボードのゲイン設定

```
キーボード①②（ステレオ = 2ch ずつ使用）

  1. PFLを押す（L/R 両方のチャンネル）
  2. 「普通の音量で弾いてください」
  3. L/R のメーターがほぼ同じレベルか確認
  4. -18〜-12dB を目安にゲイン調整
  5. 「一番強く弾くフレーズをお願いします」
  6. ピーク確認

  注意:
  - キーボード本体のボリュームが中途半端だと
    ゲインを上げすぎてノイズが増える
  - キーボード側のボリュームを 70〜80% に設定してから
    ミキサー側のゲインを合わせる

  ┌─────────────────────────────────────┐
  │ キーボード本体: 70-80%               │
  │       ↓                              │
  │ ミキサーゲイン: メーター -18〜-12dB    │
  │       ↓                              │
  │ フェーダー: ユニティ（0dB）付近       │
  └─────────────────────────────────────┘
```

### ベースギターのゲイン設定

```
ベース（DI経由）

  1. PFLを押す
  2. 「いつもの感じで弾いてください」
  3. メーター確認、-18〜-12dBを狙う
  4. 「スラップ（パシパシ叩く奏法）ありますか？」
     → ある場合はピークがかなり高くなるので注意
  5. 一番強い部分でも -6dB 以下に収める

  注意:
  - ベースは低音のエネルギーが大きい
  - メーターが一見普通でも、低音でスピーカーが飛ぶことがある
  - ハイパスフィルター（Lesson 12で学ぶ）と組み合わせて管理
```

> **中村メモ:** ゲイン設定は「一番大きい音で -6dB を超えない」が鉄則。教会の賛美は曲によって音量差が大きいので、ヘッドルームは多めに取りましょう。

## 6. よくある失敗パターン（5分）

```
❌ 失敗1: ゲインを上げすぎてフェーダーを下げて調整
  ゲイン ████████████████████ MAX近く
  フェーダー ██░░░░░░░░░░░  かなり下

  問題: ゲイン段で歪んでいる。フェーダーで音量を下げても
        歪みは消えない

❌ 失敗2: ゲインが低すぎてフェーダーを上げて調整
  ゲイン ███░░░░░░░░░░░░░░  かなり低い
  フェーダー ██████████████████ MAX近く

  問題: ノイズも一緒に増幅。「サーッ」という音が目立つ

✅ 正解: ゲインを適切に合わせてフェーダーはユニティ付近
  ゲイン ██████████░░░░░░░░  適切
  フェーダー ██████████░░░░  ユニティ付近

  結果: 綺麗な音、上下に調整の余裕あり
```

## 実践ドリル（10分）

### ドリル1: メーター読み取り

以下のメーター表示を見て、ゲインを上げるべきか下げるべきか判断してください。

```
A) ████████████████████ (+3dB)  → ゲインを [上げる / 下げる]
B) ████░░░░░░░░░░░░░░░ (-30dB) → ゲインを [上げる / 下げる]
C) ████████████░░░░░░░ (-12dB) → ゲイン [そのまま / 調整必要]
```

答え: A) 下げる B) 上げる C) そのまま（適正範囲）

### ドリル2: 実機でゲイン設定

教会のミキサーで以下を実践:

1. ヘッドフォンを接続
2. ボーカルマイク1本のゲインをPFLで設定
3. キーボード1台（ステレオ）のゲインを設定
4. 設定後、フェーダーをユニティに上げてスピーカーから音を確認
5. 綺麗に聞こえるか？ノイズはないか？歪みはないか？

### ドリル3: わざと失敗してみよう

先生と一緒に、安全な音量で以下を体験:

1. ゲインを極端に上げるとどんな音になるか聴く
2. ゲインを極端に下げてフェーダーを上げるとどうなるか聴く
3. 適切なゲインに戻して違いを感じる

## 教会での応用

### 礼拝中のゲイン管理

```
リハーサル時:
  └── 全チャンネルのゲインを設定
      （ミュージシャンがいる間に！）

礼拝本番中:
  ├── ゲインは基本いじらない
  │   （ゲインを触ると音質が変わる）
  ├── 音量調整はフェーダーで行う
  │   （フェーダーは音量だけ変える）
  └── メーターが赤に入り続ける場合のみゲイン微調整

次の週:
  └── 前回の設定を呼び出して微調整
      （デジタルミキサーなら保存できる）
```

### ミュージシャンへの声かけ例

```
「普通に歌ってもらえますか？
 ......OK、もうちょっと大きく。
 ......はい、それくらいで大丈夫です。
 じゃあ一番盛り上がるところ歌ってみてください。
 ......完璧です、ありがとうございます！」
```

> **中村メモ:** ミュージシャンは緊張すると本番でリハーサルより大きく演奏しがちです。だからヘッドルームは多めに取る。「リハで-12dBなら本番で-6dBくらい行くかも」と想定しておくと安心です。

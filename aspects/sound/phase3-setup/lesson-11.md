# Lesson 11: サウンドチェック --- 一人ずつ確認する手順と理由

> **Phase 3 / L11: サウンドチェックの正しい順番を覚えて、限られた時間で効率よく音を仕上げられるようになる**
> **担当:** 中村響（リードインストラクター）
> **所要時間:** 約60分
> **前提:** Phase 1-2 完了

---

## 今日のゴール
- ラインチェックとサウンドチェックの違いを理解する
- サウンドチェックの順番と、なぜその順番なのかを理解する
- ミュージシャンとのコミュニケーション方法を覚える

---

電源の順番、ゲインの合わせ方——ここまできたら、いよいよ音を出します。

でも、全員が同時に音を出したらどうなるか想像してみてください。キーボードがジャカジャカ弾いてる横でドラムがドコドコ叩いて、ボーカルが歌い始めて......何が何だかわかりませんよね。

だから、**一人ずつ確認する**。これがサウンドチェックの基本です。

**理由がわかれば忘れない。** なぜこの順番なのか、建物を建てるのと同じ考え方で理解していきましょう。

## 1. ラインチェックとサウンドチェック（5分）

### 二つは別のもの

```
┌──────────────────────────────────────────────────┐
│ ラインチェック（Line Check）                       │
│   目的: 「音が出るか」の確認                       │
│   内容: 各チャンネルから信号が来ているか確認        │
│   時間: 5〜10分（早い）                            │
│   やること: 「マイク1、OK」「キーボード、OK」       │
│                                                    │
│ サウンドチェック（Sound Check）                     │
│   目的: 「いい音か」の確認・調整                    │
│   内容: ゲイン設定、EQ調整、バランス、モニター     │
│   時間: 20〜40分（じっくり）                       │
│   やること: 一人ずつ演奏してもらい、音を作り込む    │
└──────────────────────────────────────────────────┘
```

### ラインチェックが先

```
全体の流れ:

  1. ラインチェック（5〜10分）
     └── 全チャンネルの接続確認
         「音が出ない！」を先に発見

  2. トラブル解決（必要に応じて）
     └── ケーブル交換、接続し直し等

  3. サウンドチェック（20〜40分）
     └── 一人ずつ音を作り込む
```

> **中村メモ:** ラインチェックを飛ばしてサウンドチェックを始めると、「あれ、ベースの音が出ない！」で中断になります。全チャンネル音が出ることを先に確認。5分で終わるけど、この5分が後のトラブルを防ぎます。

## 2. なぜ一人ずつ確認するのか？（5分）

### 建物の例え

```
家を建てるとき:

  ❌ 基礎も壁も屋根も同時に作る → 崩れる

  ✅ 基礎 → 柱 → 壁 → 屋根 → 内装
     一段ずつ確実に積み上げる

音も同じ:

  ❌ 全員で一斉に鳴らす → 何が問題かわからない

  ✅ 基礎（低音）→ 中間 → 上層（高音・ボーカル）
     一つずつ確実に積み上げる
```

### 一人ずつ確認するメリット

```
1. 問題を特定しやすい
   「ブーン」というノイズ → 全員鳴っていたら原因不明
                         → 一人ずつなら「ベースの時だけ鳴る」と特定できる

2. 各楽器の音をちゃんと聴ける
   キーボードの高音が耳に刺さる → 全体の中だと気づきにくい
                                 → 単独で聴けばすぐわかる

3. バランスを段階的に作れる
   ドラム + ベースで土台を確認
   → そこにキーボードを足す
   → 最後にボーカルを乗せる
   → 一つずつ確認しながら積み上げられる
```

## 3. サウンドチェックの順番（15分）

### 基本の順番

```
┌──────────────────────────────────────────────────┐
│         サウンドチェックの順番（下から上へ）        │
│                                                    │
│  ⑤ ボーカル（一番上・一番大事）                    │
│  ④ キーボード・ギター（メロディ・ハーモニー）      │
│  ③ ベース（低音の基礎）                           │
│  ② スネア → タム → シンバル                       │
│  ① キック（一番下・基礎の基礎）                    │
│                                                    │
│  ※ 建物と同じ。基礎（低音）から屋根（高音）へ     │
└──────────────────────────────────────────────────┘
```

### 手順1: キックドラム（土台）

```
最初にキックドラムから始める理由:
  - バンド全体のリズムの土台
  - 低音の基準になる
  - 他の楽器はキックの上に乗っかる

やること:
  1. ドラマーに「キックだけお願いします」
  2. PFLでゲインを確認
  3. フェーダーを上げて音を確認
  4. 低音の出方、アタック感を聴く
  5. 「OK、ありがとうございます」

聴くポイント:
  ├── 「ドッ」というアタックが聞こえるか
  ├── 低音がブーミー（もわっとしすぎ）でないか
  └── スピーカーから均等に聞こえるか
```

### 手順2: スネア・タム・シンバル

```
キックの次に残りのドラムを確認:

  スネア:
    「スネアだけお願いします」
    → パリッとしたアタックと胴鳴りを確認
    → キックと一緒に叩いてもらってバランス確認

  タム:
    「ハイタム→ロータムの順で叩いてください」
    → 音の粒が揃っているか確認

  シンバル（ハイハット・クラッシュ・ライド）:
    「シンバル類をそれぞれお願いします」
    → 耳に刺さる高音がないか確認

  最後に:
    「好きなビートを叩いてください」
    → ドラム全体のバランスを確認
```

### 手順3: ベースギター

```
ドラムの土台の上にベースを重ねる:

  1. ドラムのフェーダーはそのまま残す
  2. 「普通に弾いてください」
  3. ベースのゲインを確認して音を出す

  聴くポイント:
  ├── キックとベースの低音がぶつかっていないか
  │   （どちらかがもわっとしたら要調整）
  ├── 音程がはっきり聞こえるか
  └── 「ドラム + ベース」でグルーブが感じられるか

  ┌────────────────────────────────┐
  │ ドラム + ベース = リズム隊      │
  │ ここがしっかりしていれば        │
  │ バンド全体が安定する            │
  └────────────────────────────────┘
```

### 手順4: キーボード・ギター

```
リズム隊の上にメロディ楽器を乗せる:

  キーボード①（メインキーボード）:
    1. 「普通に弾いてください」
    2. ステレオのLR バランスを確認
    3. 音量はベースと同じか少し上くらい

  キーボード②（パッド・ストリングス系）:
    1. 同様に確認
    2. キーボード①とぶつかっていないか

  アコースティックギター:
    1. 「コードをジャカジャカ弾いてください」
    2. カッティングのアタックが聴こえるか
    3. 低音がもわっとしていないか

  エレキギター:
    1. 「歪み/クリーン両方聴かせてください」
    2. クリーンとドライブの音量差を確認
    3. 他の楽器と帯域が被っていないか

  全楽器で:
    「みんなで合わせてみましょう」
    → ここで一度全体のバランスを確認
```

### 手順5: ボーカル（最後で一番大事）

```
すべての楽器の上にボーカルを乗せる:

  なぜ最後か:
  ├── ボーカルは礼拝で一番聞こえなきゃいけない
  ├── 全楽器の上に「座る」必要がある
  └── 他の楽器が決まっていないとバランスが取れない

  手順:
  1. メインボーカルから
     「一番よく歌う曲のサビをお願いします」
  2. 楽器隊と一緒に歌ってもらう
  3. ボーカルが楽器の上にちゃんと乗っているか確認
  4. コーラス（サブボーカル）も一人ずつ確認
  5. 説教用マイクも確認（話し声の距離感で）

  聴くポイント:
  ├── 歌詞がはっきり聞き取れるか（最重要）
  ├── 楽器に埋もれていないか
  ├── 「サ行」が刺さらないか（歯擦音）
  └── 全ボーカルのバランス（メインが一番前に聞こえるか）
```

> **中村メモ:** 教会で一番大事な音はボーカルの言葉です。歌詞が聞こえない賛美は祈りが届かない。どんなにバンドの音がカッコよくても、歌詞が聞こえなければ意味がないんです。

## 4. モニターミックス（10分）

### モニターとは

```
【ステージ上の音の流れ】

        客席 ← メインスピーカー（PA が調整）
        ──────────────────────
        ステージ

  ボーカル     キーボード    ドラマー
    ↑              ↑           ↑
  [モニター]  [モニター]   [モニター]
    ↑              ↑           ↑
    └──── ミキサーから別々に送る ────┘

モニター = ステージ上の人が自分たちの音を聴くためのスピーカー
メイン   = 客席の人が聴くスピーカー
```

### モニターミックスの基本

```
各ミュージシャンが「自分に聞こえてほしい音」は違う:

  ボーカリスト: 「自分の声と、ピッチの基準になるキーボードが聞きたい」
  キーボード:   「自分のキーボードとボーカルが聞きたい」
  ドラマー:     「ベースとボーカルが聞きたい」
  ギタリスト:   「自分のギターとドラムが聞きたい」
```

### モニターリクエストへの対応

```
ミュージシャンからの要望:

  「もっと自分の声が聞きたい」
  → そのモニターのボーカルを上げる

  「キーボードが聞こえない」
  → そのモニターのキーボードを上げる

  「全体的に音量上げて」
  → まず特定の楽器を上げてみる。全体を上げるとハウリングの原因に

  ┌─────────────────────────────────────────────┐
  │ 大事な心構え:                                  │
  │                                                │
  │ ミュージシャンの要望には素直に応える。          │
  │ 「気のせいじゃない？」とは絶対言わない。        │
  │ ステージの上はPA席と全然違う音が聞こえている。  │
  │ 彼らの耳を信じること。                          │
  └─────────────────────────────────────────────┘
```

> **中村メモ:** モニターの調整は「コミュニケーション」です。技術的なことより、ミュージシャンの気持ちに寄り添うことが大事。「はい、上げますね」「これくらいでどうですか？」と声をかけながら調整してください。

## 5. ミュージシャンとのコミュニケーション（5分）

### 声かけの例

```
サウンドチェック開始時:
  「これからサウンドチェックします。
   一人ずつお願いするので、名前を呼んだら演奏してください」

各楽器の確認時:
  「○○さん、キーボードお願いします」
  「普通の音量で弾いてください......OK」
  「一番強く弾くところもお願いします......はい、大丈夫です」
  「モニターの音量はどうですか？」

問題がある時:
  「ちょっと待ってくださいね、調整します」
  （焦らない。「少々お待ちください」で十分）

全体確認:
  「では全員で1曲通してみましょう」
  「音量バランスで気になるところありますか？」

終了時:
  「サウンドチェック完了です、ありがとうございました！」
```

### やってはいけないこと

```
❌ 無言でいきなり音を出す（ミュージシャンがびっくりする）
❌ 「もっと小さく弾いてください」（演奏スタイルを否定しない）
❌ 「ちゃんと弾いてくれないとわからない」（偉そう）
❌ チェック中にスマホをいじる（信頼を失う）
❌ 問題があるのに「大丈夫です」と言う（後で困る）

✅ 常に声をかけて進行状況を伝える
✅ 笑顔で対応する
✅ 「ありがとうございます」を多めに言う
✅ 問題があれば正直に「少し調整させてください」と言う
```

## 6. 教会のサウンドチェック — 時間管理（5分）

### 現実的なタイムライン

```
教会のサウンドチェックは時間が限られている！

  プロのコンサート: 1〜2時間たっぷり使える
  教会の礼拝:      20〜30分しかない場合も

  だから優先順位が大事:

  最優先（必ずやる）:
  ├── ラインチェック（全チャンネル音が出るか確認）
  ├── ボーカルのゲインとバランス
  └── メインミックスの大まかなバランス

  できればやる:
  ├── 各楽器のEQ調整
  ├── モニターミックスの調整
  └── 全体通しリハーサル

  時間があればやる:
  ├── 細かいEQ
  ├── エフェクト（リバーブ等）の調整
  └── 配信ミックスの確認
```

### 時間がないときの短縮版

```
5分サウンドチェック（緊急時）:

  1. 全マイク・楽器のラインチェック     [2分]
     「順番にちょっとだけ音出してください」

  2. ボーカルのバランス確認             [1分]
     「歌ってみてください......OK」

  3. 全員で1コーラス合わせる            [2分]
     ここで大まかなバランスを耳で調整

  → 本番中に微調整する覚悟で！
```

> **中村メモ:** 完璧を目指して時間を使い切るより、「80%の出来で全部終わらせる」方が大事です。一つの楽器にこだわりすぎて、ボーカルのチェックができなかったら本末転倒。全体を見る目を持ちましょう。

## 実践ドリル（10分）

### ドリル1: 順番クイズ

以下の楽器をサウンドチェックの正しい順番に並べ替えてください。

```
A. アコースティックギター
B. ボーカル
C. キックドラム
D. ベース
E. キーボード
F. スネアドラム

正しい順番:
→ [  ] → [  ] → [  ] → [  ] → [  ] → [  ]
```

答え: C → F → D → E → A → B

### ドリル2: モニターリクエスト対応

ボーカリストが「キーボードがもっと聞きたい」と言いました。何をしますか？

```
答え:
1. 「はい、上げますね」と応答
2. ボーカル用モニターのキーボードのセンドレベルを上げる
3. 「これくらいでどうですか？」と確認
4. OKをもらうまで微調整
```

### ドリル3: 実践サウンドチェック

教会のメンバーに協力してもらい、実際のサウンドチェックを体験:

```
準備:    電源ON（Lesson 9の手順）
         ↓
ライン:  全チャンネルのラインチェック
         ↓
ゲイン:  各チャンネルのゲイン設定（Lesson 10の手順）
         ↓
チェック: キック → スネア → ベース → キーボード → ギター → ボーカル
         ↓
全体:    全員で1曲通す
         ↓
確認:    モニターの要望を聞く
```

## 教会での応用

### うちの教会のサウンドチェック順

```
教会の実際の構成での順番:

  ① ドラム全体           ch 13-20（8ch想定）
     └── キック → スネア → ハイハット → タム → OH

  ② ベース               ch 11
     └── DI経由

  ③ エレキギター         ch 12
     └── DI or マイク

  ④ アコースティックギター ch 10
     └── DI or ピックアップ

  ⑤ キーボード① (L/R)   ch 7-8

  ⑥ キーボード② (L/R)   ch 9（ステレオなら9-10）

  ⑦ ボーカルマイク × 4   ch 1-4
     └── メインボーカルを最初に

  ⑧ 説教用マイク × 2     ch 5-6

  ⑨ iPhone (BGM)         ch 21-22
     └── 曲を流して音量確認

  ⑩ PC (カウントダウン)   ch 23-24
     └── カウントダウン映像の音声確認
```

### 礼拝直前の最終チェック

```
礼拝5分前チェックリスト:

  □ 全マイクがミュートされているか（礼拝開始まで）
  □ BGMが適切な音量で流れているか
  □ 説教用マイクがスタンバイか
  □ 配信がスタンバイか
  □ ヘッドフォンのPFL/SOLOが解除されているか
  □ メーターが異常な値を示していないか
```

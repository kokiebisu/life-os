今日のテーマは地味かもしれない。ケーブル。「ただの線でしょ？」って思うよね。でもね、音が通る道がケーブルなんだ。道が悪ければ、どんなにいいマイクもミキサーもスピーカーも台無しになる。

「音には必ず理由がある」--- ノイズが入るのも、音が出ないのも、原因はケーブルだったということが本当に多い。今日は「音の血管」をしっかり学ぼう。

## 1. なぜケーブルが重要か（5分）

音の信号は電気。電気はケーブルの中を流れる。ケーブルが悪いと:

- **ノイズが入る** --- 「ブーン」「ジー」という雑音
- **音が途切れる** --- 断線や接触不良
- **音質が悪くなる** --- 高音が落ちる、音がこもる
- **最悪、機材が壊れる** --- 間違った接続は機材にダメージを与えることもある

```
  良いケーブル                悪いケーブル

  マイク ─────→ ミキサー    マイク ──╳──→ ミキサー
  きれいな信号が通る         ノイズ混入・断線・音質劣化
```

> **藤井メモ:** プロの音響さんは「音が悪いときは、まずケーブルを疑え」と言う。機材のトラブルの半分以上はケーブル関連なんだよ。地味だけど、ここを理解しておくとトラブル対応が格段に速くなる。

## 2. ケーブルの種類（15分）

### XLR ケーブル（エックスエルアール）

音響の世界で**最も重要なケーブル**。マイクとミキサーをつなぐ定番。

```
  XLR コネクタ（断面図）

  メス（機材側）         オス（ケーブル側）

    ┌───────┐            ┌───────┐
    │ ○   ○ │            │ ●   ● │
    │   ○   │            │   ●   │
    └───────┘            └───────┘
     2   3                2   3
       1                    1

  ピン 1 = グラウンド（GND / シールド）
  ピン 2 = ホット（+ / 正相の信号）
  ピン 3 = コールド（- / 逆相の信号）
```

特徴:
- **3ピン構成** --- バランス接続（後述）でノイズに強い
- **ロック機構** --- カチッとはまって抜けにくい
- **頑丈** --- プロ仕様で耐久性が高い
- **長距離OK** --- 50m でもノイズが入りにくい

用途: マイク → ミキサー、ミキサー → スピーカー、ほぼ全ての音響接続

### TRS ケーブル（ティーアールエス）

見た目は普通のヘッドホンプラグに似ているが、**3つの接点**がある。

```
  TRS プラグ（側面図）

  ┌──────────────────────────┐
  │ Tip   │ Ring  │ Sleeve  │
  │(チップ)│(リング)│(スリーブ)│
  │  信号+ │ 信号- │  GND   │
  └──────────────────────────┘
      ↑       ↑       ↑
    先端    真ん中    根元
    の接点  の接点   の接点

  ※ 黒い線（絶縁リング）が2本ある = TRS
```

特徴:
- **バランス接続可能** --- XLR と同等のノイズ耐性
- **ステレオ使用も可能** --- Tip=左、Ring=右、Sleeve=GND
- **サイズ:** 6.35mm（1/4インチ）が一般的

用途: ミキサーの入出力、キーボードの出力、ヘッドホン

### TS ケーブル（ティーエス）

TRS から Ring を抜いた、**2つの接点**のケーブル。ギターのシールドケーブルがこれ。

```
  TS プラグ（側面図）

  ┌──────────────────────┐
  │  Tip    │  Sleeve    │
  │(チップ)  │(スリーブ)   │
  │  信号    │   GND      │
  └──────────────────────┘

  ※ 黒い線（絶縁リング）が1本 = TS
```

特徴:
- **アンバランス接続** --- ノイズに弱い
- **短い距離向き** --- 5m 以上だとノイズが入りやすい
- **サイズ:** 6.35mm（1/4インチ）

用途: エレキギター → アンプ、ベース → アンプ

> **藤井メモ:** TRS と TS の見分け方は簡単。プラグの黒い線（絶縁リング）を数えるだけ。2本なら TRS、1本なら TS。よーく見てみてね。

### RCA ケーブル（アールシーエー）

家庭用オーディオでよく見る**赤と白**のケーブル。

```
  RCA コネクタ

  ┌───┐  ┌───┐
  │ ○ │  │ ○ │
  │ 白 │  │ 赤 │
  └───┘  └───┘
   左ch    右ch
  (L)     (R)
```

特徴:
- **アンバランス接続** --- ノイズに弱い
- **家庭用レベル（-10dBV）** --- プロ機材より信号レベルが低い
- **ペアで使う** --- 赤=右、白=左

用途: 家庭用オーディオ、一部のDJミキサー

### 3.5mm ミニプラグ

スマートフォンやPCのヘッドホン端子でおなじみの小さなプラグ。

```
  3.5mm ミニプラグ

  TRS（ステレオ）         TRRS（マイク付きイヤホン）
  ┌────────────┐        ┌──────────────────┐
  │ T │ R │ S  │        │ T │ R │ R2 │ S  │
  │ L │ R │GND │        │ L │ R │GND │Mic │
  └────────────┘        └──────────────────┘
  黒線2本               黒線3本
```

用途: iPhone（BGM再生）→ ミキサー、PC → ミキサー

> **藤井メモ:** 3.5mm からミキサーにつなぐときは「3.5mm → TRS（6.35mm）」や「3.5mm → XLR」の変換ケーブルが必要になるよ。教会には1本は用意しておきたいね。

## 3. バランス vs アンバランス（10分）

ケーブルの種類を学んだところで、最も大事な概念 ---**バランス接続**--- を理解しよう。

### アンバランス接続（unbalanced）

信号線が1本 + グラウンド。TS ケーブルや RCA がこれ。

```
  アンバランス接続

  送信側                              受信側
  ┌──────┐                          ┌──────┐
  │      │─── 信号（+） ──────────→│      │
  │      │─── GND ─────────────── │      │
  └──────┘                          └──────┘
               ↑
          ここにノイズが混入する
          （電磁波、電源ノイズ等）
```

問題: ケーブルが長いと、周囲の電磁波（蛍光灯、電源ケーブル等）がノイズとして信号線に乗ってしまう。

### バランス接続（balanced）

信号線が2本（ホット + コールド）+ グラウンド。XLR と TRS がこれ。

ここが天才的な仕組み。**同じ信号を正相（+）と逆相（-）の2本で送る**。

```
  バランス接続（ノイズキャンセルの仕組み）

  送信側                              受信側
  ┌──────┐                          ┌──────┐
  │      │─── ホット（+信号）────→ │      │
  │      │─── コールド（-信号）──→ │ 差分 │
  │      │─── GND ──────────────│ 検出 │
  └──────┘                          └──────┘

  ★ ノイズが入る過程:

  ホット:    +信号 ＋ ノイズ  ──→ ┐
                                    ├→ 差分を取る
  コールド:  -信号 ＋ ノイズ  ──→ ┘

  差分 = (+信号＋ノイズ) - (-信号＋ノイズ)
       = +信号 + ノイズ + 信号 - ノイズ
       = 2倍の信号（ノイズは消える！）
```

ポイント:
- ノイズは両方のケーブルに**同じように**乗る
- 受信側で「ホット - コールド」の差分を取ると、ノイズだけが消える
- 信号は2倍になって残る

```
  視覚的に理解するバランス接続

  ホット:     ╱╲╱╲╱╲   ← 正弦波（信号）
  コールド:   ╲╱╲╱╲╱   ← 逆さまの正弦波

  ノイズ:     ~~~         ← 両方に同じノイズが乗る

  ホット+ノイズ:   ╱╲╱╲╱╲~~~
  コールド+ノイズ: ╲╱╲╱╲╱~~~

  差分を取ると:
  信号: ╱╲╱╲╱╲ × 2（2倍に）
  ノイズ: ~~~ - ~~~ = 0（消える！）
```

> **藤井メモ:** バランス接続は「同じノイズを2回受けて、引き算で消す」という天才的な仕組み。だからXLR ケーブルは 50m 引いてもきれい。TS ケーブル（アンバランス）で 50m 引いたら、ノイズだらけになっちゃうよ。

## 4. 信号レベル（10分）

ケーブルの中を流れる電気信号には「レベル（大きさ）」がある。3つの主要なレベルを覚えよう。

### 3つの信号レベル

```
  信号レベルの比較（大きさのイメージ）

  楽器レベル     マイクレベル              ラインレベル
  -20 dBu        -60 〜 -40 dBu          +4 dBu (プロ)
                                          -10 dBV (民生)

  |              |                        |
  |██            |                        |████████████
  |██            |█                       |████████████
  |██            |█                       |████████████
  |              |                        |
  小さい ←──────────────────────────→ 大きい
```

### マイクレベル（mic level）

- **最も弱い信号** --- -60 dBu 〜 -40 dBu くらい
- マイクが直接出す信号の大きさ
- そのままスピーカーに入れても音は聞こえない。**必ずプリアンプ（ゲイン）で増幅する**
- ミキサーの「マイク入力（XLR）」に接続する

### ラインレベル（line level）

- **最も強い信号** --- +4 dBu（プロ）or -10 dBV（民生）
- キーボード、CD プレーヤー、ミキサーの出力などの信号レベル
- すでに増幅済みなので、そのまま処理・出力できる
- ミキサーの「ライン入力（TRS/RCA）」に接続する

### 楽器レベル（instrument level）

- **マイクとラインの中間** --- -20 dBu くらい
- エレキギターやベースのピックアップが直接出す信号
- ラインレベルより弱く、インピーダンス（電気抵抗）が高い
- **DI ボックス** を通してからミキサーに入れる

### プロラインと民生ラインの違い

| レベル名 | 大きさ | 主な機器 |
|---------|--------|---------|
| プロライン | +4 dBu | ミキサー、プロ用機材 |
| 民生ライン | -10 dBV | iPhone、PC、家庭用機材 |

> **藤井メモ:** iPhone の出力は「民生ライン」。プロ用ミキサーの入力は「プロライン」を想定している。レベルが合わないと、ゲインを思いっきり上げないといけなくなり、ノイズが増える。DI ボックスや変換アダプタでレベルを合わせるのが正解だよ。

## 5. DI ボックス（10分）

DI ボックス（ダイレクトボックス / Direct Injection Box）は、**信号の変換器**。教会で地味に活躍する重要な機材。

### DI ボックスの3つの仕事

```
  DI ボックスがやること

  ① アンバランス → バランスに変換
  ② インピーダンスを変換（Hi-Z → Lo-Z）
  ③ 信号レベルを変換（楽器レベル → マイクレベル）

  ギター/ベース                    ミキサー
  ┌──────┐     ┌─────────┐      ┌──────┐
  │      │──→ │ DI ボックス│ ──→│      │
  │TS出力 │     │         │      │XLR入力│
  │アンバランス│  │ 変換！   │     │バランス│
  │Hi-Z   │     │         │      │Lo-Z  │
  └──────┘     └─────────┘      └──────┘
```

### なぜ DI ボックスが必要？

**エレキギター・ベースの場合:**
- TS ケーブル（アンバランス）でミキサーまで長い距離を引くとノイズだらけになる
- DI ボックスで**ステージ上で**バランスに変換すれば、長い XLR ケーブルでもきれい

**キーボードの場合:**
- 出力がアンバランスの機種もある
- ステレオ（L/R）の場合は DI ボックスを2台使う

**iPhone / PC の場合:**
- 3.5mm 出力はアンバランスで信号レベルも低い
- DI ボックスで変換してからミキサーに入れると安定する

### アクティブ DI vs パッシブ DI

| 種類 | 電源 | 向いている入力 |
|------|------|---------------|
| パッシブ DI | 不要 | キーボード等（信号が強い） |
| アクティブ DI | 必要（電池 or ファンタム） | ギター・ベース等（信号が弱い） |

> **藤井メモ:** 「なんでギターを直接ミキサーに差しちゃダメなの？」という質問をよく受ける。理由は、ギターの出力はインピーダンス（電気の抵抗みたいなもの）が高すぎて、ミキサーのマイク入力とは合わないから。DI ボックスが「通訳」してくれるんだよ。

```
  │                                          │
  └──────────────────────────────────────────┘
         │
         v ミキサーからの出力
  ┌──────────────────────────────────────────┐
  │  Main L/R ──XLR──→ メインスピーカー       │
  │  Aux 1    ──XLR──→ フロアモニター         │

  │  Aux 3    ──XLR──→ 配信用 PC              │
  └──────────────────────────────────────────┘
```

### まとめ表

| 機材 | ケーブル | 経由 | 理由 |
|------|---------|------|------|
| ワイヤレスマイク | 電波 → XLR | 受信機 | ワイヤレスだから |
| 有線マイク | XLR | 直接 | マイクレベル + バランス |
| キーボード | TRS → XLR | DI | バランス変換 |
| アコギ | TS → XLR | DI | インピーダンス + バランス変換 |
| エレキ | TS → XLR | DI or マイク | 直接 or アンプにマイク |
| ベース | TS → XLR | DI | インピーダンス + バランス変換 |
| ドラム | XLR | 直接 | マイクレベル + バランス |
| iPhone | 3.5mm → XLR | DI | レベル + バランス変換 |
| PC | 3.5mm → XLR | DI | レベル + バランス変換 |
| スピーカー | XLR | 直接 | ラインレベル + バランス |

> **藤井メモ:** 気づいた？ ミキサーから先（出力）は全部 XLR。ミキサーに入る前（入力）は、楽器によってバラバラ。だから DI ボックスが大活躍するんだ。「入力を全部 XLR に統一する」のが DI の一番の仕事とも言えるね。

## 教会での応用

### 日曜日にやってみること

1. **ケーブルを観察する** --- ステージ上のケーブルを1本ずつ見て、XLR / TRS / TS を見分けてみよう
2. **DI ボックスを探す** --- ステージ上に四角い小さな箱がないか探してみて。それが DI かも
3. **ミキサーの入力端子を見る** --- XLR とTRS の端子が並んでいるはず。どっちに何がつながっている？
4. **ケーブルの長さを意識する** --- TS ケーブルで長い距離を引いている箇所はないか確認
5. **予備ケーブルの場所を確認** --- ケーブルが壊れたときに交換できるよう、予備の置き場所を把握しておこう

### ケーブルの取り扱い

| やること | やらないこと |
|----------|-------------|
| コネクタを持って抜き差しする | ケーブルを引っ張って抜く |
| 8の字巻きで保管する | 腕にぐるぐる巻きつける |
| 使用前に端子を目視確認する | 曲がったピンのまま差し込む |
| ケーブルの上を踏まない | ケーブルの上にスピーカーを置く |

### 8の字巻き（プロの巻き方）

```
  ① 順巻き        ② 逆巻き        ③ 繰り返す

   ╭───╮           ╭───╮
   │   │           │   │        順 → 逆 → 順 → 逆...
   │   │           │   │
   ╰───╯           ╰───╯        こうすると、次に使うとき
  （手前に回す）  （奥に回す）     ねじれずにスルッとほどける
```

ケーブルを長持ちさせる基本中の基本。腕にぐるぐる巻くとケーブル内部の線がねじれて断線しやすくなる。

Phase 2「機材を知る」の4レッスンが終了！ 学んだことを振り返ろう:

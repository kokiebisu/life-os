前回はマイクの世界を旅したね。マイクが音を電気信号に変えてくれた。でも、教会には20本以上の入力がある。ボーカル4本、説教2本、キーボード4ch、ギター、ベース、ドラム... これを全部まとめて、スピーカーから出す --- その「交通整理」をしてくれるのがミキサーだよ。

## 1. ミキサーとは何か（5分）

ミキサー（mixer / mixing console）は、**複数の音声信号を受け取り、調整し、混ぜ合わせて出力する装置**。

やっていることは3つだけ:

```
  ① 受け取る     ② 調整する     ③ 混ぜて出す

  マイク 1 ─→ ┌──────────┐
  マイク 2 ─→ │          │ ─→ メインスピーカー L/R
  マイク 3 ─→ │ ミキサー  │ ─→ モニタースピーカー
  マイク 4 ─→ │          │ ─→ サブウーファー
  Key L/R ──→ │ 各chの音量 │ ─→ 配信ミックス
  ギター  ──→ │ 音質・バランス│ ─→ 録音
  ベース  ──→ │ を調整    │
  ドラム  ──→ │          │
  iPhone ──→ └──────────┘
  PC     ──→
```

レストランで例えると: ミキサーは**厨房のシェフ**。食材（各楽器の音）を受け取って、味付け（EQ）して、量を調整（フェーダー）して、お皿に盛り付け（ミックス）して、テーブル（スピーカー）に出す。

## 2. チャンネルストリップ（15分）

ミキサーの1本の入力を処理する縦一列を**チャンネルストリップ（channel strip）**と呼ぶ。信号は上から下に流れる。

```
  ┌─────────────────────────────────┐
  │        チャンネルストリップ        │
  │         （1ch ぶんの処理）        │
  │                                  │
  │  ┌───────────┐                   │
  │  │ ① INPUT   │ ← マイクやDIから  │
  │  │   入力     │   信号が入る      │
  │  └─────┬─────┘                   │
  │        ↓                         │
  │  ┌───────────┐                   │
  │  │ ② GAIN    │ ← 入力信号の      │
  │  │   ゲイン   │   大きさを揃える  │
  │  └─────┬─────┘                   │
  │        ↓                         │
  │  ┌───────────┐                   │
  │  │ ③ HPF     │ ← 低域の         │
  │  │ ハイパス   │   不要な音を切る  │
  │  └─────┬─────┘                   │
  │        ↓                         │
  │  ┌───────────┐                   │
  │  │ ④ EQ      │ ← 音質を         │
  │  │ イコライザー│   調整する       │
  │  └─────┬─────┘                   │
  │        ↓                         │
  │  ┌───────────┐                   │
  │  │ ⑤ DYNAMICS│ ← コンプレッサー  │
  │  │ ダイナミクス│   ゲート等       │
  │  └─────┬─────┘                   │
  │        ↓                         │
  │  ┌───────────┐                   │
  │  │ ⑥ AUX SEND│ ← モニターや     │
  │  │ Aux センド │   エフェクトに送る│
  │  └─────┬─────┘                   │
  │        ↓                         │
  │  ┌───────────┐                   │
  │  │ ⑦ PAN     │ ← 左右の         │
  │  │   パン     │   定位を決める   │
  │  └─────┬─────┘                   │
  │        ↓                         │
  │  ┌───────────┐                   │
  │  │ ⑧ FADER   │ ← 最終的な       │
  │  │ フェーダー  │   音量を決める   │
  │  └─────┬─────┘                   │
  │        ↓                         │
  │    メインバスへ                    │
  └─────────────────────────────────┘
```

各セクションを詳しく見ていこう:

### ① INPUT（入力）
マイクやDIボックスからの信号を受け取る入口。XLR端子やTRS端子がある。

### ② GAIN（ゲイン = 入力の増幅量）
入ってきた信号を適切な大きさに揃える。**一番最初のボリューム調整**。小さすぎるとノイズが目立ち、大きすぎると音が歪む。

> **藤井メモ:** ゲインは「水道の蛇口」と思ってね。蛇口を開きすぎると水があふれる（歪む）。少なすぎると使い物にならない。ちょうどいい量に調整するのがゲインの仕事。Phase 3 の Lesson 10 で実際に設定するよ。

### ③ HPF（ハイパスフィルター = 高い音だけ通す）
低い周波数の不要な音（空調のゴー、足音のドン）をカットする。「高い音を通す（pass）」からハイパス。

### ④ EQ（イコライザー = 音質調整）
特定の周波数帯域を上げたり下げたりして音質を整える。Phase 1 で学んだ周波数の知識がここで活きる。

### ⑤ DYNAMICS（ダイナミクス処理）
音量の大小差を整える。コンプレッサー（大きい音を抑える）やゲート（小さい音を消す）が入る。

### ⑥ AUX SEND（Aux センド = 別経路に音を送る）
チャンネルの音を、メインとは別の経路に送る。モニタースピーカーやエフェクト（リバーブ等）に使う。

### ⑦ PAN（パン = 左右の定位）
音を左右のどこに置くか決める。真ん中、やや左、完全に右、など。

### ⑧ FADER（フェーダー = 最終音量）
そのチャンネルの最終的な音量を決めるスライダー。ミキシング中にこれを動かしてバランスを取る。

## 3. バス --- 音の高速道路（15分）

ここがミキサーで一番大事な概念。**バス（bus）**とは、複数のチャンネルの信号をまとめる「道」のこと。

高速道路をイメージして。いろんな街（チャンネル）からの車（音声信号）が合流して、1本の高速道路（バス）を通って、目的地（スピーカー等）に向かう。

### バスの3つの種類

```
  ┌─────────────────────────────────────────────┐
  │           3種類のバス                         │
  │                                              │
  │  ① メインバス（Main L/R）                     │
  │     = 会衆に聞こえる最終ミックス               │
  │                                              │
  │  ② Aux バス（Aux Send）                       │
  │     = モニターやエフェクト用の別ルート          │
  │                                              │
  │  ③ サブグループ（Subgroup / DCA）              │
  │     = 複数チャンネルをまとめて制御する          │
  └─────────────────────────────────────────────┘
```

### ① メインバス（Main L/R Bus）

すべてのチャンネルが最終的に合流する「本線」。ここを通った音がメインスピーカーから出る。

```
  ch1 (Vo1)    ──→ ┐
  ch2 (Vo2)    ──→ ├──→ Main L/R バス ──→ メインスピーカー
  ch3 (Key L)  ──→ ├──→                    L/R
  ch4 (Key R)  ──→ ┘
   ...
```

### ② Aux バス（Auxiliary Bus）

「補助的な（auxiliary）」ルート。各チャンネルから独立した音量で別の出力先に送れる。

```
                        Aux 1（モニター用）
  ch1 (Vo1)  ──→ [Aux1つまみ:大] ──→ ┐
  ch2 (Key)  ──→ [Aux1つまみ:小] ──→ ├→ フロアモニター
  ch3 (ドラム) ──→ [Aux1つまみ:中] ──→ ┘

                        Aux 2（配信用）
  ch1 (Vo1)  ──→ [Aux2つまみ:大] ──→ ┐
  ch2 (Key)  ──→ [Aux2つまみ:大] ──→ ├→ 配信ミックス
  ch3 (ドラム) ──→ [Aux2つまみ:小] ──→ ┘
```

> **藤井メモ:** Aux バスのポイントは「メインとは独立したバランスが作れる」こと。会場ではドラムが大きく聞こえるけど、ステージ上のモニターではボーカルを大きくしたい --- こういう「出力先ごとに違うバランスが欲しい」ときに Aux を使うんだ。

### ③ サブグループ / DCA

複数のチャンネルを1つのフェーダーでまとめて操作する機能。

```
  ch1 (Vo1)  ──→ ┐
  ch2 (Vo2)  ──→ ├→ [Vocal グループ] ──→ Main L/R
  ch3 (Vo3)  ──→ ┤   フェーダー1本で
  ch4 (Vo4)  ──→ ┘   全ボーカルの音量を制御

  ch5 (Key L) ──→ ┐
  ch6 (Key R) ──→ ├→ [Band グループ] ──→ Main L/R
  ch7 (Guitar) ──→ ┤   フェーダー1本で
  ch8 (Bass)  ──→ ┘   全バンドの音量を制御
```

20ch以上を1本ずつ操作するのは大変。グループにまとめれば「ボーカル全体をちょっと下げる」が1つのフェーダーでできる。

## 4. マスターセクション（5分）

すべてのバスが最終的に集まる場所が**マスターセクション**。

```
  ┌────────────────────────────┐
  │      マスターセクション      │
  │                            │
  │  Main L/R ──→ [マスター    │ ──→ メインスピーカー L/R
  │               フェーダー]  │
  │                            │
  │  Aux 1    ──→ [Aux 1      │ ──→ フロアモニター
  │               マスター]    │
  │                            │
  │  Aux 2    ──→ [Aux 2      │ ──→ サブウーファー
  │               マスター]    │
  │                            │
  │  Aux 3    ──→ [Aux 3      │ ──→ 配信ミックス
  │               マスター]    │
  │                            │
  │  [メーター表示]             │ ← 各出力のレベルを監視
  └────────────────────────────┘
```

マスターフェーダーは「全体のボリューム」。各チャンネルのバランスはそのままで、出力の総量だけを上げ下げできる。

## 5. デジタルミキサーのメリット（5分）

うちの教会はデジタルミキサーを使っている。アナログミキサーとの違いを見てみよう。

| 機能 | アナログ | デジタル |
|------|----------|----------|
| シーンリコール | なし（毎回手動） | あり（設定を保存・呼び出し） |
| 内蔵エフェクト | なし（外部機器が必要） | あり（リバーブ、コンプ等） |
| ルーティング | 物理的な配線 | 画面上で自由に変更 |
| EQ / ダイナミクス | つまみを回す | 画面で精密に操作 |
| サイズ | チャンネル数に比例して巨大 | コンパクト |
| リモート操作 | 不可 | タブレット等で可能 |

> **藤井メモ:** デジタルミキサーの最大の武器は「シーンリコール（scene recall）」。礼拝用の設定、練習用の設定、特別集会用の設定... 全部保存しておいて、ボタン1つで切り替えられる。アナログだと、つまみの位置を写真に撮って毎回合わせ直していたんだよ。

## 6. 教会のミキサー信号フロー（10分）

ここが今日のハイライト。教会の全入力が、ミキサーを通ってどこに出るか、全体図を描こう:

```
  ┌─ 入力（約20ch）──────────────────────────────────┐
  │                                                   │
  │  ワイヤレス Vo × 4 ──→ ch 1-4                     │
  │  説教マイク × 2   ──→ ch 5-6                      │
  │  Key 1 (L/R)     ──→ ch 7-8                      │
  │  Key 2 (L/R)     ──→ ch 9-10                     │
  │  アコギ          ──→ ch 11                        │
  │  エレキ          ──→ ch 12                        │
  │  ベース          ──→ ch 13                        │
  │  ドラム Kick     ──→ ch 14                        │
  │  ドラム Snare    ──→ ch 15                        │
  │  ドラム HH       ──→ ch 16                        │
  │  ドラム OH (L/R) ──→ ch 17-18                     │
  │  iPhone (BGM)    ──→ ch 19-20                     │
  │  PC (カウントダウン) ──→ ch 21-22                   │
  │                                                   │
  └───────────────────┬───────────────────────────────┘
                      ↓
  ┌─ ミキサー内部 ────────────────────────────────────┐
  │                                                   │
  │  各チャンネル:                                     │
  │  GAIN → HPF → EQ → DYNAMICS → AUX SEND → PAN → FADER│
  │                                                   │
  │  ┌─ グループ ─────────────────────────┐            │
  │  │ [Vocal] ch1-4                     │            │
  │  │ [説教]  ch5-6                     │            │
  │  │ [Keys]  ch7-10                    │            │
  │  │ [Band]  ch11-13                   │            │
  │  │ [Drums] ch14-18                   │            │
  │  │ [Media] ch19-22                   │            │
  │  └───────────────────────────────────┘            │
  │                                                   │
  │  ┌─ バス ─────────────────────────────┐            │
  │  │ Main L/R    → メインスピーカー      │            │
  │  │ Aux 1       → フロアモニター        │            │
  │  │ Aux 2       → サブウーファー        │            │
  │  │ Aux 3       → 配信ミックス          │            │
  │  │ (Aux 4-6)   → 予備 / IEM          │            │
  │  └───────────────────────────────────┘            │
  │                                                   │
  └───────────────────┬───────────────────────────────┘
                      ↓
  ┌─ 出力 ───────────────────────────────────────────┐
  │                                                   │
  │  Main L  ──→ メインスピーカー（左）                 │
  │  Main R  ──→ メインスピーカー（右）                 │
  │  Aux 1   ──→ フロアモニター                        │
  │  Aux 2   ──→ サブウーファー                        │
  │  Aux 3   ──→ 配信用 PC / キャプチャーカード         │
  │                                                   │
  └──────────────────────────────────────────────────┘
```

> **藤井メモ:** この全体図を頭に入れておくと、「あの楽器の音がモニターから出ない」というときに「Aux 1 のセンドが上がっていないのかも」とすぐ推測できる。信号がどこを通っているか知っていれば、トラブルの原因も見つけやすいんだよ。

## 実践ドリル（10分）

### ドリル 1: チャンネルストリップの順番

以下を正しい信号の流れ順に並べ替えて:

```
EQ / FADER / GAIN / PAN / INPUT / AUX SEND / HPF / DYNAMICS
```

<details>
答え: INPUT → GAIN → HPF → EQ → DYNAMICS → AUX SEND → PAN → FADER
</details>

### ドリル 2: バスの使い分け

以下の状況で、どのバスを使うか答えて:

1. ボーカリストに「自分の声をもっと聞かせて」と言われた → ___
2. 全体の音量を下げたい → ___
3. ドラム4本のチャンネルをまとめて操作したい → ___
4. 配信用に会場とは違うバランスで音を送りたい → ___

<details>
答え: 1=Aux（モニター用）, 2=メインバスのマスターフェーダー, 3=サブグループ/DCA, 4=Aux（配信用）

## 教会での応用

### 日曜日にやってみること

1. **ミキサーの画面を観察する** --- チャンネルがいくつあるか、グループはどう分かれているか見てみよう
2. **フェーダーの位置を見る** --- どのチャンネルが大きくて、どれが小さいか。ボーカルは？ ドラムは？
3. **Aux の行き先を確認する** --- Aux 1 はどこにつながっている？ Aux 2 は？
4. **メーター表示を見る** --- 各チャンネルと Main の信号レベルが見えるはず。赤くなっていないか確認
5. **シーンを確認する** --- 保存されているシーンが何個あるか、名前は何か見てみよう

### ミキサー操作時の心構え

| やること | やらないこと |
|----------|-------------|
| 1つずつつまみを動かす | 一度に複数のつまみを変える |
| 変更前の状態を覚えておく | 適当にいじって戻せなくなる |
| メーターを見ながら操作する | 耳だけで判断する |
| わからないボタンは触らない | 「なんだろう？」で押してみる |

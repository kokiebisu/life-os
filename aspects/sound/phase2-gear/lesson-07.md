マイクが音の旅の「入口」なら、スピーカーは「出口」。どんなにいいマイクを使っても、どんなに丁寧にミキシングしても、最後のスピーカーがダメだと全部台無しになる。「音には必ず理由がある」--- スピーカーの種類も、置く場所も、全部理由があるんだ。

## 1. スピーカーの仕組み（10分）

スピーカーは**マイク（ダイナミック型）の逆**の仕組みで動く。

マイクは「振動板が動く → コイルが動く → 電気が生まれる」だった。
スピーカーは「電気が流れる → コイルが動く → 振動板が動く → 空気が振動する」。

```
  スピーカーの内部構造（断面図）

  電気信号（ミキサーから）
        │
        v
  ┌───────────┐
  │  コイル     │ ← 電気で振動する
  │(ボイスコイル)│
  └─────┬─────┘
        │
  ┌─────┴─────┐
  │  磁石      │ ← 固定（コイルの振動を助ける）
  │  N    S   │
  └───────────┘
        │
        v
  ┌───────────────────┐
  │     コーン（振動板）  │ ← コイルと一緒に前後に動く
  │    ╱             ╲  │
  │  ╱                ╲ │
  │ ╱     ← 空気を押す  ╲│
  └───────────────────────┘
        │
        v
  空気の振動 = 音！
```

ポイント: スピーカーのコーン（cone = 円錐形の振動板）が前後に動くことで、空気を押したり引いたりする。この空気の振動が、人の耳に届いて「音」として認識される。

> **藤井メモ:** マイクとスピーカーは双子みたいな関係。実は、スピーカーにケーブルをつないで声を出すと、マイクとして使えることもある（音質はひどいけど）。原理が同じだからこそできる裏技だよ。

## 2. スピーカーの種類（15分）

教会で使う3種類のスピーカーを見ていこう。

### フルレンジ（メインスピーカー）

**人の声から楽器まで、幅広い周波数を出せるスピーカー。** 教会では会衆に向けて設置する L/R のメインスピーカーがこれ。

```
  出せる周波数帯域:

  低音 ←──────────────────────────→ 高音
  80Hz                           20kHz
  |████████████████████████████████|
      フルレンジがカバーする範囲
```

内部には通常2つ以上のユニット（ドライバー）が入っている:

```
  フルレンジスピーカーの内部

  ┌─────────────────────┐
  │    ┌───┐             │
  │    │ ○ │ ← ツイーター │  高音担当（小さい）
  │    └───┘             │
  │   ┌───────┐          │
  │   │       │          │
  │   │   ○   │ ← ウーファー│  中低音担当（大きい）
  │   │       │          │
  │   └───────┘          │
  └─────────────────────┘
```

- **ツイーター（tweeter）**: 高音を出す小さなドライバー
- **ウーファー（woofer）**: 中低音を出す大きなドライバー

### フロアモニター（ウェッジモニター）

**ステージ上のミュージシャンに音を返すためのスピーカー。** 床に置いて斜め上に向ける形（ウェッジ = くさび型）。

```
  フロアモニターの設置

  ミュージシャン
      🧑
      │
      │ 音が↗上向きに届く
      │╱
  ┌──╱────┐
  │╱ モニター│  ← 斜めに設置
  └────────┘
  ─────────── 床 ───────────
```

なぜ必要？
- ステージ上では、メインスピーカーの音が**会衆席のほうに向いている**から、演奏者には聞こえにくい
- 自分の声や他の楽器が聞こえないと、テンポがずれたり音程が取れなくなる
- モニターは「演奏者が心地よく演奏するため」に存在する

### サブウーファー

**低音だけを専門に出すスピーカー。** 大きなコーンで、低い周波数（20Hz〜150Hz くらい）を担当する。

```
  出せる周波数帯域:

  低音 ←──────────────────────────→ 高音
  20Hz        150Hz              20kHz
  |████████████|
   サブウーファー     |████████████████████|
   の担当範囲         メインスピーカーの担当範囲
```

なぜ分けるの？
- 低音を出すには**大きなコーンを大きく動かす**必要がある
- メインスピーカーだけで低音も出そうとすると、中高音の質が落ちる
- 専門家に任せた方がいい。低音はサブウーファー、中高音はメインスピーカー

> **藤井メモ:** サブウーファーの音は「聞く」というより「体で感じる」もの。教会でバスドラムの「ドン！」やベースの「ブーン」を体に響かせたいなら、サブウーファーが活躍するよ。

## 3. パッシブ vs アクティブ（5分）

スピーカーには電源があるかないかで2種類ある。

### パッシブスピーカー（passive = 受動的）

**スピーカー自体にはアンプ（増幅器）がない。** 外部のパワーアンプで信号を増幅してからスピーカーに送る。

```
  ミキサー ──→ パワーアンプ ──→ パッシブスピーカー
               (信号を増幅)     (音を出すだけ)
```

### アクティブスピーカー（active = 能動的、パワードスピーカーとも）

**スピーカーの中にアンプが内蔵されている。** ミキサーから直接つなげば音が出る。電源コードが必要。

```
  ミキサー ──→ アクティブスピーカー
               ┌─────────────┐
               │ アンプ内蔵   │
               │ + スピーカー │
               │ (電源必要)   │
               └─────────────┘
```

### 比較

| 項目 | パッシブ | アクティブ |
|------|----------|-----------|
| アンプ | 外部に必要 | 内蔵 |
| 配線 | スピーカーケーブル | XLR / TRS + 電源ケーブル |
| セットアップ | やや手間 | 簡単（つなぐだけ） |
| 重さ | 軽い | やや重い（アンプ内蔵） |
| 主な用途 | 大規模設備 | 小〜中規模、持ち運び |

> **藤井メモ:** 最近の教会はアクティブスピーカーが多いよ。アンプの選び方を間違える心配がないし、ミキサーから直接つなげるから配線もシンプル。「初心者にやさしい」のがアクティブの良さだね。

## 4. クロスオーバー（10分）

クロスオーバー（crossover）は、**音の周波数帯域を分割して、適切なスピーカーに振り分ける仕組み**。

### なぜ必要？

1本のスピーカーで全周波数を出すのは効率が悪い。低音は大きなコーン、高音は小さなコーンが得意。だから周波数で分けて、得意な相手に任せる。

```
  クロスオーバーの仕組み

  ミキサーからの信号（全周波数）
        │
        v
  ┌──────────────┐
  │ クロスオーバー  │
  │  (周波数で分割) │
  └──┬───────┬───┘
     │       │
     v       v
  低音域    中高音域
  (〜150Hz) (150Hz〜)
     │       │
     v       v
  サブ      メイン
  ウーファー スピーカー
```

### アクティブスピーカーの場合

アクティブスピーカーの中にはクロスオーバーが内蔵されていることが多い:

```
  アクティブスピーカー内部

  ┌────────────────────────────┐
  │ 信号入力                    │
  │    │                       │
  │    v                       │
  │ ┌──────────┐               │
  │ │クロスオーバー│              │
  │ └──┬───┬───┘               │
  │    │   │                   │
  │    v   v                   │
  │  低音  高音                  │
  │  アンプ アンプ               │
  │    │   │                   │
  │    v   v                   │
  │ ウーファー ツイーター         │
  └────────────────────────────┘
```

### メインとサブウーファーの間のクロスオーバー

教会のシステムでは、デジタルミキサーやプロセッサーがクロスオーバーの役割をすることもある:

```
  ミキサーの Main L/R 出力
        │
        ├──→ 150Hz以上 ──→ メインスピーカー L/R
        │                  (中音・高音を担当)
        └──→ 150Hz以下 ──→ サブウーファー
                           (低音を担当)
```

> **藤井メモ:** クロスオーバーの周波数（「ここで分ける」というポイント）は、スピーカーの特性に合わせて設定する。80Hz〜150Hz くらいが一般的。デジタルミキサーなら画面で簡単に調整できるよ。

## 5. スピーカーの配置（10分）

教会でのスピーカー配置を考えよう。大事なのは**「誰に」「何を」届けるか**。

### 教会の配置図

```
  ┌──────────────────────────────────────────┐
  │                                          │
  │              ── ステージ ──               │
  │                                          │
  │   [Mon]    [Vo] [Vo] [Vo] [Vo]   [Mon]  │
  │     △      🎤   🎤   🎤   🎤     △    │
  │  (モニター)                    (モニター)  │
  │                                          │
  │      [Key1] [Key2] [AG] [EG]             │
  │                                          │
  │           [Bass]  [Drums]                │
  │              [Mon]                        │
  │               △                          │
  │            (モニター)                      │
  │                                          │
  ├──────────────────────────────────────────┤
  │                                          │
  │ [Main L]                      [Main R]   │
  │    ◁                              ▷     │
  │  (ステージ端                (ステージ端    │
  │   会衆を向く)                会衆を向く)   │
  │                                          │
  │          ┌──────────────┐                │
  │          │              │                │
  │          │   会衆席      │                │
  │          │              │                │
  │          │   ここに音が  │                │
  │          │   均等に届く  │                │
  │          │   ように配置  │                │
  │          │              │                │
  │          └──────────────┘                │
  │                                          │
  │          [SUB]                            │
  │           ◻                              │
  │        (サブウーファー)                    │
  │     ステージ下 or 中央床置き               │
  │                                          │
  └──────────────────────────────────────────┘
```

### 配置の理由

| スピーカー | 位置 | 理由 |
|-----------|------|------|
| メイン L | ステージ左端、会衆向き | 左側の会衆に音を届ける |
| メイン R | ステージ右端、会衆向き | 右側の会衆に音を届ける |
| モニター | ステージ上、演奏者の足元 | 演奏者に自分や他の音を返す |
| サブウーファー | ステージ下 or 中央 | 低音は指向性が低い（どこに置いても広がる） |

### なぜ L/R に分けるの？

```
  モノラル（1本）        ステレオ（L/R 2本）

      [SP]                [L]         [R]
       │                   │           │
       v                   v           v
  ──全部同じ──         左から     右から
  音が1点から         ピアノ     ギター
                     ← 広がり →
```

- ステレオにすると**音に広がり**が生まれる
- キーボードの左手（低い音）を L に、右手（高い音）を R に振ると、自然な響きになる
- ただし教会では「全員に均等に聞こえる」ことが最優先。極端なパンニングは避ける

### SPL とカバレッジ

- **SPL（Sound Pressure Level = 音圧レベル）**: スピーカーがどれだけ大きな音を出せるか。単位は dB
- **カバレッジ（coverage = 指向角度）**: 音が届く角度。90度×60度 のように表記

```
  カバレッジの例（上から見た図）

  狭いカバレッジ（60度）     広いカバレッジ（120度）

        ╱ ╲                    ╱     ╲
       ╱   ╲                 ╱         ╲
      ╱ 60° ╲               ╱   120°    ╲
     ╱       ╲             ╱             ╲
    [SP]                  [SP]

  奥まで届くが横に狭い    横に広いが奥に届きにくい
```

> **藤井メモ:** 教会の形によって最適なカバレッジは変わる。横に広い会場なら広いカバレッジ、細長い会場なら狭いカバレッジが合う。「スピーカーを置けば音が出る」ではなく、「会場の形に合わせて選ぶ」のが音響設計の基本だよ。

## 実践ドリル（10分）

### ドリル 1: スピーカーの種類を選ぼう

以下の状況で、どのスピーカーを使うか答えて:

1. 会衆全体に歌とメッセージを届けたい → ___
2. ドラマーに自分のキックの音を聞かせたい → ___
3. ベースの「ズンッ」を体で感じさせたい → ___

## 教会での応用

### 日曜日にやってみること

1. **スピーカーの位置を確認する** --- メイン L/R はどこにある？ モニターは何台？ サブウーファーは？
2. **メインスピーカーの向きを見る** --- 会衆席に向いているか確認。角度は均等か？
3. **モニターの向きを見る** --- 誰に向いている？ ボーカリスト？ ドラマー？
4. **パッシブかアクティブか調べる** --- スピーカーに電源コードがあればアクティブ。ないならパッシブ
5. **座席を移動して聞き比べる** --- 前列・後列・左端・右端で音の聞こえ方が違うか体験してみよう

### スピーカーの取り扱い注意

| やること | やらないこと |
|----------|-------------|
| 電源ON前にボリュームを下げる | いきなり大音量で電源ON |
| ケーブルを差してから電源ON | 電源ONのまま抜き差し |
| スピーカーの前に物を置かない | 音の通り道を塞ぐ |
| 天候に注意（屋外使用時） | 雨の中でアクティブスピーカーを使う |

## 1. マイクロフォンとは何か（10分）

マイクロフォン（microphone）は、**音（空気の振動）を電気信号に変換する装置**。

仕組みはシンプル:

```
  音波（空気の振動）
        |
        v
  ┌─────────────┐
  │  振動板       │  ← 薄い膜が空気の振動を受け取る
  │ (ダイアフラム) │
  └──────┬──────┘
         │
         v
  ┌─────────────┐
  │  変換機構     │  ← 振動を電気信号に変える（方式は2種類）
  └──────┬──────┘
         │
         v
  電気信号（オーディオ信号）
        │
        v
  ケーブルでミキサーへ
```

すべてのマイクに共通しているのは「振動板（ダイアフラム = diaphragm）」という薄い膜があること。声や楽器の音が空気を振動させて、その振動が振動板を揺らす。ここまでは全マイク同じ。

違いは「その振動をどうやって電気に変えるか」。これが2つの大きな種類を生むんだ。

## 2. ダイナミックマイク（15分）

### 仕組み

ダイナミックマイク（dynamic microphone）は、**磁石とコイル**で電気を作る。中学校の理科で「電磁誘導」って習ったかな？ 磁石の近くでコイルを動かすと電気が流れる、あれと同じ原理。

```
  ダイナミックマイクの内部構造（断面図）

       音波 →→→
                  ┌──────────┐
                  │ 振動板    │ ← 音で揺れる
                  │(ダイアフラム)│
                  └────┬─────┘
                       │
                  ┌────┴─────┐
                  │  コイル    │ ← 振動板と一緒に動く
                  │ (ぐるぐる) │
                  └────┬─────┘
                       │
                  ┌────┴─────┐
                  │  磁石     │ ← 固定されている
                  │  N   S   │
                  └──────────┘
                       │
                       v
                  電気信号が発生！
```

振動板にくっついたコイルが、固定された磁石の中で振動する。すると電気が生まれる。シンプルでしょ？

### 特徴

| 項目 | ダイナミックマイク |
|------|-------------------|
| 電源 | 不要（自分で電気を作る） |
| 頑丈さ | とても頑丈。落としても大丈夫 |
| 感度 | やや低い（近くの音を拾う） |
| 湿気・温度 | 強い |
| 価格 | 比較的安い |
| 代表機種 | Shure SM58（ボーカル）、SM57（楽器） |

### なぜ教会で使うの？

- ステージ上は音がたくさんある。感度が低いぶん、**狙った音だけ**拾いやすい
- 落としたりぶつけたりしても壊れにくい
- 電源が不要なので、つなぐだけで使える

> **藤井メモ:** SM58 は「マイクの王様」と呼ばれていて、世界中のステージで使われている。教会でもライブでも、迷ったら SM58。それくらい信頼性が高いんだよ。

## 3. コンデンサーマイク（10分）

### 仕組み

コンデンサーマイク（condenser microphone）は、**2枚の金属板の間の電気**で動く。片方の板が振動板で、音を受けると2枚の板の距離が変わる。距離が変わると電気の量（静電容量）が変わる。これを信号として取り出す。

```
  コンデンサーマイクの内部構造（断面図）

       音波 →→→
                  ┌──────────┐
                  │ 振動板    │ ← 音で揺れる（極薄の金属膜）
                  │(前面電極) │
                  └────┬─────┘
                    ↕ 隙間（距離が変わる）
                  ┌────┴─────┐
                  │ 背面電極   │ ← 固定されている
                  └──────────┘
                       │
                  ┌────┴─────┐
                  │ プリアンプ  │ ← 電源が必要（ファンタム電源）
                  └──────────┘
                       │
                       v
                  電気信号が発生！
```

### ファンタム電源（phantom power）って何？

コンデンサーマイクは内部にプリアンプ（小さな増幅回路）があるから、**外から電気を送ってあげる**必要がある。これを「ファンタム電源（phantom power = 48V）」と呼ぶ。ミキサーのチャンネルに「48V」や「Phantom」というボタンがあるはず。

```
  ミキサー側で 48V をオン
        │
        │ XLR ケーブル経由で電気を送る
        v
  コンデンサーマイクが動作する
```

> **藤井メモ:** ダイナミックマイクにファンタム電源を送っても壊れないけど、リボンマイクという特殊なマイクには送ると壊れることがある。うちの教会にはリボンマイクはないから心配しなくて大丈夫。でも「知らないマイクにはむやみに48Vを入れない」は覚えておいて。

### 特徴

| 項目 | コンデンサーマイク |
|------|-------------------|
| 電源 | 必要（ファンタム電源 48V） |
| 頑丈さ | デリケート。衝撃・湿気に弱い |
| 感度 | 非常に高い（繊細な音も拾う） |
| 音質 | 高域まできれいに拾える |
| 価格 | やや高い |
| 代表機種 | ドラムのオーバーヘッド、レコーディング用 |

### ダイナミック vs コンデンサー まとめ

```
  ┌─────────────────────────────────────────────────┐
  │          ダイナミック vs コンデンサー 比較         │
  ├──────────────┬──────────────┬───────────────────┤
  │              │ ダイナミック  │ コンデンサー       │
  ├──────────────┼──────────────┼───────────────────┤
  │ 電源         │ 不要         │ 必要（48V）       │
  │ 頑丈さ       │ ★★★★★      │ ★★☆☆☆           │
  │ 感度         │ ★★★☆☆      │ ★★★★★           │
  │ 価格         │ 安い〜普通    │ 普通〜高い        │
  │ 主な用途     │ ボーカル・楽器 │ OH・レコーディング │
  │ ステージ向き  │ ◎           │ △               │
  └──────────────┴──────────────┴───────────────────┘
```

## 4. 指向性（ピックアップパターン）（10分）

マイクには「どの方向の音を拾うか」という性質がある。これを**指向性（pickup pattern / polar pattern）**と呼ぶ。

### カーディオイド（cardioid = 心臓型）

**正面の音をよく拾い、背面の音をあまり拾わない。** 教会で一番よく使うパターン。

```
        正面（よく拾う）
          ★★★
        ★★★★★★★
      ★★★★★★★★★
    ★★★★★★★★★★★
  ★★★★★★ ● ★★★★★★     ● = マイク位置
    ★★★★★   ★★★★★
      ★★★     ★★★
        ★       ★
        背面（拾わない）
```

なぜカーディオイドが教会に向いているか:
- 歌っている人の声（正面）だけ拾いたい
- 後ろにあるスピーカーの音は拾いたくない（ハウリング防止）

### 無指向性（omnidirectional = 全方向）

**すべての方向の音を均等に拾う。**

```
        ★★★★★★★
      ★★★★★★★★★
    ★★★★★★★★★★★
  ★★★★★★★★★★★★★
  ★★★★★★ ● ★★★★★★
  ★★★★★★★★★★★★★
    ★★★★★★★★★★★
      ★★★★★★★★★
        ★★★★★★★
```

用途: 会議録音、環境音の収録。教会ではラベリアマイク（ピンマイク）に多い。

### 双指向性（figure-8 = 8の字）

**正面と背面の音を拾い、横の音を拾わない。**

```
        ★★★★★
      ★★★★★★★
    ★★★★★★★★★
      ★★★★★★★
        ★ ● ★
      ★★★★★★★
    ★★★★★★★★★
      ★★★★★★★
        ★★★★★
```

用途: 対面インタビュー、特殊なステレオ録音。教会のPA ではほぼ使わない。

> **藤井メモ:** 教会PAでは 90% 以上カーディオイドを使う。「正面の音だけ拾う」というのは、ステージ上で大事なことなんだ。だって、会衆席側のスピーカーの音がマイクに戻ってきたら、ハウリング（キーン！という音）が起きちゃうから。

## 5. ワイヤレスマイクシステム（10分）

教会のボーカルマイクは4本ともワイヤレス。ケーブルなしで動く仕組みを見てみよう。

### 構成

```
  ┌─────────┐    電波     ┌──────────┐
  │ 送信機   │ ～～～～～→ │ 受信機    │
  │(トランス │            │(レシーバー)│
  │ ミッター)│            └─────┬────┘
  └─────────┘                  │ XLR ケーブル
  ↑ マイクと一体              │
  │ or ボディパック            v
  │                     ミキサーへ
  歌い手が持つ
```

- **送信機（transmitter）**: マイク本体に内蔵（ハンドヘルド型）、またはベルトにつけるボディパック型
- **受信機（receiver）**: ミキサーの近くに設置。送信機からの電波を受けて音声信号に戻す
- **周波数（frequency）**: 各マイクは異なる周波数の電波を使う。同じ周波数だと混信する

### 4本のワイヤレスマイクの周波数管理

```
  マイク 1: 周波数 A ～～～→ 受信機 ch1
  マイク 2: 周波数 B ～～～→ 受信機 ch2
  マイク 3: 周波数 C ～～～→ 受信機 ch3
  マイク 4: 周波数 D ～～～→ 受信機 ch4

  ※ A, B, C, D はすべて異なる周波数
  ※ 周波数が近すぎると干渉（ノイズ）が起きる
```

### ワイヤレスの注意点

| 注意点 | 理由 |
|--------|------|
| 電池を毎回チェック | 礼拝中に電池切れ = 音が消える |
| 送信機の電源は使うときだけON | 電池の節約 + 不要な電波を出さない |
| 金属の障害物に注意 | 電波が遮られて音が途切れる |
| 周波数の管理 | 他のワイヤレス機器と干渉しないように |

> **藤井メモ:** ワイヤレスマイクのトラブルで一番多いのは「電池切れ」。地味だけど、これが礼拝中に起きるとパニックになる。だから「毎週新しい電池を入れる」をルーティンにするのが一番の対策だよ。

## 6. 近接効果とハンドリングノイズ（5分）

### 近接効果（proximity effect）

マイクに口を**近づけると低音が強くなる**現象。カーディオイドマイク特有の性質。

```
  距離と低音の関係:

  口 ←5cm→ マイク   低音 ★★★★★★★★  ← ボワッとする
  口 ←15cm→ マイク  低音 ★★★★★       ← ちょうどいい
  口 ←30cm→ マイク  低音 ★★★          ← 薄くなる
```

教会での対策:
- ボーカリストにはマイクと口の距離を「こぶし1個分」と伝える
- 近すぎて低音がこもったら、EQ で低域をカットして補正できる（Lesson 12 で詳しく）

### ハンドリングノイズ

マイクを**握り直したり叩いたりすると出る「ボコッ」という音**。振動板が衝撃を拾ってしまう。

対策:
- マイクは下のほうをしっかり持ち、演奏中は握り直さない
- マイクの頭（グリル部分）を手で覆わない → 指向性が変わってハウリングしやすくなる

```
  ❌ ダメな持ち方          ✅ 正しい持ち方

   ┌───┐                   ┌───┐
   │   │ ← グリルを        │   │ ← グリルは
   │   │   手で覆う        │   │   開けておく
   ├───┤                   ├───┤
   │███│ ← 手             │   │
   │███│                   │ █ │ ← 下のほうを
   │███│                   │ █ │   しっかり握る
   └───┘                   └───┘
```

## 7. 教会のマイク配置マップ（5分）

うちの教会で「どこにどのマイクを使うか」を整理しよう:

```
  ┌──────────────── ステージ ────────────────┐
  │                                          │
  │  [Vo1] [Vo2] [Vo3] [Vo4]  ← ワイヤレス  │
  │   🎤    🎤    🎤    🎤    (ダイナミック)  │
  │                                          │
  │  [説教1] [説教2]         ← 有線 or       │
  │   🎤     🎤              ワイヤレス       │
  │                                          │
  │  [Key1]  [Key2]  ← DI ボックス経由       │
  │  L  R    L  R      (ステレオ × 2台)      │
  │                                          │
  │  [AG]  [EG]  [Bass]  ← マイク or DI      │
  │                                          │
  │  [Drums]                                  │
  │  kick / snare / HH / OH                   │
  │  (ダイナミック + コンデンサー)              │
  │                                          │
  └──────────────────────────────────────────┘
```

| 入力 | マイクの種類 | 理由 |
|------|------------|------|
| ボーカル × 4 | ワイヤレス・ダイナミック | 動きの自由 + 頑丈 |
| 説教 × 2 | ハンドヘルド or ラベリア | 長時間の語り + 移動 |
| アコギ | ダイナミック（SM57等） | ステージ上の音を分離 |
| エレキ | ダイナミック（SM57等） | アンプの前に設置 |
| ベース | DI ボックス | ライン入力が基本 |
| ドラム（キック） | ダイナミック（専用） | 大音量に耐える |
| ドラム（スネア） | ダイナミック（SM57等） | 近接で音を拾う |
| ドラム（OH） | コンデンサー × 2 | シンバルの高域を拾う |

> **藤井メモ:** ドラムのオーバーヘッド（OH = overhead）だけコンデンサーマイクを使うのは、シンバルの「シャーン」という高い音をきれいに拾いたいから。ダイナミックマイクだと高域が少し丸くなっちゃうんだ。

## 教会での応用

### 日曜日にやってみること

1. **受信機の電池残量を確認する** --- ワイヤレスマイク4本すべて
2. **マイクの型番を見る** --- ステージにあるマイクの型番をメモしてみて。SM58？ SM57？ 知らないマイク？
3. **指向性を確認する** --- 各マイクの背面や底面にカーディオイドのマークがないか見てみよう
4. **近接効果を体験する** --- リハーサル中、マイクとの距離を変えて低音の変化を聞いてみて
5. **ファンタム電源のスイッチを探す** --- ミキサーの「48V」ボタンがどこにあるか確認しよう

### よくあるトラブルと原因

| トラブル | まず疑うこと |
|----------|-------------|
| ワイヤレスマイクから音が出ない | 電池切れ or 送信機の電源OFF |
| マイクの音がこもる | 近接効果（口が近すぎ） or グリルを手で覆っている |
| 「ボコボコ」いう | ハンドリングノイズ or 吹かれ（息が当たっている） |
| 「キーン」と鳴る | マイクがスピーカーに向いている（ハウリング） |

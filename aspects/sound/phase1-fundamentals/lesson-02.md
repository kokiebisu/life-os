波には「速い波」と「ゆっくりな波」がありますよね。海の波でも、ザッパーンと大きくゆっくり来る波と、チャプチャプと細かく速い波がある。音の波にも同じことが言えます。そして、波の速さの違いが「音の高さ」の違いなんです。

## 1. 周波数とは何か（15分）

### 1秒間に何回振動するか

前回のレッスンで見た空気の「密」と「疎」を思い出してください。

```
  低い音（ゆっくりな振動）:

  ●●● ●  ●  ● ●●● ●  ●  ●
  [密]   [疎]    [密]   [疎]
  ├─── 1回の波 ───┤
         ↑
    1秒間にこれが100回 → 100Hz


  高い音（速い振動）:

  ●● ● ●● ● ●● ● ●● ● ●● ● ●● ●
  [密][疎][密][疎][密][疎][密][疎]
  ├1回┤
     ↑
    1秒間にこれが1000回 → 1000Hz
```

**周波数 ＝ 1秒間に振動が繰り返される回数**

単位は **Hz（ヘルツ）** です。ドイツの物理学者ハインリッヒ・ヘルツさんの名前から来ています。

- 100Hz → 1秒間に100回振動 → 低い音
- 1,000Hz → 1秒間に1,000回振動 → 中くらいの音
- 10,000Hz → 1秒間に10,000回振動 → 高い音

> **藤井メモ:** 「1,000Hz」は「1kHz（キロヘルツ）」とも書きます。k（キロ）= 1,000 です。ミキサーのツマミに「1k」「4k」「10k」と書いてあったら、それは「1,000Hz」「4,000Hz」「10,000Hz」のことです。

### 周波数が高い ＝ 音が高い

これだけ覚えてください。

```
  周波数が低い ──────────────→ 周波数が高い
  振動がゆっくり                振動が速い
  低い音                       高い音

  20Hz ......... 1kHz ......... 20kHz
  ドドドド      普通の会話      キーーーン
  （重低音）    （中音域）      （超高音）
```

### ピアノで見る周波数

ピアノの鍵盤と周波数の関係を見ると、すごくわかりやすいです。

```
  ピアノの鍵盤（一部）:

  低い ←──────────────────────────────→ 高い

  A1     A2     A3     A4     A5     A6
  55Hz   110Hz  220Hz  440Hz  880Hz  1760Hz
         ×2     ×2     ×2     ×2     ×2
```

面白いことに気づきましたか？ **1オクターブ上がるごとに、周波数がちょうど2倍になる** んです。

「ラ」の音（A4）は440Hz。これは音楽の世界共通の基準音で、チューニングのときに使います。

## 2. 人間の耳が聞こえる範囲（10分）

### 20Hz から 20,000Hz（20kHz）

人間の耳が聞こえる音の範囲は、だいたい **20Hz〜20,000Hz（20kHz）** です。

```
  人間の聴覚範囲:

  20Hz                                              20kHz
  │←──────────── 聞こえる範囲 ──────────────→│
  │                                                   │
  │  これより下:                    これより上:        │
  │  超低周波                       超音波             │
  │  （地震、風）                   （コウモリ、犬笛） │
```

ただし、これは「若くて健康な耳の理想値」です。実際には：

- **年齢とともに高音が聞こえなくなる**: 30代で16kHz、50代で12kHz くらいまで落ちることも
- **大音量に長時間さらされると劣化する**: ライブやコンサートに通っていると特に

> **藤井メモ:** PA操作者は耳が命。イヤホンの音量を上げすぎない、ライブハウスでは耳栓をする、など耳を大事にしてください。プロの音響エンジニアはみんな耳を守っています。

### 音の帯域を4つに分けて覚えよう

20Hz〜20kHzを全部細かく覚える必要はありません。まずはざっくり4つに分けましょう。

```
  ┌──────────┬──────────┬──────────┬──────────┐
  │  低音域   │ 中低音域  │ 中高音域  │  高音域   │
  │  Low     │ Low-Mid  │ High-Mid │  High    │
  │          │          │          │          │
  │ 20-250Hz │250-2kHz  │ 2k-6kHz  │ 6k-20kHz │
  │          │          │          │          │
  │ ズンズン  │ モコモコ  │ パキパキ  │ シャリシャリ│
  │ ドスドス  │ 声の芯   │ 声の明瞭  │ 空気感    │
  └──────────┴──────────┴──────────┴──────────┘
```

- **低音域（Low）**: ベース、バスドラムの「ドン」。体で感じる振動
- **中低音域（Low-Mid）**: 声やギターの「太さ」「温かさ」がある領域
- **中高音域（High-Mid）**: 言葉の聞き取りやすさ、音の「抜け」を決める領域
- **高音域（High）**: シンバルのシャーン、声の「さ行」の空気感

## 3. 教会の楽器を周波数で見る（20分）

              250Hz      2kHz        6kHz

### 各楽器の詳細

#### ベース（Bass）: 40Hz〜1kHz

```
  ベースの周波数分布:
  40Hz──────200Hz──────500Hz──────1kHz
  │ ████████████████████████████████ │
  │ ↑ズーンと響く  ↑弦の音    ↑指の │
  │  基音        中域の太さ   アタック│
```

教会のベースは礼拝全体の「土台」。40〜100Hz あたりの低い音が会場を包み込みます。サブウーファーが鳴らすのは主にこの帯域です。

#### ドラム（Drums）: 30Hz〜15kHz

ドラムは実は周波数帯域がとても広い楽器です。パーツごとに違います。

```
  キック（バスドラム）: 30-200Hz   ドン！ドン！
  スネア:              100Hz-8kHz  パン！タン！
  ハイハット:          1kHz-15kHz  チッチッチッ
  シンバル:            500Hz-20kHz シャーーーン
  タム:                60Hz-5kHz   トントン
```

#### アコースティックギター（A.Gt）: 80Hz〜8kHz

```
  80Hz──────300Hz──────1kHz──────5kHz──────8kHz
  │ ████████████████████████████████████████ │
  │ ↑ボディの   ↑弦の太さ  ↑ストローク ↑指の │
  │  共鳴                    のアタック  摩擦 │
```

アコギの「ジャカジャカ」というストローク音は 2kHz〜5kHz あたり。

#### エレキギター（E.Gt）: 80Hz〜12kHz

エレキギターはエフェクターやアンプの設定で帯域が大きく変わりますが、中音域（500Hz〜3kHz）が中心です。

#### キーボード（Keys）: 30Hz〜15kHz

教会の2台のキーボードは、ピアノ音色なら非常に広い帯域を使います。パッド（持続音）系の音色は中低域が中心。キーボードは帯域が広いぶん、他の楽器とぶつかりやすいです。

> **藤井メモ:** キーボードがステレオ（L/R 2チャンネル）で接続されているのは、広がり感を出すため。でもPA的には「2チャンネルぶん帯域を使う」ことでもあるので、ミックスの中でバランスを取るのが大事です。

#### 男性ボーカル: 100Hz〜6kHz

```
  100Hz────300Hz────1kHz────3kHz────6kHz
  │ ████████████████████████████████ │
  │ ↑声の      ↑声の芯  ↑言葉の    │
  │  低い響き           聞き取り     │
  │  （胸声）            やすさ      │
```

#### 女性ボーカル: 200Hz〜8kHz

女性のほうが基音が高いぶん、全体的に少し上の帯域を使います。

```
  200Hz────500Hz────2kHz────5kHz────8kHz
  │ ████████████████████████████████ │
  │ ↑声の      ↑声の芯  ↑明瞭さ    │
  │  温かみ             ↑息の成分   │
```

### なぜこれが大事なのか

教会で全ての楽器とボーカルが一斉に鳴るとき、周波数帯域が **重なる** 場所があります。

```
  問題が起きやすい帯域:

  200-500Hz:  ベース + ギター + キーボード + ボーカル
              → 全部ここに音がある！「モコモコ」して不明瞭に

  1k-3kHz:    ギター + キーボード + ボーカル
              → 「うるさい」「キンキンする」と感じる原因に

  100Hz以下:  ベース + キック + キーボード
              → 「ボワボワ」して何を弾いているかわからない
```

これを整理するのが **EQ（イコライザー）** の仕事です。ミキサーの各チャンネルについているEQで、楽器ごとに「必要な帯域は残す、不要な帯域はカットする」ことで、全体をスッキリさせます。

> **藤井メモ:** EQ の詳しい使い方は Phase 2 以降で学びます。今日は「楽器ごとに居場所（周波数帯域）がある」「居場所が被ると音がぶつかる」ということだけ覚えてもらえれば十分です。

## 4. iPhone と PC の周波数特性（5分）

教会では iPhone（BGM用）と PC（カウントダウン映像用）も音源として使いますね。

```
  iPhone (BGM):
  - 音楽ファイルの再生 → 20Hz〜20kHz の全帯域を含む
  - ただし iPhone のイヤホン出力は低音域がやや弱い
  - ミキサーに直接つなぐので、ミキサー側で調整可能

  PC (カウントダウン):
  - 映像の音声出力 → 通常は音楽や効果音
  - HDMI や 3.5mm ケーブルでミキサーに接続
  - PC の音量設定が低すぎるとノイズが目立つので注意
```

## 教会での応用

今日学んだ「周波数帯域」の知識は、PA操作でこう使います：

1. **「ボーカルが聞こえない」** → ボーカルの帯域（1kHz〜3kHz）がキーボードやギターに埋もれている可能性。楽器側のその帯域を少し下げる
2. **「低音がボワボワする」** → ベースとキックの低音域（60Hz〜200Hz）が重なっている。どちらかの不要な低音をカットする
3. **「説教が聞き取りにくい」** → 説教マイクの中高音域（2kHz〜4kHz）を少し上げると言葉の明瞭度が上がる
4. **「シンバルがうるさい」** → 高音域（6kHz〜10kHz）が出すぎている。ドラムのオーバーヘッドマイクの高音を少し下げる

> **藤井メモ:** 「聞こえない」と言われたとき、ボリュームを上げるだけが解決策ではありません。「その楽器の周波数帯域で、他の楽器が邪魔をしていないか？」を考える。邪魔している楽器を下げるほうが、全体のバランスが崩れず綺麗に解決できます。これを「引き算のミックス」と言います。

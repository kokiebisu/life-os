# Lesson 1: 音とは何か --- 空気の振動から耳に届くまで

> **Phase 1 / L1: 「音＝空気の振動」を体感レベルで理解する**
> **担当:** 藤井波（音響理論）
> **所要時間:** 約60分
> **前提:** なし（完全初心者OK）

---

## 今日のゴール
- 「音＝空気の振動」を自分の言葉で説明できる
- 音が空気中をどうやって伝わるか、イメージできる
- 教会の礼拝堂で「前の席と後ろの席で音が違う理由」がわかる

---

はじめまして、藤井波です。これからPA（音響）の勉強を一緒にしていきましょう。

最初に一つだけ覚えてほしい言葉があります。

**「音には必ず理由がある」**

礼拝中に「なんかモヤモヤする音だな」とか「後ろの席だと聞こえにくいな」って思ったこと、ありませんか？ それには全部、物理的な理由があるんです。理由がわかれば、対処できる。今日はその第一歩、「そもそも音って何？」から始めましょう。

## 1. 音の正体 --- 空気の振動（15分）

### スピーカーの前に手をかざしてみよう

教会のメインスピーカーの前に手をかざすと、空気が「ブルブル」動いているのを感じませんか？ 低い音（ベースやドラムのキック）のときは特にわかりやすいです。

あれが「音」の正体です。

**音 ＝ 空気の振動**

これだけです。これ以上でも以下でもありません。

### どうやって振動が生まれるか

たとえば、ボーカリストが声を出すとき：

```
  声帯が振動する
      ↓
  口の前の空気が押される
      ↓
  押された空気が隣の空気を押す
      ↓
  隣の空気がさらに隣の空気を押す
      ↓
    ......（ずっと続く）
      ↓
  あなたの耳の鼓膜に届く
```

ドミノ倒しに似ています。最初の1枚（声帯）が倒れると、次々と隣に伝わっていく。

### 「圧縮」と「希薄」--- 音の波

空気が押されると、その場所の空気が「ギュッ」と詰まります（圧縮）。
すると隣に押し出されて、元の場所はスカスカになります（希薄）。

```
  空気の粒を「●」で表すと...

  静かな状態（振動なし）:
  ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ●

  音が通過しているとき:
  ●●●● ●  ●  ● ●●●● ●  ●  ● ●●●●
  [密]  [疎]     [密]  [疎]     [密]

  ←────── 音の進む方向 ──────→
```

この「密」と「疎」の繰り返しが波のように進んでいく。これが **音波（おんぱ）** です。

> **藤井メモ:** 音波は「縦波（たてなみ）」と呼ばれます。空気の粒が音の進む方向と同じ方向に前後に揺れるからです。水面の波（横波）とは違います。でも今は「音＝空気の密と疎の波」とだけ覚えてくれれば大丈夫。

## 2. 音はどうやって伝わるか（15分）

### 空気がないと音は伝わらない

宇宙空間は真空（空気がない）なので、音は伝わりません。SF映画で宇宙船の爆発音が聞こえるのは演出です。

音が伝わるには **「振動を伝える物質」が必要** です。空気、水、木、コンクリート......なんでもいいのですが、何かがないとダメ。

### 音の速さ

空気中の音の速さは **約340メートル/秒** です。

これがどれくらいかというと：

```
  ┌──────────────────────────────────────┐
  │         教会の礼拝堂（約20m）          │
  │                                      │
  │  [スピーカー]                    [後列] │
  │      ↓                               │
  │      ========== 約20m ==========>    │
  │                                      │
  │      音が届くまで: 約0.06秒           │
  │      （60ミリ秒）                     │
  └──────────────────────────────────────┘
```

20メートルなら約0.06秒。一見「すぐ届く」ように思えますよね。

でもこれが大きなホール（50m）になると約0.15秒。0.15秒の遅れは音楽だと結構気になります。テンポ120BPMの曲で、16分音符1個分くらいのズレです。

> **藤井メモ:** 音速は温度で少し変わります。暖かい日は少し速く（約350m/s）、寒い日は少し遅く（約330m/s）。でも礼拝堂くらいの距離なら、気にしなくて大丈夫です。

### 音は「広がる」

スピーカーから出た音は、まっすぐ前だけに飛んでいくわけではありません。ある程度広がります。

```
          （上から見た図）

              /  /  /
            /  /  /
    [SPK] ======/=========> メインの方向
            \  \  \
              \  \  \

    音はスピーカーの正面が一番強いけど、
    斜めにも広がっていく
```

この「どれくらい広がるか」はスピーカーの設計によって違います。教会のメインスピーカーは、会衆席全体をカバーするように角度が設計されています。

### 音は「反射」する

壁、天井、床にぶつかると、音は跳ね返ります。

```
    [スピーカー] ──直接音──→ [あなたの耳]
         │                      ↑
         │                      │
         └──→ [壁] ──反射音──→──┘
```

あなたの耳には **直接音**（まっすぐ届く音）と **反射音**（壁や天井で跳ね返った音）の両方が届きます。

反射音は壁まで行って帰ってくるぶん、直接音より少し遅れて届きます。この遅れが大きいと「エコー」になります。小さいと「残響（リバーブ）」として、音に広がりや温かみを加えます。

## 3. 耳はどうやって音を聞いているか（10分）

### 耳の仕組み（超シンプル版）

```
  空気の振動
      ↓
  ┌─────────┐    ┌─────────┐    ┌─────────┐
  │ 外耳    │ →  │ 中耳    │ →  │ 内耳    │ → 脳
  │（耳たぶ）│    │（鼓膜+  │    │（蝸牛） │
  │         │    │ 小さな骨）│    │         │
  └─────────┘    └─────────┘    └─────────┘
   音を集める    振動を増幅     振動を電気信号
                               に変換
```

1. **外耳（がいじ）**: 耳たぶが音を集めて、耳の穴に送り込む
2. **中耳（ちゅうじ）**: 鼓膜が振動する → 小さな骨が振動を増幅する
3. **内耳（ないじ）**: 蝸牛（かぎゅう）という巻貝みたいな器官が、振動を電気信号に変換 → 脳に送る

脳が電気信号を「音」として認識する。

### PA操作者として知っておくべきこと

- **耳は疲れる**: 長時間大きな音を聞くと、耳が鈍くなる（一時的な聴力低下）
- **耳は慣れる**: 同じ音量でもしばらく聞いていると「普通」に感じてしまう
- **耳は2つある**: 左右の耳に届く音の微妙な違いで、音の方向を判断している

> **藤井メモ:** PA操作をしていると「もうちょっと上げたほうがいいかな」と思ってどんどん音量を上げがち。これは耳が慣れてしまうから。定期的に耳を休める（30秒くらい耳を塞ぐ）と、リセットされて正しい判断ができるようになります。

## 4. 教会の礼拝堂で起きていること（10分）

### なぜ前の席と後ろの席で音が違うのか

今日学んだことを使って、教会の礼拝堂で何が起きているか考えてみましょう。

```
  ┌──────────────────────────────────────────────┐
  │                教会の礼拝堂                    │
  │                                              │
  │   [L]                              [R]       │
  │   SPK                              SPK       │
  │    ↓↓↓                            ↓↓↓       │
  │                                              │
  │   ■■■  前列  ■■■     ← 直接音が強い         │
  │                          反射音が少ない       │
  │   ■■■  中列  ■■■     ← バランスが良い       │
  │                                              │
  │   ■■■  後列  ■■■     ← 直接音が弱い         │
  │                          反射音が多い         │
  │                                              │
  │   [壁]                             [壁]      │
  └──────────────────────────────────────────────┘
```

**前列の人が聞いている音:**
- スピーカーからの直接音が強い
- 壁からの反射音はまだ少ない
- 結果: クリアだけど、ちょっと音が強すぎることも

**後列の人が聞いている音:**
- スピーカーからの直接音が弱くなっている（距離があるから）
- 壁や天井からの反射音がたくさん混ざる
- 結果: モヤっとして、言葉が聞き取りにくいことがある

### 距離と音量の関係

音は距離が **2倍** になると、強さが **1/4** になります。

```
  [SPK] ──2m──→ [A さん]    音量: ████████ 100%
  [SPK] ──4m──→ [B さん]    音量: ████     25%
  [SPK] ──8m──→ [C さん]    音量: ██       6%
```

前列（2m）と後列（8m）で、音の強さがこんなに違う！ これがPA操作者の大きな課題です。前の人には十分聞こえても、後ろの人には足りない。でも後ろに合わせて上げると、前の人にはうるさすぎる。

> **藤井メモ:** これを解決するためにスピーカーの設計や配置を工夫します。教会のメインスピーカーが高い位置に設置されているのは、前列と後列の距離の差を小さくするためでもあるんです。

### サブウーファーの特別な性質

教会にはサブウーファー（低い音専用のスピーカー）がありますね。低い音は高い音と違って、壁や障害物を回り込んで進む性質があります。

```
  高い音:  →→→ [壁] ×（壁で止まりやすい）

  低い音:  →→→ [壁] →→→（回り込んで裏側にも届く）
```

だから、ベースやバスドラムの「ズンズン」という音は、どの席にいても結構均等に聞こえます。一方、ボーカルの子音（「さ行」「た行」など高い成分）は後ろの席に届きにくい。

## 実践ドリル（10分）

### ドリル 1: 音の伝わり方を体験する

教会の礼拝堂で試してみてください：

1. ステージ上で手を叩く
2. 前列に座って同じことをする
3. 後列に座って同じことをする

それぞれ「パン！」と聞こえてから、壁からの反射音（残響）がどれくらい続くか聞いてみてください。後ろの席のほうが反射音が多く聞こえるはずです。

### ドリル 2: 音源を当てるゲーム

友達と2人でやってみてください：

1. 目を閉じる
2. 相手に礼拝堂の色々な場所から手を叩いてもらう
3. どの方向から聞こえたか当てる

左右の方向は当てやすいけど、前後や上下は意外と難しいはずです。これは左右の耳の「時間差」で方向を判断しているからです。

## 教会での応用

今日学んだことは、PA操作の全てに繋がっています：

1. **「音＝空気の振動」** → マイクは振動をキャッチする機械。スピーカーは振動を作り出す機械。ミキサーはその間を繋ぐもの
2. **「音は広がる・反射する」** → スピーカーの向きや位置が重要な理由。ハウリング（キーン！）が起きる原因
3. **「前列と後列で音が違う」** → PA操作者は自分の席（だいたい中央〜後方）だけでなく、全席のバランスを想像する必要がある
4. **「耳は慣れる・疲れる」** → 礼拝前のサウンドチェックと、礼拝中の微調整は違う判断が必要

> **藤井メモ:** 今日のレッスンで一番大事なのは「音には必ず理由がある」ということ。「なんか変だな」と思ったら、それは空気の振動がどこかでおかしなことになっている証拠。原因を探す第一歩は、音がどこから来て、どこを通って、どこに届いているかを想像すること。

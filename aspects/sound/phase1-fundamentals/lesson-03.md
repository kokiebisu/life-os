## 1. 音の大きさ ＝ 振動の幅（10分）

### 振幅（しんぷく）

前回、空気の「密」と「疎」の繰り返しが音だと学びました。この「密」の度合い、つまり **空気がどれだけ強く圧縮されるか** が「音の大きさ」を決めます。

```
  小さい音（振幅が小さい）:

  ──────╱╲──────╱╲──────╱╲──────
        ╲╱      ╲╱      ╲╱
  ← 空気の圧力変化が小さい →


  大きい音（振幅が大きい）:

      ╱╲          ╱╲          ╱╲
     ╱  ╲        ╱  ╲        ╱  ╲
  ──╱    ╲──────╱    ╲──────╱    ╲──
          ╲    ╱      ╲    ╱
           ╲  ╱        ╲  ╱
            ╲╱          ╲╱
  ← 空気の圧力変化が大きい →
```

周波数（波の「間隔」）は同じでも、振幅（波の「高さ」）が大きいほど音が大きい。

これを **音圧（おんあつ）** と呼びます。音圧が大きい＝音が大きい。

## 2. なぜデシベルを使うのか（15分）

### 普通の数字だと大変なことになる

人間の耳が聞き取れる一番小さい音（聞こえるギリギリ）と、痛みを感じる大きさの音。この差を普通の数字（音圧）で表すと：

```
  聞こえるギリギリの音圧:  0.00002 パスカル
  痛みを感じる音圧:        20 パスカル

  その差: 1,000,000倍（100万倍）
```

日常会話で「この音は0.002パスカルで、あの音は0.2パスカルです」なんて言ってられません。桁が違いすぎて扱いにくいんです。

### デシベルのアイデア

そこで考え出されたのが **デシベル（dB）** です。

100万倍の差を、**0〜120くらいの数字** に圧縮して表現します。

```
  普通の数字:                    デシベル:

  1倍  ──→                      0 dB
  10倍 ──→                      20 dB
  100倍 ──→                     40 dB
  1,000倍 ──→                   60 dB
  10,000倍 ──→                  80 dB
  100,000倍 ──→                 100 dB
  1,000,000倍 ──→               120 dB
```

ポイントは **「10倍になるごとに20dB増える」** ということ。

> **藤井メモ:** 数学的には「対数（ログ）」を使っているのですが、数式は覚えなくて大丈夫。大事なのは感覚です。「dBの数字が大きいほど音が大きい」「+20dBは10倍の差」。これだけ覚えてください。

### 人間の耳もデシベルと相性がいい

実は、人間の耳の感覚も「倍々」で変化します。

- 音圧が2倍になっても、「2倍うるさくなった」とは感じない
- だいたい「10倍（+20dB）」になって、やっと「おー、だいぶ大きくなったな」と感じる
- **+10dB で「だいたい2倍の大きさに聞こえる」** というのが経験則

```
  +3 dB  →  「あれ、ちょっと上がった？」（かろうじてわかる変化）
  +6 dB  →  「お、上がったね」（はっきりわかる変化）
  +10 dB →  「おお、倍くらいになった」（体感で2倍）
  +20 dB →  「うわ、めっちゃ大きくなった！」（体感で4倍）
```

だから、人間の感覚に合わせた単位としてdBが便利なんです。

## 3. 日常の音をデシベルで知る --- dB SPL（10分）

### dB SPL（Sound Pressure Level）

日常の音の大きさを表すときは **dB SPL** を使います。SPL は「音圧レベル」の略。「人間の聞こえるギリギリ」を 0 dB SPL として、そこからの差をdBで表します。

```
  ┌─────────────────────────────────────────────────┐
  │                 日常の音量ガイド                    │
  ├──────────┬──────────────────────────────────────┤
  │  dB SPL  │  音の例                               │
  ├──────────┼──────────────────────────────────────┤
  │   0      │  聞こえるギリギリ（無響室の静寂）       │
  │  20      │  木の葉のそよぎ、深夜の寝室             │
  │  30      │  ささやき声                            │
  │  40      │  静かな図書館                          │
  │  50      │  静かなオフィス、エアコンの音           │
  │  60      │  普通の会話（1mの距離）                 │
  │  70      │  掃除機、にぎやかなレストラン           │
  │  80      │  ピアノの練習、交通量の多い道路         │
  │  85      │  ★ここから上は長時間で聴覚にダメージ★    │
  │  90      │  カラオケ店内、芝刈り機                 │
  │  95      │  教会の礼拝（賛美が盛り上がる場面）     │
  │ 100      │  ライブハウス、電車のガード下           │
  │ 110      │  ロックコンサート最前列                 │
  │ 120      │  痛みを感じる（ジェットエンジン近く）   │
  │ 130+     │  聴覚に即座にダメージ                  │
  └──────────┴──────────────────────────────────────┘
```

### 教会の礼拝での目安

```
  礼拝の場面別の音量:

  説教（話し声）:         65-75 dB SPL
  賛美（静かな曲）:       75-85 dB SPL
  賛美（盛り上がり）:     85-95 dB SPL
  ドラムだけ（生音）:     80-100 dB SPL

  ▲ 注意ライン: 85 dB SPL
  （これを超えると長時間で聴覚にダメージの可能性）
```

> **藤井メモ:** 教会のPA操作で「音量が大きすぎる」というクレームが来ることがあります。特に高齢の方は高音域が聞こえにくくなっているので、高音が強いと「うるさい」と感じやすい。単純に音量を下げるだけでなく、周波数のバランス（前回の内容）も関係します。

## 4. ミキサーのデシベル --- フェーダーの数字（15分）

### ミキサーの dB は「基準との差」

ミキサーのフェーダー（音量スライダー）に書いてある数字を見たことがありますか？

```
  ミキサーのフェーダー:

       +10 ─── ほぼ使わない（音が歪む危険）
        +5
    ──→  0 ─── 基準レベル（Unity Gain）★
        -5
       -10
       -15
       -20
       -30
       -40
       -∞ ─── 完全に音が出ない（ミュートと同じ）
```

ここでの「0 dB」は「音がゼロ」という意味ではありません。**「設計上の基準レベル」** という意味です。「ユニティゲイン（Unity Gain）」とも呼ばれます。

- **0 dB**: 入ってきた信号をそのままの大きさで通す
- **-10 dB**: 入ってきた信号を少し小さくして通す
- **+5 dB**: 入ってきた信号を少し大きくして通す

### 教会のデジタルミキサーでの目安

```
  各チャンネルのフェーダー設定イメージ:

  ボーカルマイク 1:    [-5 ]  ← 声量によって調整
  ボーカルマイク 2:    [-8 ]
  ボーカルマイク 3:    [-6 ]
  ボーカルマイク 4:    [-3 ]
  説教マイク:          [-5 ]
  キーボード L:        [-10]  ← 楽器は声より控えめ
  キーボード R:        [-10]
  アコースティックGt:  [-8 ]
  エレキギター:        [-12]
  ベース:              [-10]
  ドラム（キック）:    [-8 ]
  iPhone (BGM):        [-15]  ← BGMは控えめ
  PC (カウントダウン): [-15]

  メインフェーダー:    [ 0 ]  ← 基準レベル
```

> **藤井メモ:** これはあくまで一般的なイメージです。実際の設定は教会の部屋の大きさ、スピーカーの性能、楽器の音量によって全く違います。でも大事なポイントは「全部を0dBに揃えるのではなく、バランスを取る」ということ。ボーカルが一番聞こえるべきなので、楽器より少し大きめに設定します。

### メーターの読み方

デジタルミキサーには音量メーターが表示されます。

```
  メーター表示:

  ████████████████████ CLIP!  ← ここまで来たら歪む！
  ██████████████████   +6
  ████████████████     +3
  ██████████████       0     ← ここを目安にする
  ████████████        -6
  ██████████         -12
  ████████           -18
  ██████             -24
  ████               -30
  ██                 -40
  │                  -∞

  色の目安:
  ■ 赤（CLIP/+6以上）: 危険！音が歪む。すぐ下げる
  ■ 黄/橙（0〜+3）:    ギリギリ。ちょっと余裕がほしい
  ■ 緑（-18〜0）:      正常。ここにいてほしい
  ■ 緑（-18以下）:     小さすぎ。もう少し上げてもOK
```

メーターのピーク（一番上まで来た瞬間）が **0dB 付近** に来るのが理想的です。常に赤に張り付いていたら音が歪んでいるサイン。逆に常に -30dB 以下だと、信号が弱すぎてノイズが目立ちます。

## 5. ダイナミックレンジ（10分）

### 一番小さい音と一番大きい音の差

**ダイナミックレンジ ＝ 最も静かな部分と最も大きな部分の音量差**

```
  礼拝の音量変化:

  音量
  (dB)
   95 │                              ╱╲  ← 賛美のクライマックス
   90 │                    ╱╲      ╱    ╲
   85 │            ╱──╲  ╱    ╲  ╱        ╲
   80 │          ╱      ╲╱      ╲╱          ╲
   75 │  ╱──╲  ╱                              ╲──╲
   70 │╱    ╲╱    ← 静かな賛美                     ╲
   65 │              ← 説教                          ╲
   60 │                                               ╲
      └──────────────────────────────────────────────────→
       開始  賛美    説教     賛美（盛り上がり）    終了
                                                    時間

  ダイナミックレンジ: 約 30 dB
  （60 dB SPL 〜 95 dB SPL）
```

礼拝の中で、一番静かな瞬間（祈りのとき）と一番大きな瞬間（賛美のクライマックス）の差は 25〜30 dB くらいあります。

### なぜPA操作者がこれを管理する必要があるのか

人間の耳は素晴らしい機能を持っていますが、限界があります：

1. **小さすぎると聞こえない**: 説教が 60 dB SPL だと、後ろの席ではエアコンの音に埋もれる
2. **大きすぎると不快・有害**: 95 dB SPL を超えると「うるさい」と感じる人が出る
3. **差が大きすぎるとどちらかが犠牲になる**: 静かな部分に合わせると盛り上がりが物足りない。大きな部分に合わせると静かな部分が聞こえない

```
  PA操作者の仕事:

  音量
   95 │                    ╱ ← ここを超えないように抑える
  ─ ─│─ ─ ─ ─ ─ ─ ─ ─ ─╱─ ─ ─ ─ ─  上限ライン
   85 │            ╱──╲ ╱
   80 │    ╱──╲  ╱      ╲
   75 │  ╱      ╲╱
  ─ ─│─╱─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  下限ライン
   70 │╱  ← ここを下回らないように持ち上げる
      └──────────────────────────────→

  「小さすぎず、大きすぎず」の範囲に収める
  これがPA操作者の音量管理
```

これは **コンプレッション** という概念につながります。デジタルミキサーにはコンプレッサーという機能があって、大きすぎる音を自動的に抑えてくれます。詳しくは Phase 2 以降で学びますが、今は「PA操作者はダイナミックレンジを管理する仕事」ということだけ覚えてください。

> **藤井メモ:** 礼拝の音量管理で一番大事なのは「言葉が聞こえること」。牧師の説教、賛美の歌詞。教会のPAは「音楽を楽しむ」だけでなく「メッセージを届ける」ための道具です。だから、どんなに盛り上がっても歌詞が聞こえなくなったら本末転倒。

### ライブ配信での特別な配慮

教会にはライブ配信の設備もありますね。配信ではさらに別の課題があります：

```
  会場の音:     ダイナミックレンジ 30 dB  → OK（耳が自動調整）
  配信の音:     ダイナミックレンジ 30 dB  → 問題あり！

  理由:
  - 視聴者はイヤホンやスマホのスピーカーで聞いている
  - 小さい音は周囲の雑音に埋もれる
  - 大きい音は歪むか、音量を下げてしまう
  - → 配信用はダイナミックレンジを狭める必要がある（15-20 dB 程度に）
```

配信用のミックスでは、メイン出力とは別に音量差を小さくしたバランスが必要です。デジタルミキサーなら「Aux（オグジュアリー）」や「バス」を使って配信用の別ミックスを作れます。

## 実践ドリル（10分）

### ドリル 1: 音量を当てよう

スマホの騒音計アプリ（無料のもので OK）をダウンロードして、教会で以下の音量を測ってみてください：

1. 誰もいない礼拝堂の静けさ（エアコンの音）
2. 普通の声で話したとき（1m離れて）
3. 礼拝中の賛美（一番静かなとき）
4. 礼拝中の賛美（一番大きなとき）

> **藤井メモ:** スマホの騒音計アプリは精度が完璧ではありませんが、「だいたいの感覚を掴む」には十分です。プロはもっと正確な測定器を使いますが、まずは体感を数字と結びつけることが大事。

### ドリル 2: ミキサーのフェーダーとメーターを観察する

礼拝前のリハーサル時、ミキサーの前に座って以下を観察してください：

1. 各チャンネルのフェーダーの位置（0 より上？下？）
2. メーターの動き（緑の範囲？黄色まで行く？赤は？）
3. 曲の静かな部分と大きな部分でメーターがどう変わるか

触らなくて大丈夫です。まず「見る」だけ。

## 教会での応用

今日学んだ dB の知識は、PA操作の基本中の基本です：

1. **フェーダー操作**: 「ボーカルを3dB上げる」と言われたら、フェーダーをほんの少し上げるだけ（3dBは「ちょっと上がったかな」くらいの変化）
2. **音割れ（クリップ）の防止**: メーターが赤に入ったら、即座にフェーダーを下げる。音が歪んでいる＝聞く人にとって不快
3. **会場全体の音量管理**: 盛り上がりのピークで 90-95 dB SPL を目安に。これを超えないようにメインフェーダーで調整
4. **配信用ミックス**: 会場用とは別のバランスが必要。ダイナミックレンジを抑えて、どんな環境でも聞きやすく

> **藤井メモ:** 最後にもう一つ。「音量を上げれば解決する」と思いがちですが、音量を上げると問題も一緒に大きくなります。ハウリング（キーン！）のリスクも上がる。まずは「不要な音を減らす」（前回のEQの考え方）、次に「必要な音を適切な範囲で上げる」。引き算が先、足し算は後。これが鉄則です。

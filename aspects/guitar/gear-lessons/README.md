機材の信号経路・音作り・ケアを学ぶ。

## 推奨タイミング

```
Phase 1-2  : (メインカリキュラムに集中)
Phase 3 前 : G1（シグナルチェーン入門）← エレキの音作りに入る前に
Phase 3    : G2（Strat のコントロール）
Phase 3-4  : G3（UAFX で音を作る）
BigSky購入後: G4（リバーブを操る）
いつでも    : G5（アコギの録音 & ケア）
```

## レッスン構成

メインカリキュラムの4ステップを踏襲しつつ、TAB の代わりに「まず聴く（体験）」:

1. **まず聴く（体験）** — 音の違いを実際に聴いて感じる
2. **解説** — なぜその音になるかを理解する
3. **ドリル** — セッティングチャレンジ、引き算テスト
4. **まとめ** — セッティングシート作成、リファレンス表

# Gear G1: シグナルチェーン入門 — 音の旅を理解する

番外編 / G1
担当: 橋本 海
所要時間: 約45分
前提: なし（いつでも受講可。Phase 3 前がおすすめ）

---

## はじめに

> 橋本: 今日はギターを弾く前に、「音がどこを通って耳に届くか」を一緒に見ていこう。ギター → ケーブル → エフェクター → オーディオIF → ヘッドフォン。この順番を「シグナルチェーン（信号経路）」って呼ぶ。チェーンのどこかを変えると音が変わる。まずは全体像を掴もう。

---

## ① まず聴く（体験）— 引き算ドリル

### STEP 1: フルチェーンで音を出す

エレキの場合の接続順:

```
Strat → ケーブル → UAFX Enigmatic '82 → ケーブル → Volt 176 → HD 490 Pro
```

> 橋本: まずこの状態でコードをジャーンと弾いてみて。UAFX はクリーン寄りのセッティングで、Volt のコンプレッサーは OFF。どんな音が聴こえる？

**やること:** 開放弦をジャーンと鳴らして、音を覚える

---

### STEP 2: UAFX を外して弾く

```
Strat → ケーブル → Volt 176 → HD 490 Pro
```

> 橋本: UAFX を外して、ギターを直接 Volt に繋いでみて。同じコードを弾いて。聴いてみて、何を感じた？ さっきとどう違う？

**やること:** 同じコードを弾いて、UAFX あり/なしの音の違いを耳で確認する

> 橋本: UAFX はプリアンプ。ギターの小さな信号を増幅して、アンプの「太さ」や「キャラクター」を加える。外すと、ギターの生の信号がそのまま Volt に入る。細くて硬い音に聴こえたはず。

---

### STEP 3: ヘッドフォンを片耳だけ外す

> 橋本: 今度は片耳だけヘッドフォンを外して弾いてみて。右耳で HD 490 Pro の音、左耳でギターの生音（エレキのペチペチした音）。この違いが「シグナルチェーンを通った音」と「通ってない音」の差。

**やること:** 片耳でヘッドフォン、片耳で生音。同じフレーズを弾いて違いを体感する

---

## ② 解説 — シグナルチェーンの各パーツ

### エレキギターの経路

```
┌──────────┐    ┌────────┐    ┌──────────────┐    ┌──────────┐    ┌──────────┐
│ Strat    │ →  │ ケーブル │ →  │ UAFX         │ →  │ Volt 176 │ →  │ HD 490   │
│ (PU)     │    │ (信号線) │    │ (プリアンプ)  │    │ (録音/出力)│   │ (モニター) │
└──────────┘    └────────┘    └──────────────┘    └──────────┘    └──────────┘
  音を拾う        音を運ぶ       音を太くする        音を受け取る      音を聴く
                              音のキャラを付ける    コンプで整える
```

**各パーツの役割:**

```
パーツ           役割                        君の機材
──────────────────────────────────────────────────────────────
ギター(PU)       弦の振動を電気信号に変換      Josefina Fat 60s
ケーブル          信号を次の機器に運ぶ          Analysis Plus Yellow Oval
プリアンプ        信号を増幅+音のキャラを付ける  UAFX Enigmatic '82
オーディオIF      PCに取り込む/ヘッドフォンに出す Volt 176（176コンプ内蔵）
ヘッドフォン      音を聴く                     HD 490 Pro
```

> 橋本: ポイントは「前段で出した音を後段が受け取る」ということ。ギターで拾った音にプリアンプでキャラを付けて、Volt で出力する。順番が大事で、これを入れ替えると音が大きく変わる。

---

### アコギの経路

```
┌──────────────┐    ┌────────┐    ┌──────────┐    ┌──────────┐
│ Martin       │ →  │ ケーブル │ →  │ Volt 176 │ →  │ HD 490   │
│ (Fishman PU) │    │ (信号線) │    │ (録音/出力)│   │ (モニター) │
└──────────────┘    └────────┘    └──────────┘    └──────────┘
  弦の振動を拾う       運ぶ         受け取る          聴く
```

> 橋本: アコギの場合は Martin に内蔵されている Fishman Gold Plus（アンダーサドルピックアップ）が音を拾って、直接 Volt に入れる。アコギにはプリアンプを通さないのが基本。Fishman 自体にプリアンプが内蔵されているから、外部プリアンプは不要。

---

### 「なぜこの順番なのか」

```
NGな順番の例:

  Strat → Volt → UAFX → HD 490
                  ↑
                  Volt の出力は「ヘッドフォン/スピーカー用」。
                  ここに UAFX を繋いでも、信号レベルが合わない。
```

> 橋本: シグナルチェーンには「信号レベル」という概念がある。ギターの出力は小さい（楽器レベル）。プリアンプで大きくして（ラインレベル）、Volt に入れる。Volt はそれをヘッドフォンで聴ける音量に変換する。順番を守らないと、音が歪んだり、逆に小さすぎて聴こえなかったりする。

## ③ ドリル — 引き算テスト

### ドリル 1: 「何が変わった？」当てクイズ

> 橋本: これから3つの接続パターンで同じフレーズを弾く。それぞれ何が違うか、耳で当ててみて。

**パターン A:**
```
Strat → UAFX(クリーン) → Volt → HD 490
```

**パターン B:**
```
Strat → Volt → HD 490（UAFX なし）
```

**パターン C:**
```
Strat → UAFX(クランチ) → Volt → HD 490
```

**やること:**
1. パターン A で開放弦をジャーンと鳴らす（5秒聴く）
2. パターン B に繋ぎ替えて同じことをする（5秒聴く）
3. パターン C に繋ぎ替えて同じことをする（5秒聴く）
4. 各パターンの音の違いを自分の言葉で説明してみる

> 橋本: 「太い/細い」「暖かい/冷たい」「歪んでる/クリーン」…どんな言葉でもいい。自分の耳で感じたことを言葉にすることが大事。

---

### ドリル 2: Volt 176 コンプレッサーの ON/OFF

> 橋本: Volt 176 にはコンプレッサー（176 Compressor）が内蔵されてる。フロントパネルのボタンで ON/OFF できる。

**やること:**
1. UAFX をクリーンにセットして接続
2. コンプ OFF でフィンガーピッキング（強弱をつけて）
3. コンプ ON で同じフレーズ
4. 違いを感じる

> 橋本: コンプレッサーは「音量の差を縮める」道具。強く弾いた音を抑えて、弱く弾いた音を持ち上げる。ON にすると音が均一になって「まとまった」印象になるはず。

---

### ドリル 3: ケーブルを意識する

> 橋本: Analysis Plus のケーブルを使ってるよね。ケーブルは「変わらないもの」と思いがちだけど、実は長さや品質で音が変わる。今日はケーブルの抜き差しだけやってみよう。

**やること:**
1. ケーブルがしっかり刺さっているか確認する（ガリノイズの原因はだいたい接触不良）
2. ケーブルを抜く → 刺す → 音を出す。ノイズがないことを確認
3. ケーブルの根元を軽く動かしてみる → ガリっと音がしたら接触不良のサイン

> 橋本: ケーブルは消耗品。ライブ前に確認する癖をつけると、本番でトラブらない。

## ④ まとめ — シグナルチェーン・リファレンス

### エレキ基本チェーン

```
Strat (Josefina Fat 60s)
  → Analysis Plus Yellow Oval
    → UAFX Enigmatic '82
      → Analysis Plus Yellow Oval
        → Volt 176 (INST入力)
          → HD 490 Pro
```

### アコギ基本チェーン

```
Martin Custom (Fishman Gold Plus)
  → Analysis Plus Yellow Oval
    → Volt 176 (INST入力)
      → HD 490 Pro
```

### 将来: BigSky 追加後のチェーン

```
Strat
  → UAFX Enigmatic '82（プリアンプ）
    → BigSky（リバーブ）
      → Volt 176
        → HD 490 Pro
```

> 橋本: プリアンプの後にリバーブ。「音のキャラを作ってから空間をつける」のが基本の順番。逆にすると、リバーブのかかった音がさらに歪んで濁る。

### 覚えておくこと

```
1. シグナルチェーン = 音の通り道。順番が大事
2. ギター → プリアンプ → （リバーブ）→ オーディオIF → ヘッドフォン
3. 各パーツを ON/OFF して音の違いを聴く = 「引き算」で理解する
4. ケーブルの接触不良はノイズの一番の原因
5. アコギは Fishman 内蔵プリアンプがあるので外部プリアンプ不要
```

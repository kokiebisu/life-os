番外編 / G5

## はじめに

> 橋本: Martin Custom Shop は一生ものの楽器。今日は2つのテーマ。前半は Fishman を使った録音のコツ、後半は Martin を最高の状態に保つためのケア方法。手入れを知ることで楽器への愛着がもっと深まるよ。

---

## ① まず聴く（体験）— Fishman の音を知る

### STEP 1: Fishman Gold Plus で音を出す

**接続:**
```
Martin (Fishman Gold Plus) → ケーブル → Volt 176 (INST入力) → HD 490 Pro
```

> 橋本: Martin のサウンドホール横にあるエンドピンジャックにケーブルを挿して。Fishman Gold Plus Natural I はアンダーサドルピエゾピックアップ。サドル（弦が乗ってるパーツ）の下に薄いセンサーが入ってて、弦の振動を直接拾う。

**やること:** 開放弦をジャーンと弾く → ヘッドフォンで聴こえる音を確認

---

### STEP 2: ゲイン調整

> 橋本: Volt の GAIN ノブを回して、適切な入力レベルを探そう。

**やること:**
1. GAIN を 9時あたりから始める
2. 一番強くストラムして、Volt のメーターが黄色に届くくらいに合わせる
3. 赤く点灯したら GAIN を少し下げる（クリッピング = 音が割れてる）
4. 普通の強さで弾いたとき、緑〜黄色の間で動くのがベスト

```
Volt メーターの目安:

  🔴 赤:  クリッピング（音割れ）→ GAIN を下げる
  🟡 黄:  ピーク時にここまで来るのが理想
  🟢 緑:  通常の演奏で動く範囲
```

> 橋本: ゲインが低すぎると音が小さくてノイズが目立つ。高すぎると音が割れる。「一番強く弾いたときにギリギリ黄色」がベスト。

---

### STEP 3: コンプレッサーの効果

> 橋本: Volt 176 のコンプレッサーをアコギで試してみよう。

**やること:**
1. コンプ OFF でフィンガーピッキング → 強弱の差が大きい
2. コンプ ON で同じフレーズ → 音量が均一になる
3. コンプ OFF でストラム → ダイナミック
4. コンプ ON でストラム → まとまる

> 橋本: アコギの場合、コンプは好みが分かれる。ストラムではコンプ ON で音が整うけど、フィンガーピッキングの繊細な強弱を活かしたいなら OFF の方がいいこともある。状況に応じて使い分けて。

## ② 解説 — Fishman Gold Plus の特性

### ピエゾ PU の仕組み

```
弦 ────────────────────── 弦
       ↓ 振動が伝わる
┌─────────────────────┐
│    サドル（牛骨）      │
├─────────────────────┤
│  Fishman Gold Plus   │  ← アンダーサドルピエゾ
│  (圧電センサー)        │     弦の圧力を電気信号に変換
├─────────────────────┤
│    ブリッジ            │
└─────────────────────┘
```

> 橋本: ピエゾピックアップは弦の「振動」というより「圧力」を拾う。だからエレキのマグネティック PU（磁石で拾う）とは音の特性が違う。ピエゾはアタック（弾いた瞬間の音）がハッキリ出るのが特徴。

### Fishman のボリュームコントロール

> 橋本: Martin のエンドピンジャック付近にボリュームホイールがある。これが Fishman の出力レベルを調整する。Volt の GAIN と合わせて、2段階で音量を調整できる。

```
調整の順番:
1. Fishman のボリューム → 最大近く（80-100%）にしておく
2. Volt の GAIN → メーターが黄色に届くように調整
```

> 橋本: Fishman のボリュームを下げすぎると、信号が小さくなって Volt 側で GAIN を上げなきゃいけなくなる。するとノイズも一緒に増幅されてしまう。Fishman は高めにして、Volt で微調整するのがベスト。

## ③ ドリル — Martin のケア

### ドリル 1: Boveda で湿度管理

> 橋本: Martin はスプルース（トップ）とローズウッド（サイド&バック）の無垢材。湿度の影響を受けやすい。乾燥しすぎるとトップが割れたり、フレットが飛び出したりする。

**Boveda の使い方:**
```
1. Boveda パック（49%RH 推奨）をケースのポケットに入れる
2. 弾き終わったらケースに戻す
3. Boveda が硬くなったら交換時期（通常2-4ヶ月）

保管の基本:
  ✅ 弾かないときはハードケースに保管
  ✅ Boveda を入れておく
  ❌ スタンドに出しっぱなし（アコギは湿度リスクが高い）
  ❌ エアコンの風が直接当たる場所
  ❌ 直射日光
```

> 橋本: 横浜は夏に湿度が上がりやすく、冬は乾燥しやすい。年間を通して Boveda をケースに入れておくのが一番安心。

---

### ドリル 2: 弦交換のタイミングと手順

> 橋本: 弦は消耗品。音がくすんできたら交換時期。目安は弾く頻度にもよるけど、2週間〜1ヶ月くらい。

**弦交換のサイン:**
```
✅ 交換すべきとき:
  - 音のキラキラ感がなくなった
  - 弦が黒ずんでいる
  - チューニングが安定しにくくなった
  - ライブや人前で弾く予定がある

弦の仕様（Martin Custom 用）:
  ゲージ: Light（012-054 が一般的）
  素材: 80/20 ブロンズ or フォスファーブロンズ
```

**交換時の注意:**
```
1. 全部一気に外さない（ネックのテンションが急変する）
2. 1本ずつ外して → 1本ずつ張る（6弦→1弦の順）
3. 張ったらチューニング → 軽く弾く → もう一度チューニング
4. 新しい弦は伸びるので、最初の1-2日はチューニングが不安定（正常）
```

---

### ドリル 3: 指板のケア

> 橋本: ローズウッドの指板は乾燥するとひび割れることがある。弦交換のタイミングで指板オイルを塗ろう。

**手順:**
```
1. 弦を外す
2. 乾いたクロスで指板の汚れを拭き取る
3. 指板オイル（レモンオイルなど）を少量クロスに取る
4. 指板全体に薄く塗り広げる（フレットの間も）
5. 5分待って余分なオイルを拭き取る
6. 新しい弦を張る

頻度: 弦交換2-3回に1回（月1回程度）
```

> 橋本: エボニー指板（Martin の指板材）は密度が高いのでローズウッドほど乾燥しにくいけど、ケアしておくに越したことはない。

---

### 補足: Strat のケア

> 橋本: Strat のケアも簡単に触れておくね。

```
Strat（Heavy Relic ラッカー塗装）のケア:

  ✅ やること:
    - 弾き終わったらクロスで軽く拭く
    - スタンドに立てるとき、ゴム部分が塗装に当たる面を確認
      （ラッカー対応スタンドを使う — ゴムがラッカーを溶かす場合あり）
    - 弦交換時にフレットをクロスで拭く

  ❌ やらないこと:
    - ポリッシュの使いすぎ（Heavy Relic の風合いが変わる）
    - シリコン系のポリッシュ（ラッカーに悪影響）

  弦の仕様:
    ゲージ: 010-046
    交換頻度: 2週間〜1ヶ月
```

> 橋本: Heavy Relic はもともと「使い込んだ風合い」がコンセプトだから、神経質になりすぎなくていい。でも基本的な手入れはやっておこう。

## ④ まとめ — 録音 & ケア・リファレンス

### アコギ録音チェックリスト

```
1. Fishman ボリューム → 80-100%
2. Volt INST 入力に接続
3. Volt GAIN → 最強ストラムで黄色ギリギリ
4. コンプレッサー → ストラムなら ON、フィンガーなら好み
5. ヘッドフォンで音を確認してから弾き始める
```

### ケアスケジュール

```
毎日:
  - 弾き終わったらクロスで軽く拭く
  - Martin はハードケースに戻す + Boveda

弦交換時（2週間〜1ヶ月）:
  - 弦交換（1本ずつ）
  - 指板の汚れ拭き取り
  - 2-3回に1回: 指板オイル

定期チェック（3ヶ月ごと）:
  - Boveda の状態確認（硬くなったら交換）
  - ネックの反りチェック（目視）
  - フレットの状態確認
  - ケーブルの接触不良チェック
```

### 覚えておくこと

```
1. Fishman は高め出力 → Volt で GAIN 調整（ノイズを減らす）
2. Volt メーター: 強く弾いて黄色ギリギリが適正ゲイン
3. コンプはストラムで ON、フィンガーは好みで
4. Martin は弾かないときハードケース + Boveda（湿度管理必須）
5. 弦交換は1本ずつ（テンション急変を避ける）
6. 指板オイルは弦交換2-3回に1回
7. Strat はラッカー対応スタンド必須、ポリッシュは控えめに
```

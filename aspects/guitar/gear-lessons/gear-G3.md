# Gear G3: UAFX Enigmatic '82 実践 — アンプトーンを作る

番外編 / G3
担当: 橋本 海 + 瀬戸 涼介
所要時間: 約50分
前提: G2 完了（Strat の PU ポジション・Volume を理解していること）

---

## はじめに

> 橋本: 今日は UAFX Enigmatic '82 のノブを全部触って、自分でトーンを作れるようになろう。Enigmatic '82 は Mesa Boogie Mark IIC+ というアンプのモデリング。クリーンからハイゲインまで幅広い音が出せる。

> 瀬戸: ノブがいっぱいあると怖いけど、実は「3つの音」を作れればほとんどの場面で困らない。今日はその3つを一緒に作るよ。

---

## ① まず聴く（体験）— 3つの基本トーンを聴く

### STEP 1: クリーントーン

> 橋本: まずはクリーン。全部のノブを12時（真ん中）にリセットして、Gain だけ9時くらいまで下げてみて。

**セッティング:**
```
Gain: 9時 (低め)
Bass: 12時
Mid:  12時
Treble: 12時
Master: 12時
Presence: 12時
```

**やること:** Pos 4（ミドル+ネック）でコードを弾く → 歪まない、クリアな音

> 橋本: 聴いてみて、何を感じた？ これが UAFX の「素の音」に近い状態。Mark IIC+ はクリーンも美しいアンプなんだ。

---

### STEP 2: クランチトーン

> 瀬戸: 次は Gain を12時に上げて。ピッキングの強さで歪んだりクリーンになったりする、一番おいしいところ。

**セッティング:**
```
Gain: 12時
Bass: 12時
Mid:  12時
Treble: 12時
Master: 12時
Presence: 12時
```

**やること:** Pos 2（ブリッジ+ミドル）で強くストラムする → 歪む。軽く弾く → クリーン寄り

> 瀬戸: この「弾き方で歪みが変わる」のがクランチの醍醐味。Volume を 6-7 に下げるとさらにクリーン寄りになる。G2 で練習した Volume クリーンアップと組み合わせるんだ。

---

### STEP 3: ハイゲイントーン

> 橋本: 最後は Gain を3時（15時）まで上げて。ロック、ハードロックの音。

**セッティング:**
```
Gain: 15時 (高め)
Bass: 12時
Mid:  12時
Treble: 12時
Master: 12時
Presence: 12時
```

**やること:** Pos 1（ブリッジ）でパワーコードを弾く → しっかり歪む、ザクザクした音

> 橋本: Mark IIC+ はハイゲインでも音が潰れにくい。Metallica のジェイムズ・ヘットフィールドが使ってたことでも有名なアンプ。

## ② 解説 — 各ノブの役割

### ノブの機能一覧

```
ノブ        役割                          回す方向と効果
──────────────────────────────────────────────────────────────
Gain        歪みの量を決める               右 → 歪み増加、左 → クリーン
Bass        低域（ベース）の量              右 → ズンズン重い、左 → 軽い
Mid         中域の量                       右 → 太い・前に出る、左 → スッキリ
Treble      高域の量                       右 → キラキラ・シャリシャリ、左 → 暗い
Master      最終的な音量                    右 → 音量UP（音色には影響しにくい）
Presence    超高域の空気感                   右 → 明るく開放的、左 → 落ち着く
```

> 橋本: 重要なのは「Gain は歪み、Master は音量」という区別。Gain を上げると歪みが増えるけど音量も上がる。Master で最終的な音量を揃えるのがコツ。

> 瀬戸: EQ（Bass/Mid/Treble）は「帯域ごとの音量調整」だと思えばいい。特定の帯域を上げたり下げたりして音の「形」を作る。

---

### EQ の効き方

```
            Bass        Mid         Treble      Presence
周波数帯     低域        中域         高域        超高域
────────────────────────────────────────────────────────
上げると:    ズンと重い   太く前に出る  キラキラ     空気感・開放感
下げると:    スッキリ    引っ込む      暗い・丸い   落ち着く
```

> 橋本: 初心者がよくやるのは「全部上げる」。でも実は **下げる方が大事**。全部上げると音がゴチャゴチャになる。「何を下げるか」で音を整理するのがプロの発想。

---

## ③ ドリル — トーンメイキング

### ドリル 1: EQ の引き算

> 橋本: まずクランチセッティング（Gain 12時）で、EQ を1つずつ極端に動かして効果を体感する。

**やること:**
1. Bass を 7時（最小）まで下げて弾く → 低域スカスカを体感
2. Bass を 12時に戻す。Mid を 7時まで下げて弾く → 音が引っ込む（ドンシャリ）
3. Mid を 12時に戻す。Treble を 7時まで下げて弾く → モコモコ、暗い
4. 全部 12時に戻す

> 瀬戸: Mid を下げた音が「ドンシャリ」って呼ばれるやつ。メタル系でよく使うけど、バンドの中だと他の楽器に埋もれやすい。Mid はギターの「存在感」を出す帯域。

---

### ドリル 2: PU + UAFX の組み合わせマトリクス

> 瀬戸: PU のポジションと UAFX の Gain を組み合わせて、音のバリエーションを体感してみよう。

**やること:** 以下の組み合わせで Am7 → D7 のコードチェンジを弾く

```
組み合わせマトリクス:

              Gain 9時(クリーン)   Gain 12時(クランチ)   Gain 15時(ハイゲイン)
Pos 1(Bridge)    シャープクリーン     ロッキンクランチ      ザクザク歪み
Pos 2(B+M)       Mayer クリーン      ファンキークランチ    モダンリード
Pos 4(M+N)       甘いクリーン        ブルースクランチ      スムーズリード
Pos 5(Neck)      ジャズクリーン       ウォームクランチ     太いリード
```

> 橋本: 全部試す必要はないけど、対角線上の組み合わせ（Pos1+ハイゲイン、Pos5+クリーン）が一番音の差が大きい。そこから始めると違いがわかりやすい。

---

### ドリル 3: Volt 176 コンプレッサーとの組み合わせ

> 橋本: G1 で触った Volt のコンプレッサー。クリーンとクランチそれぞれで ON/OFF を試してみよう。

**やること:**
1. UAFX クリーン + コンプ OFF → 弾く
2. UAFX クリーン + コンプ ON → 弾く（音が均一になる）
3. UAFX クランチ + コンプ OFF → 弾く
4. UAFX クランチ + コンプ ON → 弾く（歪みが増える？変わらない？）

> 瀬戸: クリーンではコンプの効果がわかりやすい。クランチだと歪み自体がコンプレッション（音量の均一化）をするから、効果が目立たないことが多い。使いどころは「クリーンで粒を揃えたいとき」が一番。

---

## ④ まとめ — UAFX セッティング・リファレンス

### 3つの基本セッティング

```
【クリーン】        【クランチ】        【ハイゲイン】
Gain:    9時       Gain:   12時       Gain:   15時
Bass:    12時      Bass:   12時       Bass:   1時
Mid:     12時      Mid:    1時        Mid:    1時
Treble:  12時      Treble: 12時       Treble: 12時
Master:  12時      Master: 12時       Master: 11時
Presence: 12時     Presence: 12時     Presence: 1時

PU推奨:  Pos 4-5   PU推奨: Pos 2-3    PU推奨: Pos 1-2
用途:    ジャズ、   用途:   ブルース、   用途:   ロック、
        バラード          ファンク            ハードロック
```

### ノブの順番（トーンメイキングの手順）

```
1. Gain を決める     → 歪みの量（クリーン？クランチ？ハイゲイン？）
2. PU を選ぶ        → ブリッジ？ネック？ハーフトーン？
3. EQ を調整する     → Bass/Mid/Treble で音の形を整える
4. Volume で微調整   → ギター本体の Volume でクリーンアップ
5. Master で音量合わせ → 最終音量
```

### 覚えておくこと

```
1. Gain = 歪みの量、Master = 音量。混同しない
2. EQ は「上げる」より「下げる」で音を整理する
3. Mid はギターの存在感。下げすぎると埋もれる
4. PU ポジション × Gain の組み合わせで音のバリエーションが広がる
5. Volt コンプはクリーンで最も効果的
6. まず Gain → PU → EQ → Volume の順で調整する
```

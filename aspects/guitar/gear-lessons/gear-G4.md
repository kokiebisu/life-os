# Gear G4: BigSky でリバーブを操る — 空間を作る音作り

番外編 / G4
担当: 橋本 海 + 瀬戸 涼介
所要時間: 約50分
前提: G3 完了 + BigSky を入手済みであること

---

## はじめに

> 橋本: リバーブは「空間を作る」エフェクト。部屋の響き、ホールの残響、天上から降り注ぐような広がり。BigSky はその全部ができるリバーブペダル。今日は BigSky をチェーンに組み込んで、シーン別のセッティングを作ろう。

> 瀬戸: BigSky はプロのワーシップボードに必ずと言っていいほど入ってるペダル。教会で弾くなら知っておいて損はないよ。

---

## ① まず聴く（体験）— リバーブの種類を聴き比べる

### STEP 0: 接続

```
Strat → UAFX Enigmatic '82 → BigSky → Volt 176 → HD 490 Pro
```

> 橋本: プリアンプ（UAFX）の**後に**リバーブ（BigSky）を置く。G1 で学んだ通り「音のキャラを作ってから空間をつける」のが基本の順番。

**接続のポイント:**
```
UAFX の OUTPUT → パッチケーブル → BigSky の INPUT
BigSky の OUTPUT → ケーブル → Volt の INST 入力
```

---

### STEP 1: リバーブなし vs あり

> 橋本: まず BigSky を OFF（バイパス）にして Am7 をアルペジオで弾いてみて。次に ON にして同じことを弾く。

**やること:** BigSky OFF → ON を切り替えて、空間の広がりを体感する

> 橋本: リバーブがないと音が「目の前」で鳴ってる感じ。ON にすると音が「部屋の中」や「ホール」に広がる。この「空間」がリバーブの正体。

---

### STEP 2: 6つのリバーブタイプを聴き比べ

> 橋本: BigSky のノブでリバーブタイプを切り替えて、同じフレーズを弾いてみよう。

```
タイプ       音の特徴                          イメージ
────────────────────────────────────────────────────────────
Room        短い残響、近い距離感                 小さな部屋で弾いてる感じ
Hall        長い残響、広い空間                   コンサートホール
Plate       金属的な明るい響き                   金属板に反射する音（スタジオ定番）
Spring      ビヨ〜ンと揺れる、ヴィンテージ感     サーフロック、古いアンプの残響
Shimmer     オクターブ上が加わる、神秘的          天から降り注ぐような音
Swell       音が徐々に膨らむ、パッド的           アンビエント、ワーシップの間奏
```

**やること:** 各タイプで Am7 のアルペジオを弾いて、音の違いを耳で確認する（Mix を12時、Decay を12時で統一）

> 瀬戸: 全部試さなくていい。まず Room と Hall の違いを感じて、次に Shimmer を聴いてみて。その3つだけでリバーブの幅がわかるよ。

## ② 解説 — BigSky の基本操作

### 主要ノブ

```
ノブ         役割                        目安
──────────────────────────────────────────────────────────────
TYPE        リバーブの種類を選ぶ            上のセクターノブで切替
DECAY       残響の長さ                     短い(9時) → 長い(15時)
MIX         原音とリバーブ音の比率           原音寄り(9時) → リバーブ寄り(15時)
TONE        リバーブ音の明るさ              暗い(9時) → 明るい(15時)
PRE-DELAY   原音とリバーブの間の隙間         短い → 原音にまとわりつく
                                          長い → 原音とリバーブが分離する
```

> 橋本: 最初は Mix と Decay の2つだけ覚えればいい。Mix は「リバーブをどのくらい混ぜるか」、Decay は「残響がどのくらい続くか」。

---

### Mix の目安

```
Mix         効果                    用途
──────────────────────────────────────────────────────────
9時 (20%)   ほんのり残響            バッキング、歌伴
10時(30%)   しっかり空間感          ブルース、バンドの中
12時(50%)   原音とリバーブが同量    ワーシップ間奏、ソロ
15時(80%)   リバーブが主役          アンビエント、パッド的
```

> 瀬戸: バンドの中で弾くなら Mix は 20-30% くらいが無難。ソロやワーシップの静かなパートでは 40-50% まで上げることもある。上げすぎると音がぼやけるから注意。

---

### Decay の目安

```
Decay       残響時間       用途
──────────────────────────────────────────────────────
9時         短い(0.5-1秒)  タイトなバッキング
12時        中程度(2-3秒)  スタンダード
15時        長い(5秒以上)  アンビエント、パッド
最大         超長い(∞)     持続的なパッド（特殊用途）
```

## ③ ドリル — シーン別セッティング

### ドリル 1: ブルース向け — Spring リバーブ

> 瀬戸: ブルースには Spring（スプリング）が定番。古いフェンダーアンプに内蔵されてたリバーブの音。

**セッティング:**
```
TYPE:      Spring
DECAY:     10時（短め）
MIX:       10時（控えめ）
TONE:      12時
PRE-DELAY: 9時（短い）

UAFX: クランチ (Gain 12時)
PU:   Pos 2（ブリッジ+ミドル）
```

**やること:**
1. 12小節ブルース進行（A7-D7-E7）を弾く
2. Spring のビヨーンとした揺れを感じる
3. Mix を上げすぎるとサーフロックになるので注意

---

### ドリル 2: ワーシップ向け — Hall リバーブ

> 橋本: ワーシップでは Hall が王道。教会の空間を再現する広い残響。

**セッティング:**
```
TYPE:      Hall
DECAY:     1時（やや長め）
MIX:       11時（30%くらい）
TONE:      1時（やや明るめ）
PRE-DELAY: 10時

UAFX: クリーン (Gain 9時)
PU:   Pos 4（ミドル+ネック）
```

**やること:**
1. G コードから始めて、G → C → D → G のシンプルな進行を弾く
2. コードチェンジの間にリバーブの残響が残るのを感じる
3. ゆっくりストラム → リバーブが空間を満たす感覚

> 橋本: ワーシップではリバーブが「会衆を包み込む空間」を作る。派手にかけるんじゃなくて、「気づかないけどないと寂しい」くらいがちょうどいい。

---

### ドリル 3: アンビエント向け — Shimmer リバーブ

> 橋本: Shimmer はリバーブ音にオクターブ上の音を加える。神秘的で、天から降り注ぐような音になる。ワーシップの間奏や祈りのパートで使える。

**セッティング:**
```
TYPE:      Shimmer
DECAY:     2時（長め）
MIX:       12時（50%）
TONE:      12時
PRE-DELAY: 11時

UAFX: クリーン (Gain 9時)
PU:   Pos 5（ネック）
Volume: 5-6（柔らかく）
```

**やること:**
1. 単音でゆっくりメロディを弾く（Am ペンタトニックなど）
2. 1音弾くごとに、オクターブ上の音がふわっと広がるのを聴く
3. Decay を最大にして、音が消えるまで待つ

> 瀬戸: Shimmer は「弾きすぎない」のがコツ。1音弾いて、響きが広がるのを待つ。間（ま）を楽しむ。

## ④ まとめ — BigSky セッティング・リファレンス

### シーン別セッティング早見表

```
シーン        TYPE      DECAY    MIX      TONE     PRE-DELAY
────────────────────────────────────────────────────────────
ブルース      Spring    10時     10時     12時     9時
ワーシップ    Hall      1時      11時     1時      10時
アンビエント  Shimmer   2時      12時     12時     11時
バンド練習    Room      10時     9時      12時     9時
ソロ演奏      Plate     12時     11時     1時      10時
```

### BigSky の接続ルール

```
1. プリアンプの後にリバーブ: UAFX → BigSky → Volt
2. 逆にすると: リバーブ音がさらに歪んで濁る（NG）
3. パッチケーブルで最短距離接続（ノイズ対策）
```

### プリセットの保存

> 橋本: BigSky は最大300プリセットを保存できる。よく使うセッティングは保存しておくと、ライブや練習でワンタッチで呼び出せる。

```
保存方法:
1. 好みのセッティングを作る
2. フットスイッチを長押し（LEDが点滅）
3. バンク/プリセット番号を選ぶ
4. もう一度長押しで保存

おすすめの保存構成:
  プリセット 1: ワーシップ Hall
  プリセット 2: ブルース Spring
  プリセット 3: アンビエント Shimmer
```

### 覚えておくこと

```
1. リバーブは「空間を作る」エフェクト
2. 接続順: プリアンプ → リバーブ → オーディオIF（順番を守る）
3. Mix と Decay の2つが最重要ノブ
4. バンドの中では Mix 20-30% が基本（上げすぎ注意）
5. Spring = ブルース、Hall = ワーシップ、Shimmer = アンビエント
6. Shimmer は弾きすぎない。間を楽しむ
7. よく使うセッティングはプリセット保存する
```

## はじめに

> 瀬戸: L21 で Cadd9、L22 で Asus2 や Fsus2 を覚えたよね。どれもキレイな響きだった。でもさ、「とりあえず全部テンション入れとけばおしゃれになるでしょ」って思ってない？ 実はそれ、逆にゴチャゴチャして気持ち悪くなることがある。テンションは調味料。塩も砂糖も、入れすぎたら台無しだよね。

> 黒田: 今日のテーマは「判断」。add9 を入れていいタイミングと、sus4 が映えるタイミングと、シンプルに弾くべきタイミング。その見分け方を覚える。キーワードは「メロディーとの関係」。メロディーの音とコードのテンションがぶつかるかどうか、それだけ。

> 瀬戸: 素材はまた「慕い求めます」。L21 でも L22 でもやったけど、今回はメロディーの動きに注目して弾くよ。ワーシップギタリストは歌のメロディーを聴きながら伴奏するから、このスキルはめちゃくちゃ実践的。

## ① TABで弾く

### STEP 1: メロディーの動きを確認する

まずは「慕い求めます」のメロディーを単音で弾いてみよう。歌のメロディーラインをギターで確認する。

**※ すべてカポ2基準。フレット番号はカポ基準**

**メロディー例（アレンジ版）:**

ワーシップのメロディーは「長く伸ばす音（ロングトーン）」と「次々に音が変わる（動いている）」パートが交互に出てくる。ここではその特徴を再現したメロディーを使う。

e --------0-----------|-3-----------|-0-----------|-------------|

> 瀬戸: メロディーの「最後に伸ばす音」に注目してほしい。Verse の C のところでは E の音（1弦0フレット）で伸ばしてる。G のところでは G（1弦3フレット）。Am では C（2弦1フレット）。この「伸ばしてる音が何か」が、テンションを入れるかどうかの判断材料になる。

### STEP 2: ロングトーンの箇所でテンションを追加する

メロディーがロングトーン（長く伸ばしている）のとき = テンションを入れるチャンス。でもどのテンションが合うかは、メロディーの音次第。

#### 新コード①: Csus4（シーサスフォー）

```
Csus4:
e --1--   ← 人差し指        ← ここが変わる！（Cは開放だった）
B --1--   ← 人差し指
G --0--   ← 開放
D --3--   ← 薬指            ← ここも変わる！（Cは2フレットだった）
A --3--   ← 小指
E -----   ← 弾かない

構成音: F-C-E♭-G-C-F ... いや違う。
正しくは:
C sus4 = C - F - G
TAB の音（低→高）: C(A弦3f) - F(D弦3f) - G弦開放(G) - C(2弦1f) - F(1弦1f)

※ sus4 の「sus」= suspended（宙ぶらりん）。
  3度の音（E）が4度（F）に置き換わった状態。
  「解決したい」気持ちを生む、引っ張る響き。
```

> 瀬戸: L21 の STEP 5 でチラッと出てきた Csus4 だね。あのときは装飾テクニックとして「→ C に解決」って使い方だった。今日はもう一歩進んで、「いつ sus4 を使うか」の判断を覚える。

#### 新コード②: Gsus4（ジーサスフォー）

```
Gsus4:
e --3--   ← 小指
B --1--   ← 人差し指        ← ここが変わる！（G worship は3フレットだった）
G --0--   ← 開放
D --0--   ← 開放
A --3--   ← 薬指
E --3--   ← 中指

構成音: G-C-D-G-C-G ... いや。
正しくは:
G sus4 = G - C - D
TAB の音（低→高）: G(E弦3f) - C(A弦3f) - D(D弦開放) - G(G弦開放) - C(2弦1f) - G(1弦3f)

※ G の3度（B）が4度（C）に置き換わった形。
  2弦を1フレットにするだけで worship G から変わる。
```

> 瀬戸: worship G（2弦3フレット = D 音）から、2弦を1フレット（C 音）に変えるだけ。指1本ずらすだけで響きがガラッと変わる。

#### テンション追加の実験: C コードの箇所

Verse の C コードのところ、メロディーは最後に E（1弦0フレット）で伸ばしている。

**実験 1: C のところに Cadd9 を入れてみる**

```
Cadd9:
e --0--   ← E 音
B --3--   ← D 音（= 9th、add9 のテンション）
G --0--   ← G 音
D --2--   ← E 音
A --3--   ← C 音
E -----

メロディー: E（1弦0f）
テンション: D（2弦3f = 9th）

E と D = 全音（2フレット離れてる） → ぶつからない → OK！
```

> 瀬戸: メロディーが E で、add9 の音が D。この2つは全音（にちょん = 2フレット分）離れてるから、一緒に鳴ってもキレイに溶け合う。add9、ここでは合格。

**実験 2: C のところに Csus4 を入れてみる**

```
Csus4:
e --1--   ← F 音（= sus4 のテンション）
B --1--   ← C 音
G --0--   ← G 音
D --3--   ← F 音
A --3--   ← C 音
E -----

メロディー: E（歌っている音）
テンション: F（1弦1f = sus4 の音）

E と F = 半音（1フレットしか離れてない） → ガリッとぶつかる → NG！
```

> 瀬戸: 実際に弾いてみて。Csus4 を鳴らしながら E の音（1弦0フレット開放の音）を歌うか頭の中でイメージしてみて。……気持ち悪いでしょ？ 半音でぶつかってる。この「ガリッ」って感じが不協和音（ふきょうわおん = 心地よくない響き）。

**やってみよう:**

```
【体感ドリル: ぶつかりを聴く】

① Cadd9 を弾きながら、1弦0フレット（E）を一緒に鳴らす:
e --0--
B --3--
G --0--
D --2--
A --3--
→ キレイ！ D(9th) と E(メロディー) が溶け合う

② Csus4 を弾きながら、口で「ミー」（E の音）を歌う:
e --1--
B --1--
G --0--
D --3--
A --3--
→ ガリッ！ F(sus4) と E(メロディー) が半音でぶつかる
```

> 黒田: これが今日の核心。**半音でぶつかる = 不協和音**。テンションの音とメロディーの音が半音関係にあると、濁って聴こえる。逆に全音以上離れていれば、テンションは美しく響く。この「半音チェック」がテンション判断の鍵だよ。

#### テンション追加の実験: G コードの箇所

Verse の G コードのところ、メロディーは G（1弦3フレット）で伸ばしている。G は G コードのルート（根音）。

**実験 3: G のところに Gsus4 を入れてみる**

```
Gsus4:
e --3--   ← G 音
B --1--   ← C 音（= sus4 のテンション）
G --0--   ← G 音
D --0--   ← D 音
A --3--   ← C 音
E --3--   ← G 音

メロディー: G（1弦3f = ルート）
テンション: C（2弦1f = sus4 の音）

G と C = 完全4度（5フレット離れてる） → ぶつからない → OK！
```

> 瀬戸: メロディーがルートの上にいるとき、sus4 は安全。ルートと4度の間隔は「完全4度」（かんぜんよんど）って言って、すごく安定した響き。Gsus4 → G に解決する動きを入れると、曲に「引っ張って → 着地」の流れが生まれてカッコいい。

```
【Gsus4 → G 解決の弾き方】

Gsus4:          →    G worship:
e --3--              e --3--
B --1-- ← C 音       B --3-- ← D 音      ← 2弦を1f→3fにスライド
G --0--              G --0--
D --0--              D --0--
A --3--              A --2--              ← A弦は3f→2fに変える
E --3--              E --3--

弾き方: Gsus4 をジャラーン → 2弦を1f→3fにハンマリング
```

**実験 4: G のところに add9 的な音を入れてみる**

G の 9th = A 音。L21 の worship G（320033）では A 音は含まれていないが、開放5弦（A弦）を鳴らせば A 音が追加される。

```
G with add9 ニュアンス:
e --3--
B --3--
G --0--
D --0--
A --0--   ← 開放 = A 音（9th）  ★ 普段は2フレットだけど開放にする
E --3--

メロディー: G（1弦3f = ルート）
テンション: A（5弦開放 = 9th）

G と A = 全音 → ぶつからない → OK！
```

> 瀬戸: ルートの上にメロディーがいるときは、add9 も sus4 もどっちもイケる。自由度が高いポイント。好みで選んでいい。

### STEP 3: メロディーが3rd にいるときの「ぶつかり」を体感

ここが一番大事なポイント。

Chorus の C コードのところ、メロディーは E（1弦0フレット）で伸ばしている。E は C コードの3rd（3度 = メジャーの「明るさ」を決める音）。

**さっき STEP 2 で確認した通り:**

```
C の3rd = E
Csus4 の4th = F

E ← → F = 半音 = ぶつかる！

これが「3rdがメロディーにいるとき、sus4は危険」の理由。
```

もう一つ確認しよう。Am のところ。Chorus の Am コードで、メロディーが A（3弦2フレット）で伸ばしている場合。A は Am のルート。

```
Am のルート = A
もし Asus2 を使ったら:
  sus2 の音 = B（2弦0フレット開放）
  A ← → B = 全音 → ぶつからない → OK！

もし Asus4（x02230）を使ったら:
  sus4 の音 = D（2弦3フレット）
  A ← → D = 完全4度 → ぶつからない → OK！
```

> 瀬戸: Am のルート上にメロディーがいる場合、sus2 も sus4 も全然OK。でも Am の3rd = C のところにメロディーがいたら？

```
Am の3rd = C
Asus4 の4th = D
C ← → D = 全音 → ぶつからない → OK！

Am の3rd = C
Asus2 の2nd = B
C ← → B = 半音 → ぶつかる → NG！
```

> 黒田: 面白いでしょ？ Am の場合は sus2 が3rd とぶつかる。C の場合は sus4 が3rd とぶつかった。コードによって「どのテンションが危険か」が変わる。だからメロディーの音を毎回チェックする必要があるんだ。

**ぶつかり体感ドリル:**

```
【実際に弾いて確かめる】

① Csus4 + E メロディー（NG パターン）:
   Csus4（x33010）を弾く → 1弦を開放に変えて E を鳴らす
   e --0--  ← E に変更
   B --1--
   G --0--
   D --3--
   A --3--
   → F(sus4) と E(melody) が隣接 → 濁る

② Cadd9 + E メロディー（OK パターン）:
   Cadd9（x32030）をそのまま弾く
   e --0--  ← E 音がそのまま鳴る
   B --3--  ← D 音（9th）
   → D(add9) と E(melody) は全音離れてる → キレイ

③ Asus2 + C メロディー（NG パターン）:
   Asus2（x02200）を弾きながら、2弦1フレット（C）を意識する
   2弦開放 = B(sus2) と C(melody) が半音 → 濁る

④ Am7 + C メロディー（OK パターン）:
   Am7（x02010）をそのまま弾く
   3弦開放 = G(7th) と C(melody) → 全く問題なし → キレイ
```

> 瀬戸: ①②を弾き比べたら、「あ、本当にぶつかる！」って体でわかるはず。理屈じゃなくて耳で覚えることが大事。何回も弾いて、この「ガリッ」を体に染み込ませて。

### STEP 4: 判断フレームワークを使って「慕い求めます」を完成させる

ここまでの実験をまとめると、テンション判断は4つのルールで整理できる:

```
┌─────────────────────────────────────────────────────┐
│  テンション判断 4つのルール                           │
│                                                     │
│  ① メロディーがルート or 5th → add9/sus4 が映える   │
│  ② メロディーが3rd → sus4 は危険、add9 は OK        │
│  ③ メロディーが動いてる → シンプルに弾く            │
│  ④ メロディーがロングトーン → テンション入れるチャンス│
└─────────────────────────────────────────────────────┘
```

このルールを使って、「慕い求めます」の全コード進行にテンション判断を書き込もう。

→ 全部おしゃれコード。「キレイだけど、なんか味が一定」

## ② 事後理論

> 黒田（理論メモ）: 今弾いたものの「なぜ」を整理するよ。

### テンション判断フレームワーク — 4つのルール

```
ルール①: メロディーがルート or 5th にいる
  → add9 も sus4 も映える
  → 理由: ルート・5thはコードの「骨格」。テンションと十分な音程差がある

ルール②: メロディーが3rd にいる
  → sus4 は危険（半音でぶつかる）
  → add9 は OK（全音離れてるから溶け合う）
  → 理由: 3rd と 4th は半音関係。これは全キー・全コードで同じ

ルール③: メロディーが動いてる（細かく音が変わる）
  → テンションを入れず、シンプルなコードで弾く
  → 理由: 動いてるメロディーは一瞬ごとにぶつかりポイントが変わる。
           シンプルなコードなら安全。テンションを入れると「当たり外れ」が出る

ルール④: メロディーがロングトーン（長く伸ばしてる）
  → テンションを入れるチャンス
  → 理由: 伸ばしてる = その音が長く聴こえる = テンションとの関係が目立つ。
           キレイに響けば効果大、ぶつかれば被害も大
```

### なぜ sus4 は 3rd とぶつかるのか

> 黒田: 音楽理論で「半音」（はんおん = 隣り合うフレット）の距離は、最も緊張感が高い音程。

```
C メジャースケール上で見る:

... B  C  D  E  F  G  A  B  C ...
        ↑          ↑
        ルート      5th
              ↑
              add9(=2nd)
                 ↑  ↑
                 3rd 4th(sus4)
                 ↕
              半音！

E（3rd）と F（4th = sus4）は半音しか離れていない。
だからメロディーが E のとき、コードに F が入ると濁る。

D（add9 = 2nd）と E（3rd）は全音離れてる。
だからメロディーが E のとき、コードに D が入っても大丈夫。
```

> 黒田: これは C コードだけの話じゃない。**どのメジャーコードでも 3rd と 4th は半音関係**。だから「メロディーが 3rd にいたら sus4 は避ける」は万能ルールなんだ。

### Phase 2 との接続: add9 の拡張

> 黒田: L5 で覚えた DM7(9) を覚えてる？ あれは「D メジャー7th + 9th」だった。今日の add9 は「トライアド + 9th」。原理は同じで、9th（= 2nd のオクターブ上）を足してるだけ。

```
Cadd9    = C + E + G + D       ← トライアド + 9th
Cmaj7(9) = C + E + G + B + D   ← トライアド + 7th + 9th

つまり add9 は「7th なしの9th」。
DM7(9) は「7th ありの9th」。仲間だよ。
```

> 黒田: テンション判断のルールは、7th コードにテンションを足すときも全く同じ。メロディーとの半音チェックさえすれば、どんなコードにも応用できる。

### ドリル 2: テンション ON/OFF 弾き比べ

同じ4小節を「テンション全部入り」と「判断済み」で弾き比べて、違いを耳で確認する。

```
【素材: Verse の最初の4小節】

パターン A: テンション全部入り（判断なし）
| Cadd9   | Gsus4→G | Am7     | Fsus2   |
  ↑ OK      ↑ OK      ↑ OK      ↑ OK
→ たまたま全部 OK のケース。でも実は......

パターン B: テンション全部入り + sus4 も全部入り
| Csus4→C | Gsus4→G | Asus2   | Fsus2   |
  ↑ NG!     ↑ OK      ↑ OK      ↑ OK
→ 最初の Csus4 でメロディー(E) とぶつかる

パターン C: 判断済み
| Cadd9   | Gsus4→G | Am7     | Fsus2   |
→ メロディーに合わせて最適化。パターン A と同じに見えるけど、
   「なぜこれを選んだか」が明確。
   別の曲でメロディーが違えば、判断結果も変わる。
```

> 瀬戸: 「慕い求めます」の Verse はたまたま全部テンション OK のケースが多い。でもこれは曲によって変わる。大事なのは「なんとなく入れた」じゃなくて「チェックした上で入れた」ということ。判断できるようになれば、初めて弾く曲でもテンションの使い方に迷わなくなる。

## ④ 自分で組み合わせる

### チャレンジ: テンション判断 + L22 色替え = 完全版バッキング

L22 の「セクション別ヴォイシング色替え」と、今日の「テンション判断」を両方使って、「慕い求めます」の最終バッキングを作ろう。

**組み合わせの指針:**

```
ヴァース（静か、テンション活用）:
  → ロングトーンが多い → テンション入れるチャンス多い
  → 繊細なヴォイシング（add9, Am7, Fsus2）
  → Gsus4 → G の解決も使う

コーラス（力強い、テンション控えめ）:
  → メロディーが動いてるところが多い → シンプル寄り
  → 力強いヴォイシング（F barre, C/G）
  → ただしロングトーンでは Gsus4 → G を使う

全体を通して:
  → メロディーの 3rd では sus4 を避ける
  → 動いてるところではテンションを我慢する
  → ロングトーンで「ここぞ」とテンションを入れる
```

**完全版スコア:**

```
Verse（静か、テンション豊か）:
| Cadd9      | Gsus4→G wor.| Am7        | Fsus2      |
| Cadd9      | Gsus4→G wor.| Am7        | Fsus2      |
| Dm         | G wor.      | Cadd9      |            |
  弾き方: 軽いストローク or アルペジオ
  Gsus4→G: 2弦1f→3f ハンマリングで解決

Chorus（力強い、メリハリ）:
| F          | Cadd9       | Dm         | Gsus4→G wor.|
| F          | C/G         | Am→Am7     | Gsus4→G wor.|
| Dm         | G wor.      | Cadd9      |            |
  弾き方: フルストローク
  F: barre で力強く
  C/G: 6弦3f でどっしり
  Gsus4→G: コーラスでも解決の動きで盛り上げる
```

> 瀬戸: これが L21〜L23 の集大成。カポ2のワーシップコード（L21）、セクション別の色替え（L22）、テンション判断（L23）。3つのレッスンが全部ここに入ってる。教会で弾くとき、この3つを意識するだけでバッキングのクオリティがめちゃくちゃ上がるよ。

> 黒田: 理論的に整理すると、テンション判断は「メロディーとハーモニーの関係を聴く力」を鍛えてる。これはワーシップだけじゃなく、どのジャンルでも使える基礎力。ポップスでもジャズでも、伴奏するときは必ずメロディーとの関係を考える。今日学んだ4つのルールは、どんな曲にも応用できるからね。

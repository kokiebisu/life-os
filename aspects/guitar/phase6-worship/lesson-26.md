## はじめに

> 瀬戸: Phase 6 の最終回。今日は新しいテクニックは一切やらない。L21 〜 L25 で覚えたこと——カポ、ワーシップヴォイシング、色替え、テンション判断、モードの性格、ベースライン制御——を「全部使って、知らない曲をおしゃれにする」ための実践回。今までは俺が「ここはこう弾いて」って教えてきたけど、今日は自分で判断する練習。コード譜を渡されたとき、何をどう考えて、どう弾くかを叩き込む。

## ① TABで弾く

### STEP 1: 5ステップ・リハーモナイズ手順

L21 〜 L25 の知識を、1枚のチートシートにまとめる。

> 瀬戸: これから紹介する5ステップは、コード譜を受け取ってから「自分のバッキング」を作るまでの手順。順番通りにやれば、どんな曲でもおしゃれにできる。まずは全体像を見てから、1曲目で実演するよ。

```

│        5ステップ・リハーモナイズ手順（チートシート）                │
│                                                                   │
│  ① キー確認 → ダイアトニックコードを把握する（L24）               │
│     「この曲のキーは何？ ダイアトニック（= そのキーの基本コード）   │
│      は何？」                                                     │
│     → カポの位置を決める                                          │
│                                                                   │
│  ② セクション分け → ヴォイシングを割り当てる（L22: 色替え）       │

│     → Chorus = フルに鳴らすヴォイシング                           │

│     → ぶつかる場合はテンションを入れない                         │
│                                                                   │
│  ④ 感情の流れ → モード的コード機能を確認する（L24: 性格）        │
│     「このセクションの感情は？ 明るい→暗い→解放？」               │
│     → Ionian(I) = 帰還・安定                                     │
│     → Dorian(ii) = 浮遊・ジャズ                                  │
│     → Lydian(IV) = 神秘・希望                                    │
│     → Mixolydian(V) = 推進力                                     │
│     → Aeolian(vi) = 祈り・内省                                   │
│                                                                   │
│  ⑤ ベースライン確認 → 経過コード / slash chord を検討する（L25） │
│     「ベースが3度以上飛ぶところはどこ？」                         │
│     → 下降: C/B, Am/G 等を挟む                                   │
│     → 上昇: C/E, G/B 等を挟む                                    │
│     → 固定: ペダルポイントで安定感                                │
│     → 目安: 1小節に1つの「おしゃれ要素」まで                     │

> 瀬戸: これ、プリントアウトしてギターケースに入れとくといいよ。リハ前にチラッと見るだけで「あ、そうだ、ベースラインチェックしなきゃ」って思い出せる。

> 黒田: 5ステップの順番にも意味がある。①②は「大枠」、③④は「細部」、⑤は「つなぎ」。大きいところから小さいところに降りていく。いきなり⑤からやると全体が見えなくなるから、必ず①から順番にやってね。

### STEP 2: 1曲目 — 先生がデモ（5ステップを実演）

**曲:「主よ あなたの愛で」（ワーシップ曲の例）**
Key of G（カポなしで弾く — カポ2以外のキーでも同じ手順が使えることを示す）

**基本コード譜（リーダーから送られてきた想定）:**

```
Verse:
| G       | D       | Em      | C       |
| G       | D       | Em      | C       |

Chorus:
| C       | G       | D       | Em      |
| C       | G       | D       |         |

Bridge:
| Em      | C       | G       | D       |
```

> 瀬戸: よくあるワーシップ曲の進行だね。これだけ見ると「ああ、I-V-vi-IV ね」で終わりそうだけど、ここから5ステップを回すとどうなるか、やってみるよ。

---

#### ステップ①: キー確認 → ダイアトニックコード把握

```
Key = G major

ダイアトニックコード:
I = G,  ii = Am,  iii = Bm,  IV = C,  V = D,  vi = Em,  vii° = F#dim

カポの判断:
→ Key of G なら、カポなしで開放弦コードがそのまま使える。
→ 開放弦（G, D, Em, C, Am）が豊富に使えるので、カポなしが最適。
```

> 黒田: L21 でやった「カポ2 で Key of D を C シェイプで弾く」と同じ発想。今回は Key of G だから、カポなしで G シェイプがそのまま使える。カポが不要なケースもある、っていうのが大事なポイント。

---

#### ステップ②: セクション分け → ヴォイシング割り当て

```
Verse（穏やかに語りかける）:
→ Gadd9, Dsus4, Em7, Cadd9（柔らかい開放弦ヴォイシング）

Chorus（力強く歌い上げる）:
→ C full, G power(6弦ルート), D open, Em open（フルストローク）

Bridge（親密に、祈るように）:
→ Em7, Cmaj7, G/B, Dsus4→D（最小限の音、しっとり）
```

> 瀬戸: L22 の「色替え」そのまま。Verse は add9 や sus4 で柔らかく、Chorus はシンプルにフルで鳴らして声量に合わせる、Bridge は音を減らして親密さを出す。

---

#### ステップ③: メロディーチェック → テンション追加判断

```
Verse のメロディー（仮想）:
  G 上でメロディーが A（9th）を長く伸ばす → Gadd9 OK ✓
  D 上でメロディーが E（sus4→3rd）に解決 → Dsus4→D ✓
  Em 上でメロディーが D（7th）を含む → Em7 OK ✓
  C 上でメロディーが D（9th）に乗る → Cadd9 OK ✓

Chorus のメロディー:
  C 上でメロディーが C（ルート）→ テンション不要、素のCでOK
  G 上でメロディーが G → テンションなしでOK
  D 上でメロディーが E（sus4）を経過 → Dsus4→D
  Em 上でメロディーが B（5th）→ テンション不要

Bridge のメロディー:
  Em 上でメロディーが D → Em7 ✓
  C 上でメロディーが B（maj7）→ Cmaj7 ✓
```

> 黒田: L23 のフレームワーク。「ロングトーンがテンションと合うか？」だけを確認する。合わなかったらテンションを入れない。シンプルなルール。

---

#### ステップ④: 感情の流れ → モード的コード機能確認

```
Verse: G(I) → D(V) → Em(vi) → C(IV)
  I-V-vi-IV = 最も定番の進行。Ionian(安定)→Mixolydian(推進)→Aeolian(祈り)→Lydian(希望)
  → 感情: 安心 → 前進 → 内省 → 希望。自然な流れ ✓

Chorus: C(IV) → G(I) → D(V) → Em(vi)
  IV-I-V-vi = IV スタート。Lydian(希望)→Ionian(帰還)→Mixolydian(推進)→Aeolian(祈り)
  → 感情: 希望から始まって帰還し、また祈りへ。Chorus に IV スタートの明るさ ✓

Bridge: Em(vi) → C(IV) → G(I) → D(V)
  vi-IV-I-V = vi スタート。Aeolian(祈り)→Lydian(神秘)→Ionian(帰還)→Mixolydian(解放)
  → 感情: 祈りから始まって神秘を経て解放へ。Bridge にぴったり ✓
```

> 黒田: L24 の「5つの性格」を使って、セクションごとの感情の流れを確認してる。ここでは「流れが自然か」を見るだけ。不自然な感情の飛躍があったら、代理コード（同じ機能の別コード）を検討する場面もあるけど、今回は自然な流れだからそのまま OK。

---

#### ステップ⑤: ベースライン確認 → 経過コード / slash chord 検討

```
Verse のベース移動:
  G → D: G→D = 完全4度（飛ぶ）→ G/B を挟める？
         ただし Verse は穏やかなセクション → 入れすぎ注意
  D → Em: D→E = 全音（スムーズ）→ そのままOK
  Em → C: E→C = 3度（少し飛ぶ）→ Em/D を挟める
  C → G: C→G = 完全5度（飛ぶ）→ C→D/F#→G? 上昇アプローチ

判断:
  Verse は「穏やか」がコンセプト → ベースの動きも控えめに。
  Em → C の Em/D だけ入れる（1箇所のみ）。

Chorus のベース移動:
  C → G: 5度（飛ぶ）→ そのまま力強く飛ばす（Chorus だから OK）
  D → Em: 全音 → そのままOK
  G → D: 4度 → そのまま

判断:
  Chorus は力強いセクション → ベースの跳躍も力強さ。
  あえて経過コードを入れない。

Bridge のベース移動:
  Em → C: 3度 → Em/D を挟む（ベース: E → D → C で下降）
  C → G: 5度 → C→D/F#→G? → いや、やりすぎ。そのまま。
  G → D: 4度 → G/B 不要（Bridge のラストは解放感、飛んでOK）

判断:
  Bridge の Em → C に Em/D だけ入れる（1箇所のみ）。
```

> 瀬戸: ⑤まで来た。結局、ベースラインの経過コードは Verse と Bridge でそれぞれ1箇所ずつ。「1小節に1つのおしゃれ要素」っていうルールに従うと、自然とこのくらいに収まる。全部に入れたくなる気持ちはわかるけど、「引き算」が大事。

> 瀬戸: 全部自分で考えてみて。正解は1つじゃない。「自分はこう弾きたい」っていう意志が大事。考えたら、下の模範解答と比べてみよう。

#### 模範解答（瀬戸のリハーモナイズ版）

**ステップ① キー確認:**

```
Key = D major（カポ2 で C シェイプ）
ダイアトニック（弾く形で表記）: C(I), Dm(ii), Em(iii), F(IV), G(V), Am(vi), Bdim(vii°)
```

**ステップ② ヴォイシング割り当て:**

```
Verse: 穏やか → Cadd9, Am7, Fsus2, G worship
Chorus: 力強い → F full, G full, Am open, Cadd9→C
Bridge: 祈り → Am7, Fmaj7, C/E, Gsus4→G
```

**ステップ③ テンション判断:**

```
Cadd9: メロディーが D(9th)に乗る → OK ✓
Am7: メロディーが G(7th)を含む → OK ✓
Fsus2: メロディーが G(sus2=9th)を経過 → OK ✓
G worship: 3弦開放の響きが加わるだけ → 常に OK ✓
Fmaj7: メロディーが E(maj7)に触れる → OK ✓
Gsus4: メロディーが C(sus4)から B(3rd)へ解決 → OK ✓
```

**ステップ④ 感情の流れ:**

```
Verse: C(I)→Am(vi)→F(IV)→G(V)
  Ionian(安定)→Aeolian(祈り)→Lydian(希望)→Mixolydian(推進)
  → 安定から祈りに入り、希望へ。自然 ✓

Chorus: F(IV)→G(V)→Am(vi)→F(IV)→G(V)→C(I)
  Lydian→Mixolydian→Aeolian→Lydian→Mixolydian→Ionian
  → 希望→推進→祈り→希望→推進→帰還。盛り上がりのアーチ ✓

Bridge: Am(vi)→F(IV)→C(I)→G(V)
  Aeolian(祈り)→Lydian(神秘)→Ionian(帰還)→Mixolydian(解放)
  → 祈りから光に向かう流れ。As the Deer の Bridge と同じ構造 ✓
```

**ステップ⑤ ベースライン:**

```
Verse:
  C → Am: C→A = 3度（飛ぶ）→ C/B を挟む（C→B→A で下降）
  Am → F: A→F = 3度（飛ぶ）→ Am/G を挟む（A→G→F で下降）
  F → G: F→G = 全音 → そのままOK
  G → C: G→C = 4度 → Verse 冒頭に戻るだけ → そのままでOK

## ② 事後理論

> 黒田（理論メモ）: 今回は新しい理論はない。L21 〜 L25 の理論の「統合」をする。

### リハーモナイズに「正解」はない — 曲の文脈と感性で判断

```
同じ C-Am-F-G でも:
  ・テンション全入りバージョン: Cadd9-Am9-Fsus2-Gsus4
  ・テンション控えめバージョン: Cadd9-Am7-F-G
  ・ベースライン重視バージョン: C-C/B-Am-Am/G-F-G
  ・シンプルバージョン: C-Am-F-G（そのまま！）

全部「正解」。曲の文脈、バンドの音量、会衆の雰囲気で変わる。
リハーモナイズは「できる」ことが大事で、「必ずやる」ことではない。
```

> 黒田: 理論は「地図」みたいなもの。「ここに行けるよ」って教えてくれるけど、「必ずそこに行け」とは言ってない。地図を持ってるのと持ってないのとでは、迷ったときの安心感が全然違う。5ステップはその地図。

### 「やりすぎ」の3つのサイン

```
サイン1: テンションの渋滞
  → テンションコードが2つ以上連続すると、「おしゃれ」ではなく「うるさい」になる。
  → 例: Cadd9 → Am9 → Fsus2 → Gsus4 = 4連続テンション → やりすぎ
  → 修正: Am9 → Am7 に落とす、Gsus4 → G にする、など間引く

サイン2: ベースラインの渋滞
  → 全コードに slash chord / 経過コードを入れると、ベースが常に歩いてて落ち着かない。
  → 目安: 4小節中1-2箇所。残りはルートで安定させる。

サイン3: 歌のじゃまをしてる
  → テンションが歌のメロディーとぶつかると、会衆が音を取りにくくなる。
  → 迷ったらテンションを外す。シンプルが正義。
```

### L21 の心得を最上位に置く

> 黒田: L22-L25 で色々なテクニックを覚えたけど、L21 の3つの心得が「憲法」。テクニックは「法律」。法律は憲法に従う。おしゃれなコードワークが「シンプルでいい」「リーダーを聴け」に反するなら、テクニックの方を引っ込める。

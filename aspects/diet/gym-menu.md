# ジムメニュー

## 方針

- デカくしない。体脂肪を落として体力をつける
- メイン: **BIG3（ベンチプレス・スクワット・デッドリフト）** をスミスマシンで
- サブ: 往復徒歩が有酸素運動を兼ねる
- 週3回、1回約30分（+ 往復徒歩40分）

## 場所

- FIT PLACE24 みなとみらい
- 片道徒歩20分（往復40分 = ウォームアップ＋クールダウン）
- 合計所要時間: 約70分（徒歩20分 + ワークアウト30分 + 徒歩20分）

## メニュー: BIG3（スミスマシン）

全種目共通: **3セット × 15回 / セット間休憩60秒**

---

### 1. ベンチプレス — 胸・腕・肩

> バーベルを胸の上で上げ下げする。上半身の「押す」動作の基本。
> スミスマシン（バーの軌道が固定）で安全にフォームを覚える。

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 回数 | 15回 |
| セット間の休憩 | 60秒 |
| 重量 | バーのみ（20kg）→ 慣れたら +2.5kg ずつ |
| 鍛える場所 | 胸（大胸筋）・腕（三頭筋）・肩 |

**やり方:**
1. ベンチに仰向けに寝る。足は床にしっかりつける
2. バーを肩幅より少し広めに握る
3. ラックからバーを外し、胸の上に構える
4. ゆっくりバーを胸に下ろす（胸に軽く触れるくらい）
5. 胸から押し上げてスタートポジションに戻す → これで1回

**注意:**
- **必ずセーフティバーを設定する**（一人で潰れると危険）
- 手首を返さない（バーの真下に手首がくるように）
- 肩甲骨を寄せて胸を張る（肩ではなく胸で押す意識）
- お尻をベンチから浮かせない

---

### 2. スクワット — 太もも・お尻・体幹

> バーを肩に担いでしゃがんで立ち上がる。下半身と体幹を同時に鍛える「キング・オブ・エクササイズ」。
> スミスマシンならバランスを気にせずフォームに集中できる。

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 回数 | 15回 |
| セット間の休憩 | 60秒 |
| 重量 | バーのみ（20kg）→ 慣れたら +2.5kg ずつ |
| 鍛える場所 | 太もも（大腿四頭筋）・お尻（大臀筋）・体幹 |

**やり方:**
1. バーを肩の後ろ（僧帽筋の上）に担ぐ。首の骨に当てない
2. 足を肩幅に開き、つま先をやや外向きにする
3. お尻を後ろに引きながら膝を曲げてしゃがむ（太ももが床と平行まで）
4. かかとで地面を押して立ち上がる → これで1回

**注意:**
- **膝がつま先より前に出すぎない。お尻を後ろに引く意識**
- 背中を丸めない。胸を張って前を見る
- かかとが浮かないように（浮くなら足の位置をやや前にずらす）
- 膝を内側に入れない（つま先と同じ方向に曲げる）

---

### 3. デッドリフト — 背中・太もも裏・体幹

> 床のバーを持ち上げて立ち上がる。背面全体を鍛える。
> BIG3の中で最も多くの筋肉を同時に使う種目。

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 回数 | 15回 |
| セット間の休憩 | 60秒 |
| 重量 | バーのみ（20kg）→ 慣れたら +2.5kg ずつ |
| 鍛える場所 | 背中（脊柱起立筋・広背筋）・太もも裏（ハムストリング）・お尻・体幹 |

**やり方:**
1. バーの前に立つ。足は腰幅、バーがすねに近い位置
2. 腰を折って（股関節から）バーを肩幅で握る
3. 胸を張り、背中をまっすぐにしたまま、足で地面を押して立ち上がる
4. 立ち上がったら肩甲骨を軽く寄せる
5. 同じ軌道でゆっくり下ろす → これで1回

**注意:**
- **腰を丸めない。これだけは絶対守る**（腰を壊す最大の原因）
- バーは体に近い軌道で動かす（体から離れると腰に負担）
- 持ち上げるとき腕で引っ張らない。足と背中で上げる意識
- 重量は控えめから。フォームが最優先

---

## 重量の進め方（プログレッシブ・オーバーロード）

「15回3セットが楽にできる」ようになったら、次回から重量を上げる。

| 種目 | 初期重量 | 増加幅 |
|------|---------|--------|
| ベンチプレス | バーのみ（20kg） | +2.5kg |
| スクワット | バーのみ（20kg） | +2.5kg |
| デッドリフト | バーのみ（20kg） | +2.5kg |

- 上げて15回できなくなったら、その重量で3セット15回できるまで続ける
- 焦らない。フォームが崩れるくらいなら重量を戻す
- **デッドリフトは特に慎重に。他の2種目より軽めで始める**

## 持ち物

- 着替え（Tシャツ + 短パン）
- タオル
- 水（500ml以上）
- イヤホン
- ジム用シューズ

- トリガー: 昼食後 → そのままジムへ（毎回判断しない）
- 朝の時点でバッグを準備しておく
- 帰宅後シャワー → 午後の開発へ（リフレッシュ効果で集中力回復）
